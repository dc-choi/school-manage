# 기능 소개 - "부모님 연락처, 전용 필드로 정리됐어요"

## 메타

| 항목 | 내용 |
|------|------|
| 유형 | 피드 게시물 (카드뉴스, 캐러셀) |
| 타겟 | 교리교사 |
| CTA | 프로필 링크 |
| 카드 수 | 5장 |
| 사이즈 | 1080×1080px (1:1 정사각형) |
| 대상 기능 | 학생 `parentContact` 필드 (선택 입력, 자유 형식) |

### 공통 스타일 프롬프트
```
카드뉴스 스타일. 따뜻한 크림/베이지 배경에 부활시기 포인트 컬러(흰색·골드). 보조 색으로 부드러운 하늘색. 제목은 세리프 느낌의 품위 있는 서체, 본문은 깔끔한 산세리프 서체. 성당 게시판의 현대판 — 신뢰감 있고 친근하면서 깔끔한 느낌. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻하고 밝은 톤의 부드럽고 품위 있는 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 카드 구성

### Card 1 — 표지

**메인 카피**:
```
부모님 연락처도
한곳에 정리돼요
```

**서브 카피**:
```
비고란에 적어두지 말고, 전용 필드로
```

```
카드뉴스 표지. 따뜻한 크림색 배경에 골드/아이보리 포인트 컬러, 보조로 부드러운 하늘색. 상단에 서비스 로고(주일학교 출석부). 중앙에 세리프 느낌의 크고 품위 있는 한국어 텍스트 "부모님 연락처도 한곳에 정리돼요" — 두 줄로 배치. 하단에 작은 산세리프 텍스트 "비고란에 적어두지 말고, 전용 필드로". 중앙 하단에 스마트폰 한 대가 놓여 있고 화면에 학생 정보 입력 폼이 보이며, "부모님 연락처" 라벨이 있는 필드 옆에 전화기 아이콘이 작게 반짝인다. 전체적으로 따뜻하고 품위 있는 부활 시기 분위기 — 밝고 희망찬 톤. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻하고 밝은 톤. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

```
이제 '부모님 연락처' 전용 필드가 생겨서
깔끔하게 따로 정리할 수 있어요
```

```
카드뉴스 기능 소개 슬라이드. 따뜻한 크림색 배경에 골드 포인트 컬러. 상단에 세리프 느낌의 품위 있는 한국어 텍스트 "비고란에 적어두던 연락처, 이제 전용 필드로". 중앙에 Before/After 구성. 왼쪽 Before: 회색 테두리의 학생 메모 카드 — "비고란" 라벨 아래 "부모님 010-... / 알레르기 / 참고사항"이 줄줄이 뒤섞여 있어 약간 어수선하다. 오른쪽 After: 골드 테두리의 깔끔한 학생 정보 카드 — '이름', '학년', '부모님 연락처' 항목이 각자 전용 필드로 정돈되어 있고 '부모님 연락처' 옆에 전화기 아이콘. 두 카드 사이에 오른쪽 방향 화살표. After 쪽에 작은 골드 체크마크와 반짝이. 하단에 산세리프 본문 텍스트. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻하고 밝은 톤. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

```
어떤 형식이든 입력돼요
```

**본문**:
```
010-1234-5678, 010 1234 5678, 01012345678
편한 대로 입력하셔도 돼요

선택 입력이라 필요한 학생에게만
적어두셔도 됩니다
```

```
카드뉴스 기능 소개 슬라이드. 따뜻한 크림색 배경에 골드 포인트 컬러. 상단에 세리프 느낌의 품위 있는 한국어 텍스트 "어떤 형식이든 입력돼요". 중앙에 "부모님 연락처" 라벨이 붙은 입력 필드 목업 3개가 세로로 쌓여 있다 — 각각 "010-1234-5678", "010 1234 5678", "01012345678"이 입력되어 있고 각 필드 오른쪽에 골드색 체크마크가 보인다. 상단 오른쪽 구석에 작은 "선택 입력" 뱃지(부드러운 하늘색 배경). 하단에 산세리프 본문 텍스트. 전체적으로 신뢰감 있고 편안한 분위기. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻하고 밝은 톤. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

### Card 4 — 입력 전 확인 한 가지

**헤더**:
```
입력 전에 한 가지만
확인해 주세요
```

**본문**:
```
연락처는 같은 모임 선생님만 볼 수 있어요

다만 부모님 개인정보라
등록제로 받은 부모님 연락처만
입력해 주시면 더 좋습니다
```

```
카드뉴스 안내 슬라이드. 따뜻한 크림색 배경에 부드러운 하늘색 포인트 컬러. 상단에 세리프 느낌의 품위 있는 한국어 텍스트 "입력 전에 한 가지만 확인해 주세요". 중앙에 두 항목이 세로로 배치된 체크리스트 — 각 항목 왼쪽에 작은 골드색 체크마크: (1) "같은 모임 선생님만 볼 수 있어요" — 작은 자물쇠 아이콘, (2) "등록제로 받은 부모님 연락처만 입력" — 체크가 된 명부/등록 서류 일러스트. 아이콘은 따뜻한 일러스트 톤. 하단에 산세리프 본문 텍스트. 과하게 경고하는 분위기가 아니라 부드럽고 친근한 안내 분위기. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻하고 밝은 톤. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션

```
부모님 연락처 필드가 새로 생겼어요.

그동안은 학생 비고란(메모)에 적어두는 경우가 많았는데,
이제 '부모님 연락처' 전용 필드로 따로 정리할 수 있어요.

- 어떤 형식이든 입력 가능 (010-1234-5678, 010 1234 5678 모두 OK)
- 선택 입력이라 필요한 학생에게만 채우셔도 됩니다
- 같은 모임 선생님만 볼 수 있어요

부모님 개인정보라, 등록제로 받은 부모님 연락처만 입력해 주시면 좋아요.

처리방침에도 '보호자 연락처'가 수집 항목으로 반영됐고,
기존에 동의하셨던 분들은 다음 로그인 시 변경 사항 안내 화면이 자동으로 떠요.
```
