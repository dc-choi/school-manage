# 랜딩 페이지 공개 - "이제 프로필 링크 하나로 다 보여드립니다"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사 / 모임 운영자
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 남성 개발자, 안경을 쓰고 후드티를 입고 있으며 가슴에 작은 십자가 뱃지를 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
"이 서비스가 뭔데?" 하시는 분들 많으셨죠.
그래서 한눈에 볼 수 있는 랜딩 페이지를 만들었습니다.
프로필 링크에서 확인해보세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "이 서비스가 뭔데요?"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 20대 남성 개발자 캐릭터(안경, 후드티, 가슴에 작은 십자가 뱃지) 앞에 여러 명의 작은 캐릭터들이 물음표를 띄우며 고개를 갸웃하고 있는 장면. 작은 캐릭터들의 말풍선에 "이 서비스가 뭔데요?", "어떤 기능이 있어요?". 개발자 캐릭터는 살짝 당황하며 손을 흔드는 포즈. 머리 위에 땀방울 이펙트. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "그래... 한번에 보여줄 수 있는 페이지가 필요하겠다"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 턱을 괴고 고민하는 장면. 머리 위에 전구 아이콘이 켜지려는 모습. 옆에 노트북 화면이 보이고, 화면에는 기존 로그인 페이지가 표시되어 있다. 말풍선에 "그래... 한번에 보여줄 수 있는 페이지가 필요하겠다". 생각하는 표정, 눈이 살짝 빛나기 시작하는 모습. 4컷 만화 중 2컷 스타일, 아이디어가 떠오르는 순간을 표현.
```

### 3컷 (전)
- 대사: "출석 관리, 통계, 무료... 다 담았다!"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 노트북 앞에서 활기차게 코딩하는 장면. 노트북 화면에 랜딩 페이지 레이아웃이 보이고, 화면 안에 작은 아이콘들이 떠있다: 체크마크(출석), 막대그래프(통계), "무료" 글씨가 들어간 뱃지. 말풍선에 "출석 관리, 통계, 무료... 다 담았다!". 캐릭터는 뿌듯한 표정, 손가락으로 화면을 가리키는 포즈. 주변에 작은 별 이펙트. 4컷 만화 중 3컷 스타일, 완성의 보람을 코믹하게 표현.
```

### 4컷 (결)
- 대사: "이제 프로필 링크 하나면 됩니다"
- 하단: "서비스가 궁금하셨다면, 프로필 링크를 눌러보세요" + "@weekly-school"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 정면을 바라보며 한 손으로 스마트폰을 들고 있는 장면. 스마트폰 화면에 랜딩 페이지가 보이고, 화면 안에 "출석 관리를 더 쉽게" 텍스트가 살짝 보인다. 다른 손은 엄지를 치켜세우는 포즈. 말풍선에 "이제 프로필 링크 하나면 됩니다". 밝고 자신감 있는 표정, 살짝 미소. 하단에 텍스트 박스: "서비스가 궁금하셨다면, 프로필 링크를 눌러보세요" 그리고 작은 글씨로 "@weekly-school". 4컷 만화 중 4컷(마무리) 스타일, 따뜻하고 편안한 톤.
```
