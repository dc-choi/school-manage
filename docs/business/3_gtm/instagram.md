# 인스타그램 운영 방향

> 이 문서는 인스타그램 채널 운영 계획을 정리합니다.

## 개요

### 목적
- 교리교사 대상 인지도 확보
- 서비스 소개 및 사용 사례 공유
- DM을 통한 직접 문의/신청 경로 제공

### 타겟
- 주일학교 교리교사 (중고등부 우선)
- 본당 청소년 사목 담당자

## 계정 설정

### 계정명 (후보)
- `@sundayschool_attendance`
- `@주일학교출석부`
- `@교리교사도구`

### 프로필 구성
- 프로필 사진: 서비스 로고
- 소개: "주일학교 출석/학생/통계 관리 도구"
- 링크: 서비스 URL (로그인 페이지)
- CTA: "DM으로 문의하세요"

## 콘텐츠 전략

### 콘텐츠 유형

| 유형 | 빈도 | 예시 |
|------|------|------|
| 기능 소개 | 주 1회 | 출석 체크 화면, 통계 화면 스크린샷 |
| 사용 팁 | 격주 1회 | "학생 일괄 등록하는 법" |
| 사용 후기 | 월 1회 | 장위동 사례 (익명화) |
| 주일학교 운영 팁 | 월 2회 | 출석률 높이는 방법, 행사 준비 체크리스트 |

### 콘텐츠 형식

| 형식 | 용도 |
|------|------|
| 피드 (이미지) | 기능 소개, 통계 시각화 |
| 릴스 (영상) | 사용법 튜토리얼 (30초 이내) |
| 스토리 | 공지, 리마인드, 일상 |
| 하이라이트 | 기능별 모음, FAQ, 문의 방법 |

### 게시 주기 (1인 운영 기준)
- 피드: 주 1~2회
- 스토리: 주 2~3회
- 릴스: 월 2회

## 콘텐츠 예시

### 피드 콘텐츠

**1. 기능 소개**
```
[이미지: 출석 체크 화면 스크린샷]

출석 체크, 터치 한 번이면 끝!

✅ 출석/결석/지각 간편 체크
✅ 자동 저장
✅ 주간/월간 통계 자동 계산

#주일학교 #교리교사 #출석관리
```

**2. 통계 공유**
```
[이미지: 출석률 그래프]

이번 달 우리 반 출석률은?

📊 한눈에 보는 출석 통계
📈 월별 추이 확인
📋 사제님 보고용 리포트

#주일학교통계 #출석률
```

**3. 사용 후기**
```
[이미지: 사용 화면 + 텍스트 오버레이]

"엑셀로 정리하던 시간이 반으로 줄었어요"
- 서울 OO성당 교리교사

#주일학교 #교리교사후기
```

### 릴스 콘텐츠

**1. 사용법 튜토리얼**
- "30초 만에 학생 등록하기"
- "출석 체크 이렇게 하세요"
- "통계 확인하는 방법"

**2. Before/After**
- 엑셀 관리 vs 서비스 사용 비교

## CTA (Call to Action)

### 문의 유도
- 프로필 링크 클릭 유도
- "DM 주시면 안내해드려요"
- "프로필 링크에서 바로 시작하세요"

### 신청 프로세스
```
1. DM 문의
2. 간단한 안내 (사용 방법, 가입 링크)
3. 계정 생성 지원
4. 온보딩 체크리스트 공유
```

## 운영 원칙

### Do
- 실제 사용 화면 중심 (진정성)
- 교리교사 관점의 언어 사용
- DM 24시간 내 응답
- 댓글 적극 소통

### Don't
- 과장된 마케팅 문구
- 너무 잦은 게시 (스팸 느낌)
- 개인정보 노출 (학생 이름 등)
- 타 서비스 비방
