# PG 선정 인터뷰 가이드라인

> 메인 보고서: [pg-subscription.md](./pg-subscription.md)

## 대상 (총 6명)

- 교리교사 얼리어답터 3명: 가재울·구룡·옥수동 중 선택
- 교리교사 신규 도입 2명: 도입 3개월 이내, 얼리어답터 편향 보정용
- 사제 1명: 5/6 협력 신부 미팅에 결제 어젠다 통합

## 결제 주체 질문

1. "지금 이 도구에 비용이 생긴다면 본인이 직접 결제하겠는가, 본당 예산으로 처리하겠는가?"
2. "본당 예산 처리 시 누구에게 허락을 받아야 하는가? 얼마나 걸리는가?"
3. "개인 카드로 먼저 결제 후 정산받는 것이 가능한가? 번거롭지 않은가?"

## 결제 수단 질문

4. "월 5,000원을 어떤 방식으로 결제하시겠는가? (카드 자동이체 / 계좌이체 / 카카오페이 등)"
5. "사제/사무국 추가 요청이 필요하면 결제 의향이 바뀌는가?"

## 영수증 질문 (가톨릭 본당 회계 문화 고려, 2026-04-30 추가)

6. "본당 사무실/사제께 결제 후 정산받을 때 어떤 형태의 영수증·증빙이 필요한가? (현금영수증 / 세금계산서 / 카드 매출전표 / 별도 양식)"
7. "본당이 사업자등록번호를 보유하고 있는가? 세금계산서 수취 가능한 법인 형태인가?"
8. "영수증·증빙이 자동 이메일/PDF로 발행되면 사후 정산 절차가 단축되는가?"

## WTP 실결제 테스트 (인터뷰 종료 직전)

9. "지금 바로 1개월치 5,000원 결제해주시겠어요? 실 결제이며 즉시 해지 가능합니다."

> 의향-행동 갭(20~50%) 보정용. CTA 사전 예약 100건보다 신뢰도 높은 WTP 데이터를 만든다.

## 사제 어젠다 (5/6 미팅)

- 본당 예산으로 월 5,000원 결제하는 결재 프로세스의 현실성
- 교리교사 개인 카드 선결제 후 사후 정산 구조 충돌 여부
- 사제가 직접 결제 주체가 되는 케이스 가능성
- **본당 회계 처리상 어떤 영수증·증빙이 필수인가** (2026-04-30 추가)

## 인터뷰 일정 고려사항

- R18 시험기간 가설: 4월 중간고사(통상 4/14~4/25) 직후 5월 초 일정 우선
- 5/6 사제 미팅 활용으로 사제 인터뷰 추가 리드타임 0
- 노쇼 리스케줄링 포함 실 리드타임 2~3주 예상
