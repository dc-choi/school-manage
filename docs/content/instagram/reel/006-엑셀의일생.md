# 엑셀 출석부의 일생

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 스토리텔링 / 밈

## 콘셉트
엑셀 파일이 카톡으로 날아다니다가 "최종_최종_진짜최종.xlsx"가 되는 과정. 마지막에 "...그래서 만들었습니다"

## 구성

### 장면 1 (0~3초)
- 화면: 엑셀 파일 "출석부.xlsx"
- 자막: "3월. 새 출석부를 만들었다"

### 장면 2 (3~6초)
- 카톡으로 파일 공유
- 자막: "카톡으로 보냈다"

### 장면 3 (6~9초)
- 파일 이름 변화:
  - "출석부_수정.xlsx"
  - "출석부_최종.xlsx"
  - "출석부_최종_수정.xlsx"
- 자막: "버전이 늘어났다"

### 장면 4 (9~12초)
- "출석부_최종_진짜최종_이거씀.xlsx"
- 자막: "...어느 게 최신이지?"

### 엔딩 (12~15초)
- 검은 화면
- 자막: "...그래서 만들었습니다"

## 캡션
```
출석부_최종_진짜최종_이거씀.xlsx
...이런 적 있으시죠?
그래서 만들었습니다.
```
