# 사용자 0명에서 시작한 출석부 앱

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 빌드 인 퍼블릭 / 브이로그

## 콘셉트
개발자이자 교리교사인 사람이 직접 만든 출석부 앱 스토리. 코딩 화면 + 자막 스토리텔링.

## 구성

### 장면 1 (0~3초)
- 코딩 화면 (또는 노트북 타이핑)
- 자막: "교리교사인 제가"

### 장면 2 (3~6초)
- 엑셀 출석부 화면
- 자막: "매주 엑셀 출석부가 불편해서"

### 장면 3 (6~9초)
- 코딩 화면
- 자막: "직접 만들었습니다"

### 장면 4 (9~12초)
- 앱 화면 빠르게 보여주기 (달력, 출석, 통계)
- 자막: "출석, 축일, 통계까지 한곳에"

### 엔딩 (12~15초)
- 카메라 응시 또는 성당 배경
- 자막: "지금 5곳에서 쓰고 있습니다"

## 캡션
```
교리교사가 직접 만든 출석부 앱.
매주 일요일, 이거 하나 열면 됩니다.
프로필 링크에서 확인해 보세요.
```

## 해시태그
```
#교리교사 #주일학교 #출석부 #빌드인퍼블릭 #가톨릭 #성당 #교리교사일상 #위클리스쿨 #개발자
```
