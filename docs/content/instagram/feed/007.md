# 공감 콘텐츠 - "은서야~ 저 은지인데요"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
솔직히 한 번쯤은 다 틀려봤잖아요.
은서 은지 은수... 미안해 얘들아 다음 주엔 꼭 맞출게.
나만 그런 거 아니죠?
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "은서야~ 오늘 성경 읽어볼까?"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 교실에서 밝은 미소로 학생을 가리키며 말을 거는 장면. 자신감 넘치는 표정. 말풍선에 "은서야~ 오늘 성경 읽어볼까?". 앞에 여학생 캐릭터가 앉아 있다. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: 학생 "...저 은지인데요"

```
같은 캐릭터 스타일 유지. 앞에 앉은 여학생 캐릭터가 손을 살짝 들며 무표정하게 정정하는 장면. 말풍선에 "...저 은지인데요". 교리교사 캐릭터가 옆에서 당황한 표정으로 얼굴이 빨개진 모습. 이마에 땀방울 하나. 4컷 만화 중 2컷 스타일, 민망한 순간을 코믹하게 표현.
```

### 3컷 (전)
- 대사: 교리교사 "아 맞다 미안! 은지야~" / 학생 "............저 은수인데요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 손을 모으며 황급히 사과하는 장면(왼쪽). 말풍선에 "아 맞다 미안! 은지야~". 여학생 캐릭터가 눈을 반쯤 감고 한숨 섞인 무표정으로 대답(오른쪽). 말풍선에 "............저 은수인데요". 교리교사 머리 위에 물음표와 느낌표가 동시에 떠 있다. 충격받은 표정. 4컷 만화 중 3컷 스타일, 두 번째 실수의 충격을 극대화.
```

### 4컷 (결)
- 대사: (교리교사 영혼이 빠져나간 표정) "...다음 주엔 꼭 맞출게"
```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 책상에 엎드려 영혼이 빠져나간 듯한 표정. 머리 위에 유령처럼 혼이 빠져나가는 이펙트. 말풍선에 "...다음 주엔 꼭 맞출게". 옆에 학생들이 킥킥 웃고 있는 모습이 작게 보인다. 4컷 만화 중 4컷(마무리) 스타일, 코믹한 탈진과 가벼운 여운.
```
