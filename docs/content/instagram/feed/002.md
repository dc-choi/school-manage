# 기능 소개 - "출석상 줄 애, 누구더라?"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
출석상 줄 때 엑셀 뒤져가며 세어본 적 있으시죠?
우수 출석 멤버가 자동으로 정리됩니다.
프로필 링크에서 확인하세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "출석상 대상자 명단 이번 주까지 정리해 주세요~"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 카카오톡 메시지를 보고 있는 장면. 스마트폰 화면에 말풍선 "출석상 대상자 명단 이번 주까지 정리해 주세요~" 메시지가 보인다. 교리교사 표정은 평온. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "어... 한 명씩 세야 하나?" / (엑셀 앞에서 멘붕)

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 노트북 앞에서 머리를 감싸 쥐고 있는 장면. 화면에 엑셀 시트가 보이고, 출석 O/X가 빼곡하게 적혀 있다. 한 손에 계산기를 들고 있다. 말풍선에 "어... 한 명씩 세야 하나?". 머리 위에 소용돌이. 이마에 땀방울. 4컷 만화 중 2컷 스타일, 코믹하게 막막한 표정 강조.
```

### 3컷 (전)
- 대사: "잠깐, 여기 순위가 다 나오잖아?"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 눈이 반짝이며 스마트폰 화면을 바라보는 장면. 스마트폰 화면에 출석 우수 학생 순위 리스트가 보인다 (1등 OOO 95%, 2등 OOO 90%, 3등 OOO 88% 형태). 말풍선에 "잠깐, 여기 순위가 다 나오잖아?". 캐릭터 머리 위에 반짝이는 별 이펙트. 4컷 만화 중 3컷 스타일, 발견의 기쁨을 표현.
```

### 4컷 (결)
- 대사: "3초 만에 끝났다..."
- 하단: "출석 우수 멤버 자동 순위 / 출석률 자동 계산" + "프로필 링크에서 시작하세요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 의자에 기대앉아 편안하고 뿌듯한 표정. 한 손에 스마트폰, 다른 손은 브이(V) 포즈. 말풍선에 "3초 만에 끝났다...". 주변에 꽃잎/별 이펙트. 하단에 텍스트 박스: "출석 우수 멤버 자동 순위 / 출석률 자동 계산" 그리고 작은 글씨로 "프로필 링크에서 시작하세요". 4컷 만화 중 4컷(마무리) 스타일, 밝고 시원한 톤.
```
