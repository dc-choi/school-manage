# Content Templates — 피드 카드뉴스

피드 전용 카드뉴스 형식 템플릿입니다.

> 역할/톤/규칙은 `rules/content.md` 참조
> 릴스: `rules/content-templates-reel.md` / 통합: `rules/content-templates-combined.md`

> **핵심 원칙**: 모든 복붙 대상(캡션, 해시태그, 프롬프트)은 코드블록(```)으로 감싸서 바로 복사할 수 있게 작성한다.

## 인스타그램 피드 (카드뉴스) — 기본 형식

> 4컷 만화는 가시성 피드백으로 중단됨 (02-26). 신규 피드는 카드뉴스 형식으로 제작한다.

````markdown
# [제목]

## 메타

| 항목    | 내용                           |
| ------- | ------------------------------ |
| 유형    | 피드 게시물 (카드뉴스, 캐러셀) |
| 타겟    | [교리교사 / 모임 운영자]       |
| CTA     | [프로필 링크 / DM]             |
| 카드 수 | [N장]                          |
| 사이즈  | 1080×1080px (1:1 정사각형)     |

### 공통 스타일 프롬프트

```
[전체 스타일, 배경, 색감, 폰트 분위기 등 기본 설정. 나노바나나에 먼저 입력. 비율은 1:1 정사각형으로 지정. 반드시 "풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것." 문구를 포함할 것.]
```

---

## 카드 구성

```

```

```

```

### Card 2 — [주제]

```

```

```

```

```

```
````

### 카드뉴스 유형별 가이드

| 유형                 | 카드 수 | 구성                                     | 참고                    |
| -------------------- | ------- | ---------------------------------------- | ----------------------- |
| 기능 소개 / 업데이트 | 3~5장   | 표지 → Before/After 또는 기능 설명 → CTA | 실제 스크린샷 활용 권장 |
| 사용자 인터뷰        | 8~10장  | 표지 → Q&A 슬라이드 → CTA                | 019번 참고              |
| 이벤트 공지/결과     | 4~5장   | 표지 → 내용 → 대상자 → 마무리            | 020번 참고              |

> 피드 + 릴스 양쪽 게시 의도면 통합 형식 사용 → `rules/content-templates-combined.md`
