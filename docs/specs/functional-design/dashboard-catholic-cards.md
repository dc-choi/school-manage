# 기능 설계: 대시보드 가톨릭 카드 통일

> 비기능적 요구사항 (UI). 대시보드의 전례력 카드와 축일자 카드를 시각적으로 통일합니다.

## 연결 문서

- 기능 설계: `docs/specs/functional-design/liturgical-calendar.md`
- 기능 설계: `docs/specs/functional-design/student-management.md` (이달의 축일자 섹션)

## 기능 범위

| 기능 | 설명 | 상태 |
|------|------|------|
| 카드 배치 통일 | PatronFeastCard를 LiturgicalSeasonCard 옆으로 이동 | 변경 |
| 카드 스타일 통일 | 두 카드의 구조/간격/로딩 UI 일관성 맞춤 | 변경 |

---

## 현재 상태 (AS-IS)

### 레이아웃

```
┌─────────────────────────────────────────────────┐
│ [날짜선택]   [전례력 카드 ──────────── flex-1]   │  ← 상단
├─────────────────────────────────────────────────┤
│ [학년별 통계 테이블 ──────── full width]          │  ← 중단
├─────────────────────────────────────────────────┤
│ [축일자 카드] [성별 분포] [우수 출석]              │  ← 하단 3칼럼
└─────────────────────────────────────────────────┘
```

### 문제

1. **위치 분산**: 전례력과 축일자 모두 가톨릭 특화 정보인데 상단/하단으로 떨어져 있음
2. **스타일 불일치**: 전례력 카드는 헤더 없이 `p-4`, 축일자 카드는 `CardHeader`/`CardContent` 사용
3. **로딩 UI 불일치**: 전례력은 `Loader2` (lucide), 축일자는 `LoadingSpinner` (커스텀)

## 변경 후 (TO-BE)

### 레이아웃

```
┌──────────────────────────────────────────────────────┐
│ [날짜선택]   [전례력 카드 ─── flex-1] [축일자 카드 ─── flex-1] │  ← 상단
├──────────────────────────────────────────────────────┤
│ [학년별 통계 테이블 ──────── full width]                │  ← 중단
├──────────────────────────────────────────────────────┤
│ [성별 분포]              [우수 출석]                    │  ← 하단 2칼럼
└──────────────────────────────────────────────────────┘
```

### 변경 사항

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 축일자 카드 위치 | 하단 3칼럼 그리드 첫 번째 | 상단 날짜선택 옆, 전례력 카드와 나란히 |
| 축일자 카드 구조 | CardHeader + CardContent | 전례력 카드와 동일한 구조 (p-4, 헤더 없음) |
| 축일자 카드 제목 | CardTitle 사용 | 서브 제목 스타일 (muted-foreground) |
| 축일자 로딩 UI | LoadingSpinner (커스텀) | Loader2 (lucide) — 전례력과 동일 |
| 하단 그리드 | 3칼럼 (md:2, lg:3) | 2칼럼 (md:2) |

## 예외/엣지 케이스

| 상황 | 처리 |
|------|------|

## 테스트 시나리오

### 정상 케이스

1. **TC-1**: 대시보드 진입 시 전례력 카드와 축일자 카드가 같은 행에 나란히 표시
2. **TC-2**: 하단 그리드가 2칼럼(성별 분포 + 우수 출석)으로 표시
3. **TC-3**: 모바일에서 카드가 세로로 정상 쌓임

### 예외 케이스

1. **TC-E1**: 전례력/축일자 중 하나가 에러 시 나머지 카드 정상 표시
