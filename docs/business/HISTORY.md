> **03-01 데이터 재확인**: 금호동성당(그룹 1, 학생 0) + 금호동성당 청소년부(그룹 0, 학생 0) 2개 계정 존재. 한 단체에서 여러 명 개별 가입 패턴 재확인

- **방향**: 개인 계정 + 본당/모임 합류 구조로 전환 → 계정 모델 전환 참고
