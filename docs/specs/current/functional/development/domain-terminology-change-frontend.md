`apps/web/src/` 내 20개 파일의 한국어 UI 라벨을 치환합니다. 코드 로직, API, URL, DB 변경 없음.

## 구현 시 주의사항

- [ ] 문자열 치환 시 코드 주석(`//`)은 변경하지 않음
- [ ] 변수명(`group`, `student`, `member`)은 변경하지 않음
- [ ] GA4 이벤트 함수명/이벤트명은 변경하지 않음
- [ ] `StudentForm.tsx`의 validation 에러 메시지 중 "그룹"→"학년"도 포함
- [ ] 개인정보 처리방침(`PrivacyPolicyDialog.tsx`)은 이미 "학생" 사용 중이므로 변경 불필요
