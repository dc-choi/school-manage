# 브레인스토밍 결과 보고서

## 주제

GTM(Go-To-Market) 전략 — 주일학교 출석 관리 서비스 대규모 상용화

## 토론 요약

3라운드 토론을 통해 초기 3개 아이디어가 검증-실행 중심으로 재구성됨.

**핵심 전환점**:
- Round 1: WAU 전환율 22% 현실 반영, D유형 1:1 통화 선행 합의
- Round 2: GA4 데이터를 주 분석수단으로 격상, 통화는 보조
- Round 3: IDOR 병렬 착수, 앰배서더 타이밍 조건, Import 트리거 구체화

**최종 합의**: 지금은 정릉4동을 살리고 IDOR을 막는 것이 먼저. 상용화는 PMF 달성 후.

## 아이디어 목록

### 비공식 앰배서더

- 한 줄 요약: S유형 파워유저의 자발적 추천 행동을 구조화
- 사용자 가치: 신뢰 기반 구전 → 교리교사 네트워크 확산
- 실현 방법: 구룡성당/가재울 통화 → 소개 의향 확인 → 가입 링크 + 메시지 템플릿 제공
- 차별점: 가입 경로 추적 가능, 모임 합류 구조로 추천-전환 연결
- 사전 조건: 구룡성당 초등부 계정 이슈 해소 후 실행

### 교구 비공식 접점

- 한 줄 요약: 교구 연수 일정 확인 후 조건부 비공식 접촉
- 사용자 가치: 교리교사 집합 시점에서 니즈 매칭
- 실현 방법: 연수 일정 확인(30분) → 있으면 소개 자료 준비 + 비공식 대화
- 차별점: 가톨릭 교구 연수 채널은 이 서비스만 접근 가능
- 사전 조건: 연수 일정 존재 + IDOR 해소 + 법률 자문 완료(공식화 시)

### 원인 규명 선행 + 조건부 Import

- 한 줄 요약: GA4 이탈 데이터 → 통화 보완 → 조건부 Import 개발
- 사용자 가치: 이탈 원인 파악으로 퍼널 개선의 정확도 향상
- 실현 방법: GA4 분석 → 정릉4동/흑석동 통화 → Import 트리거 충족 시 개발
- 차별점: GA4 26개 이벤트 기반 자사 이탈 분석 (엑셀/밴드 불가)
- Import 착수 트리거: 학생 20명+ & 출석 0건 계정 3곳+ OR 통화 2곳+ "엑셀 옮기고 싶다"

## 최종 실행 순서

### 이번 주 (병렬)

1. GA4 이탈 지점 데이터 확인 (1~2시간)
2. 정릉4동 팔로업 통화 (1시간) — 이탈 위험 최긴급
3. 흑석동 팔로업 통화
4. 구룡성당 초등부 계정 이슈 확인 통화
5. IDOR SDD 착수 — GTM과 병렬

### 다음 주

6. 교구 연수 일정 확인 (30분)
7. 구룡성당 초등부 이슈 해소 확인 후 앰배서더 제안

### 이번 달 내

8. Import 착수 트리거 확인 — 중림동약현/성남동/보라동 팔로업
9. 돈암동 3월 팔로업

## 상용화 로드맵

### 1단계: PMF 검증 (현재, ~5월)

- 전환 조건: WAU 15곳, IDOR 해소, S유형 4주+ 지속 60%+
- 병렬: 유료 전환 의향 비공식 인터뷰, 결제 주체 가설 검증

### 2단계: 상용화 준비 (5~8월)

- 진입 조건: PMF 달성 + 유료 의향 50%+ + 법률 자문 + Refresh Token
- 할 것: 가격 구조화, 유료 UI 설계, NFC 파일럿

### 3단계: 상용화 실행 (8월~)

- 진입 조건: NFC 파일럿 지불 확인 + WAU 20곳+ + 법률 완료
- 할 것: 유료 출시, 앰배서더 공식화, 교구 채널 공식 접근
