# 브레인스토밍 결과 보고서

## 주제
인스타그램 전략 재검토 — 파일럿 17곳 중 13곳(76%)이 인스타 경유 유입이었음이 확인됨. 02-26 GTM 전략에서 "인스타 역할을 네트워크 착지점으로 축소, 주간 루틴 폐지"로 결정했으나, 유입 경로 재분류 결과 인스타가 핵심 획득 채널. 적정 운영 수준 재검토.

## 신규 데이터 — 유입 경로 재분류 (사용자 직접 제공)

| 경로 | 본당 | 수 |
|------|------|---|
| 직접 관계 | 장위동, 정릉4동, 성북동 | 3곳 |
| 순수 인스타 | 고덕동, 돈암동, 신당동, 신정동, 금호동, 가재울, 창동, 성남동 | 8곳 |
| 지인 + 인스타 경유 | 한강성당, 흑석동, 길음동, 과천, 양재2동 | 5곳 |

→ **인스타 관여 비율: 17곳 중 13곳 (76%)**

## 토론 요약

### Round 1
- 기획자 3개 아이디어 제안: (1) 저비용 정기 피드 — 주 1회 스크린샷 (2) 교구/지구 단위 태그 전략 (3) 격주 실사용 증거 게시
- 비판자 핵심 지적: 76% 데이터의 인과관계 한계(상관≠인과, 17곳은 통계적 유의성 부족), 스크린샷은 낮은 신뢰도, 해시태그 효과 감소, 교구 회장 리그램 요청은 관계 자본 소비, 파일럿 화면 수집 시 개인정보 리스크
- 합의: 76% 데이터 한계 인정, 4컷 만화 중단 유효(형식 문제), 동시 실행보다 우선순위 선별 필요

### Round 2
- 기획자 수정: (1) 스크린샷→전례 시점 가톨릭 특화 기능 피드, 월 1~2회 (2) 리그램 요청→자발적 공유 키트, 온보딩 설문으로 이동 (3) 격주→분기 1회 재게시, 파일럿 화면→텍스트 인터뷰, 개인정보 동의 명시
- 비판자 핵심 지적: 하이라이트 업로드는 법률 자문 완료 전 불가(리스크 15), 공유 경로 불명확(인스타 vs 카카오톡), DM CTA 측정 불가, 교구 회장 단수 의존
- 합의: 아이디어 2 즉시 실행(UTM 포함), 아이디어 1 월 1~2회 수용, 총 시간 Track A 미침해

### Round 3
- 기획자 최종: 법률 자문 완료 후 하이라이트 업로드(수용), 카카오톡+인스타 이중 경로+UTM, 교구 회장+장위동 레퍼런스로 공유 주체 확장, 이중 CTA(링크 1순위+DM 보조), 사순시기 포기→부활절 확정
- 비판자 최종: 부활절 피드의 수요 가정 미검증, 공유 주체 확장이 "가능성"에 머무름(트리거 액션 필요), 4월 일정 집중(계정 모델+법률 자문+하이라이트), 기존 사용자 소급 동의 누락
- 최종 합의: 조건부 합의 2건 포함 전체 합의 도달

## 아이디어 목록

### [아이디어 1] 가톨릭 특화 기능 증거 피드 (전례 시점 연동)

- 한 줄 요약: 전례 시점(부활절, 성탄절 등)에 맞춰 가톨릭 특화 기능(전례 시기 카드, 세례명 축일자, 의무축일)이 작동하는 화면을 월 1~2회 게시
- 원래 형태: 주 1회 스크린샷 피드 → 전례 시점 연동 월 1~2회로 변경
- 사용자 가치: 교리교사가 계정 방문 시 "가톨릭 전용 도구"라는 차별점을 5초 내 인식
- 실행 방법:
  - 전례력 주요 시점에 해당 기능 화면 게시
  - 캡션에 교리교사 언어로 설명
  - 댓글 고정 링크(UTM 측정) 1순위 + DM 보조
  - DM 동일 질문 3건 반복 시 FAQ 피드 전환
- 차별점: 엑셀, 밴드, 처치커뮤니티 모두 전례 시기 연동 기능이 없음
- 시간: 2h (1회성, 3월 3주차 제작 / 4월 5일 부활절 게시)
- 판정: **승인**

### [아이디어 2] 교구 회장 자발적 공유 지원 키트

- 한 줄 요약: 이미 자발적으로 홍보 의향을 보인 교구 회장(+ 장위동 레퍼런스 사용자)이 공유하기 쉽게 콘텐츠 키트를 제공
- 원래 형태: 교구/지구 해시태그 전략 → 자발적 공유 키트로 전환
- 사용자 가치: 교구 회장이 "뭘 공유하면 되는지" 모르는 문제 해결, 공유받는 교리교사가 착지점에서 서비스 이해 가능
- 실행 방법:
  - 카카오톡 + 인스타그램 이중 경로 키트 제작
  - UTM 단축 URL 2개 생성 (카카오: utm_source=kakao, 인스타: utm_source=instagram)
  - 공유 주체: 교구 회장 + 장위동 레퍼런스 사용자 (2인 이상)
  - 안착 확인 본당(가재울/창동/길음동) → 완전 안착 시 공유 요청 DM 1회 발송 (트리거 액션)
- 차별점: 밴드, 카카오톡 오픈채팅으로는 이런 네트워크 확산 구조 불가
- 시간: 1~2h (1회성)
- 판정: **승인 (최우선)**

### [아이디어 3] 인터뷰 카드뉴스 활용 신뢰 구조 (하이라이트 고정)

- 한 줄 요약: 제작 완료된 사용자 인터뷰 카드뉴스를 하이라이트 "실사용 후기"에 고정하고 분기 1회 재게시하여 착지점 신뢰도 유지
- 원래 형태: 격주 1회 실사용 증거 게시 → 분기 1회 재게시 + 하이라이트 고정
- 사용자 가치: 프로필 방문 시 "진짜 쓰는 사람이 있다"는 신뢰 확보
- 실행 방법:
  - 콘텐츠 설계(초안) 3~4월 진행 (개인정보 무관)
  - 법률 자문 + 동의 절차 구현 완료 후 하이라이트 업로드 (4월)
  - 기존 인터뷰 카드뉴스 하이라이트 고정
  - 신규 인터뷰는 텍스트만 수집, 개인정보 동의 후 게시
  - 분기 1회 재게시 시 캡션 변형 (알고리즘 패널티 방지)
- 차별점: 가톨릭 현장 교사가 직접 말하는 실사용 증거는 이 채널에서만 가능
- 시간: 1h (설계, 1회성) + 업로드 0h (설계 완료 후)
- 판정: **조건부 승인** (법률 자문 완료 전제)
