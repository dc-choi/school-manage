# 학생 이름 부르기 챌린지

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 없음 (순수 공감)
- 스타일: 챌린지/ASMR

## 콘셉트
은서/은지/은수/은채를 빠르게 부르다가 점점 꼬이는 모습. 자막: "3년차인데 아직도 이럼"

## 구성

### 장면 1 (0~3초)
- 자신감 넘치는 표정
- 자막: "출석 부를게~"

### 장면 2 (3~7초)
- "은서!" "은지!" "은수!" 빠르게 부르기
- 점점 빨라지는 속도

### 장면 3 (7~10초)
- "은...채? 은서? 아니 은지..."
- 당황하는 표정, 말 꼬이기 시작

### 장면 4 (10~13초)
- 완전히 꼬여서 멈춤
- 학생: "선생님 저 은수요"

### 엔딩 (13~15초)
- 카메라 응시
- 자막: "3년차인데 아직도 이럼"

## 캡션
```
은서 은지 은수 은채... 미안해 얘들아.
3년차인데 아직도 헷갈려요.
나만 그런 거 아니죠?
```

- 또는: 실패 효과음 (탈락 사운드)
