# 기능 소개 - "이제 핸드폰에서도 편하게 쓸 수 있어요"

## 메타

| 항목 | 내용 |
|------|------|
| 유형 | 피드 게시물 (카드뉴스, 캐러셀) |
| 타겟 | 교리교사 |
| CTA | 프로필 링크 |
| 카드 수 | 4장 |
| 사이즈 | 1080×1080px (1:1 정사각형) |

### 공통 스타일 프롬프트
```
카드뉴스 스타일. 깔끔한 흰색 배경에 부드러운 파스텔톤 포인트 컬러(연한 민트 또는 연보라). 둥글둥근 귀여운 일러스트 아이콘 포함. 인스타그램 1:1 정사각형 비율. 헤더 텍스트는 크고 굵은 서체, 본문은 읽기 쉬운 서체. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 카드 구성

### Card 1 — 표지

**메인 카피**:
```
핸드폰으로 출석 체크할 때
화면이 잘린 적 있으시죠?
```

**서브 카피**:
```
모바일 화면이 새로워졌습니다
```

```
카드뉴스 표지. 깔끔한 흰색 배경에 연한 민트색 포인트 컬러. 상단에 서비스 로고(주일학교 출석부). 중앙에 크고 굵은 한국어 텍스트 "핸드폰으로 출석 체크할 때 화면이 잘린 적 있으시죠?". 하단에 작은 텍스트 "모바일 화면이 새로워졌습니다". 스마트폰을 들고 있는 손 일러스트가 한쪽에 배치되어 있고, 스마트폰 화면에 출석부 UI가 살짝 보인다. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

---

### Card 2 — Before (이전 모바일 화면)

**헤더**:
```
Before
```

**본문**:
```
사이드바가 화면을 차지하고
테이블이 잘려서 이름이 안 보이고
글자가 너무 작아서 확대해야 했어요
```

```
카드뉴스 Before 슬라이드. 깔끔한 흰색 배경. 상단에 크고 굵은 텍스트 "Before"가 회색 톤으로 표시. 중앙에 스마트폰 목업이 있고, 화면 안에 깨진 레이아웃이 보인다 — 왼쪽에 좁은 사이드바가 화면의 4분의 1을 차지하고, 오른쪽 테이블의 컬럼들이 화면 밖으로 잘려 나가 있다. 문제 포인트에 빨간 동그라미와 화살표 3개: (1) 사이드바를 가리키며 "사이드바가 화면을 차지", (2) 잘린 테이블을 가리키며 "이름이 안 보여", (3) 작은 글씨를 가리키며 "글자가 너무 작아". 전체적으로 약간 어둡고 답답한 회색 톤 분위기. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

---

### Card 3 — After (개선된 모바일 화면)

**헤더**:
```
After
```

**본문**:
```
메뉴는 햄버거 버튼으로 깔끔하게
테이블은 옆으로 스크롤할 수 있고
글자 크기도 모바일에 맞게 조정했어요
```

```
카드뉴스 After 슬라이드. 깔끔한 흰색 배경. 상단에 크고 굵은 텍스트 "After"가 민트색으로 밝게 표시. 중앙에 스마트폰 목업이 있고, 화면 안에 깔끔하게 정리된 모바일 레이아웃이 보인다 — 사이드바 없이 상단에 햄버거 메뉴 아이콘(≡)과 페이지 제목이 있고, 테이블이 화면 너비에 맞게 정렬되어 있으며 하단에 가로 스크롤 바가 보인다. 개선 포인트에 초록색 체크마크 3개: (1) 햄버거 메뉴를 가리키며 "깔끔한 메뉴", (2) 테이블을 가리키며 "가로 스크롤", (3) 텍스트를 가리키며 "읽기 쉬운 글자". 전체적으로 밝고 깨끗한 분위기, 주변에 작은 반짝이 이펙트. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```
