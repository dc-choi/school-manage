| 항목 | 내용 |

| 타겟 | 교리교사 |
| CTA | 프로필 링크 |
| 카드 수 | 5장 |
| 사이즈 | 1080×1080px (1:1 정사각형) |

### 공통 스타일 프롬프트
```
카드뉴스 스타일. 깔끔한 흰색 배경에 부드러운 파스텔톤 포인트 컬러(연한 하늘색과 연한 노란색 조합). 둥글둥근 귀여운 일러스트 아이콘 포함. 인스타그램 1:1 정사각형 비율. 헤더 텍스트는 크고 굵은 서체, 본문은 읽기 쉬운 서체. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 카드 구성

### Card 1 — 표지

**메인 카피**:
```
학기 초 준비,
아직 명단 정리 중이신가요?
```

**서브 카피**:
```
엑셀 업로드 + 등록 관리, 두 가지가 한꺼번에 업데이트됐어요
```

```
카드뉴스 표지. 깔끔한 흰색 배경에 연한 하늘색 포인트 컬러. 상단에 서비스 로고(주일학교 출석부). 중앙에 크고 굵은 한국어 텍스트 "학기 초 준비, 아직 명단 정리 중이신가요?". 하단에 작은 텍스트 "엑셀 업로드 + 등록 관리, 두 가지가 한꺼번에 업데이트됐어요". 왼쪽 아래에 체크리스트와 연필 일러스트, 오른쪽 아래에 노트북과 엑셀 시트 일러스트가 귀엽게 배치. 봄 느낌의 작은 꽃잎 장식이 모서리에 살짝 흩뿌려져 있다. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

---

### Card 2 — 기능 1: 엑셀 Import

**헤더**:
```
엑셀로 학생 한 번에 등록
```

**본문**:
```
기존 명단 파일 그대로 업로드하면
이름, 세례명, 학년까지 한꺼번에 등록돼요

1명씩 입력하던 시간, 이제 아낄 수 있어요
```

```
카드뉴스 기능 소개 슬라이드. 깔끔한 흰색 배경에 연한 하늘색 포인트 컬러. 상단에 크고 굵은 한국어 텍스트 "엑셀로 학생 한 번에 등록". 중앙에 큰 스마트폰 또는 모니터 목업이 있고, 화면 안에 엑셀 시트에서 학생 목록이 업로드되는 장면이 보인다. 엑셀 파일 아이콘에서 화살표가 스마트폰 화면의 학생 목록으로 이어지는 흐름 일러스트. 하단에 본문 텍스트 "기존 명단 파일 그대로 업로드하면 이름, 세례명, 학년까지 한꺼번에 등록돼요. 1명씩 입력하던 시간, 이제 아낄 수 있어요". 전체적으로 밝고 깔끔한 분위기. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

---

### Card 3 — 기능 2: 등록 관리

**헤더**:
```
등록 여부도 바로 관리
```

**본문**:
```
학생 목록에서 체크만 하면 등록 완료
등록/미등록 필터로 현황을 한눈에 파악할 수 있어요

종이 명단이랑 따로 관리할 필요 없어요
```

```
카드뉴스 기능 소개 슬라이드. 깔끔한 흰색 배경에 연한 노란색 포인트 컬러. 상단에 크고 굵은 한국어 텍스트 "등록 여부도 바로 관리". 중앙에 스마트폰 목업이 있고, 화면 안에 학생 목록이 보인다 — 체크박스가 있는 학생 이름 리스트에서 일부가 체크되어 있고, 상단에 "등록 15명 / 미등록 3명" 요약 배지가 있다. 하단에 본문 텍스트 "학생 목록에서 체크만 하면 등록 완료. 등록/미등록 필터로 현황을 한눈에 파악할 수 있어요. 종이 명단이랑 따로 관리할 필요 없어요". 체크리스트와 돋보기 아이콘이 귀엽게 배치. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

### Card 4 — 대시보드 등록 통계

**헤더**:
```
대시보드에서 등록 현황까지
```

**본문**:
```
학년별 통계 테이블에 등록 인원이 함께 표시돼요
총 인원, 등록 인원, 출석 평균을 한눈에 볼 수 있어요
```

```
카드뉴스 기능 소개 슬라이드. 깔끔한 흰색 배경에 연한 하늘색과 연한 노란색이 조합된 포인트 컬러. 상단에 크고 굵은 한국어 텍스트 "대시보드에서 등록 현황까지". 중앙에 대시보드 화면 목업이 있고, 깔끔한 테이블이 보인다 — 컬럼 헤더에 "학년명 | 총 인원 | 등록 인원 | 주간 평균 | 월간 평균"이 있고 행에 "중1 | 20명 | 18명 | 15명 | 14명", "중2 | 15명 | 14명 | 12명 | 11명", 마지막에 "총계 | 35명 | 32명 | 27명 | 25명"이 표시된 형태. 테이블 행이 번갈아 연한 색으로 구분된다. 하단에 본문 텍스트 "학년별 통계 테이블에 등록 인원이 함께 표시돼요. 총 인원, 등록 인원, 출석 평균을 한눈에 볼 수 있어요". 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

```

1. 엑셀 업로드: 기존 명단 파일 그대로 올리면 학생이 한꺼번에 등록돼요.

종이 명단이랑 따로 관리하던 걸, 이제 한곳에서 해보세요.

프로필 링크에서 무료로 시작할 수 있습니다.
```
