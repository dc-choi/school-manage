# 페르소나 매트릭스

타겟 사용자/잠재 고객 추출용 페르소나 분류. 단체 타입(4) × 운영/의사결정 페르소나(6) 매트릭스 결합으로 결제 전환 모수 우선순위를 도출한다.

## 분류 축

- **1차 — 단체 타입**: 초등부 / 중고등부 / 청년(YA) / 군종 (4)
- **2차 — 운영 행동**: P1 적극 안착자 / P2 사후 일괄 / P3 도입 후 정지 (3)
- **3차 — 인구/의사결정**: P4 대학생 봉사직 / P5 분석가형 / P6 회의적 신중파 (3)

## 데이터 출처

- 페르소나 분류 당시 파일럿 스냅샷(2026-05-09) + `STATUS.md`. MAO/본당/모임의 현재 수치는 복제하지 않고 `STATUS.md`를 단일 출처로 사용
- 2026-05-08 카톡 인터뷰 4명(장현도/김유빈/김서현/이한별)
- 2026-03-27 군종교구 목포해군성당 DM 1건
- R18/R24/R25 분기 결과

## 운영 행동 페르소나 (P1/P2/P3)

- **P1 적극 안착자 (S유형)**: 즉시 실시간 입력, 1주 주기 안정. 출석/통계 가치 우선. 결제 전환 직접 모수
- **P2 사후 일괄 입력자 (입력 지연 축)**: 매주 출석 진행 + 입력만 사후 백입력. R24-A0/A1과 별도 분류하고 확정 코호트만 측정한다
- **P3 도입 후 정지자 (R24-A1)**: 첫 출석 후 후속 사용이 이어지지 않은 후보. 입력 지연을 배제한 확정 코호트만 결제 모수에서 제외한다

## 인구/의사결정 페르소나 (P4/P5/P6)

- **P4 대학생 봉사직 (R18)**: 학사 일정에 따라 주기적 사용 저하 (시험기간 4/6/10/12월). 가격 민감. 중고등부 직격
- **P5 분석가형 디지털 친화**: 단체 타입 무관, 다계정 운영 단체에 분포. 가치 체감 시 적극 결제 / 미완성도엔 박함
- **P6 회의적 신중파**: "강제성/필요성 검증 우선", 카톡 단톡방 이미 활용. 차별 가치만 결제. 데이터 1건(이한별)으로 일반화 한계

## 장기 SaaS 잠재 고객 우선순위, 현재 비활성

### 🥇 1순위 — 결제 전환 직접 모수 (P1 단체)

| #   | 잠재 고객 셀                  | 시그니처                  | 결제 자금원           | 모수 상태 | 액션                                                      |
| --- | ----------------------------- | ------------------------- | --------------------- | --------- | --------------------------------------------------------- |
| A   | **초등부 × P1 + 학부모 알림** | 알림 발송                 | 본당 예산             | 재산출    | 확정 P1만으로 알림 시그니처 4주 체험                     |
| B   | **중고등부 × P1 + 회의록**    | 회의록 자동화 (프로 해자) | 본당 예산 + 교사 자비 | 재산출    | 확정 P1/P5를 선별한 뒤 파일럿                             |
| C   | **군종 × P1 + 명단 이관**     | 명단 이관 (보편 후보)     | 군종교구 예산         | 도입 1건  | 목포해군 인터뷰 우선, 입대/전역 명단 변동 페인 측정       |

> 현재 중고등부 무료 핵심 PMF와 장기 유료 기능 가설을 분리한다. A와 B는 2026-11-30 이후 별도 재승인된 경우에만 결제 검증 코호트로 사용하며, 이번 본당 패키지 결과를 Basic/Pro WTP로 일반화하지 않는다.

- **P4 + R24-A1 결합**: 시즌 휴면과 초기 정착 실패를 분리하기 전에는 가설 검증에서 제외

## 장기 SaaS 검증 항목, 현재 보류

- 매트릭스 A 셀별 결제 전환율 비교 (P1 vs P2 vs P3)
- 단체 타입별 알림 시그니처 사용률 — 초등부 P1 4주 파일럿에서 30% 미만이면 3단계 착수 전 시그니처 교체 검토
- P5 분석가형 다계정 단체의 회의록 파일럿 활성률 60%+ (BM 해자 가설 검증)
- 군종 페르소나 추가 본당 도입 여부 (현재 1건 일반화 한계)

## 데이터 갭

- **초등부 P5/P6 분포 미상**: 카톡 인터뷰 4명 모두 청년-30대 페르소나 + 자모/베테랑 0명. 인터뷰 모수 보강 필요(STATUS 오픈 이슈)
- **청년 P2 데이터 부재**: 청년부 사후 일괄 입력 사례 없음 — 모수 자체가 작음(소수 단체)
- **군종 페르소나 일반화 한계**: 1건(목포해군) 단독. 추가 도입 전까지 가설 수준 유지
- **P6 회의적 신중파 일반화 어려움**: 이한별 1건 — 단체 타입별 분포 측정 불가
