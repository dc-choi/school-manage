# 기능 소개 - "출석 체크, 탭 한 번이면 끝"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
일요일 미사 끝나고 출석 체크, 달력 탭 한 번이면 끝.
종이 출석부 꺼낼 필요 없이 자동 저장까지.
프로필 링크에서 시작하세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "미사 끝! 출석 체크해야지~"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 일요일 미사가 끝난 직후, 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 스마트폰을 꺼내드는 장면. 배경에 성당 건물이 살짝 보인다. 주변에 학생 캐릭터 2~3명이 뛰어다니고 있다. 교리교사 표정은 밝고 여유로움. 말풍선에 "미사 끝! 출석 체크해야지~". 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "날짜 누르고... 이름 옆에 체크!"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰을 들고 화면을 탭하는 장면. 스마트폰 화면에 달력 UI가 보이고, 오늘 날짜가 동그라미로 선택되어 있다. 그 아래에 학생 이름 목록과 체크박스가 보이고, 손가락으로 체크를 탭하는 모습. 말풍선에 "날짜 누르고... 이름 옆에 체크!". 캐릭터 표정은 편안하고 집중하는 모습. 4컷 만화 중 2컷 스타일, 간편한 동작을 강조.
```

### 3컷 (전)
- 대사: "어? 자동으로 저장됐어!"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 눈을 동그랗게 뜨고 기쁜 놀라움의 표정. 스마트폰 화면에 "저장 완료" 체크마크 아이콘이 크게 보인다. 말풍선에 "어? 자동으로 저장됐어!". 캐릭터 머리 위에 반짝이는 별 이펙트. 양손으로 스마트폰을 잡고 살짝 들어올리는 포즈. 4컷 만화 중 3컷 스타일, 발견의 기쁨과 놀라움을 표현.
```

### 4컷 (결)
- 대사: "이게 끝이야? 종이 출석부는 이제 안녕~"
- 하단: "달력에서 탭 한 번, 자동 저장" + "프로필 링크에서 시작하세요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 한 손에 스마트폰을 들고 다른 손으로 브이(V) 포즈. 뒤쪽에 종이 출석부와 펜이 작게 그려져 있고, 그 위에 X 표시 또는 투명하게 사라지는 이펙트. 말풍선에 "이게 끝이야? 종이 출석부는 이제 안녕~". 밝고 뿌듯한 표정. 주변에 꽃잎/별 이펙트. 하단에 텍스트 박스: "달력에서 탭 한 번, 자동 저장" 그리고 작은 글씨로 "프로필 링크에서 시작하세요". 4컷 만화 중 4컷(마무리) 스타일, 밝고 시원한 톤.
```
