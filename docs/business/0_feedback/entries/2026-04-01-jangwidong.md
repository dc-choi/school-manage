# 2026-04-01 장위동 초등부 (직접 피드백)

**요약**: 학생 대량 등록 중 중복 확인 니즈 + 후원 시스템 접근 불가

**피드백 항목**

| 피드백 | 분류 | 대응 |
|--------|------|------|
| 학생 등록 시 중복 확인 요청 — 이름+세례명 동일하면 "이미 등록된 학생인데 그냥 할꺼임?" 확인 | 학생 등록 | 미대응. 일괄등록/개별등록 모두 해당 |
| 후원 시스템이 막혀있음 | 후원/결제 | 확인 필요. 카카오페이 후원 링크 접근 불가 상태 |

**맥락**
- 장위동초 학생 25→51명(+26) 대량 등록 중 발생한 피드백
- 다른 교사가 이미 등록한 학생과 중복 가능성 → 팀 사용 환경에서의 실제 문제
- 이탈 상태(2022-12 이후 출석 없음)였으나 학생 등록 활발 — 재활성화 신호

**패턴 분석**
- **학생 중복 등록**: 팀 사용(복수 교사) 환경에서 최초 발견. 우면동(12계정)/사창동초(11계정) 등 팀 사용 본당에서도 잠재적 이슈
- **후원 시스템**: 첫 접근 불가 리포트. 실제 후원 의사가 있는 사용자에서 발생
