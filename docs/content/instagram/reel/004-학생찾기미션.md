# 미사 끝나고 학생 찾기

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 없음 (순수 공감)
- 스타일: POV / 미션 임파서블 패러디

## 콘셉트
미사 끝나고 학생 찾으러 성당 여기저기 두리번거리는 POV. 미션 임파서블 BGM. "분명 아까 여기 있었는데..."

## 구성

### 장면 1 (0~3초)
- 자막: "미사 끝남"
- 자막: "MISSION: 학생 찾기"

### 장면 2 (3~6초)
- POV: 성당 내부 두리번
- 자막: "분명 아까 여기 있었는데..."

### 장면 3 (6~9초)
- POV: 성당 앞마당 두리번
- 자막: "여기도 없고..."

### 장면 4 (9~12초)
- POV: 주차장/매점 쪽
- 자막: "여기도 없어..."

### 엔딩 (12~15초)
- 자막: "MISSION FAILED"
- 자막: "학생 전원 이미 집에 감"

## 캡션
```
미사 끝나고 3분이면 사라지는 학생들.
어떻게 그렇게 빨리 가는 거야...
```
