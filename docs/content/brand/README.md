# 위클리스쿨 브랜드 에셋

위클리스쿨(`@weekly-school`)의 시각적 정체성 가이드입니다. 로고, 캐릭터, 컬러 시스템을 한곳에 모아두고, 인스타 콘텐츠/제품 UI/외부 자료에 일관되게 적용합니다.

## 산출물

| 파일                                                       | 내용                                                                              |
| ---------------------------------------------------------- | --------------------------------------------------------------------------------- |
| [logo.md](./logo.md)                                       | **확정 로고 v2** (워드마크 + 웃는 ◎ + amber 별, 햇살 제거), 베리에이션, 사용 규칙 |
| [character.md](./character.md)                             | **확정 캐릭터 주이(Joo-i)**, 6종 표정 시트, 사용 규칙, 시나리오                   |
| [assets/logo-reference.jpg](./assets/logo-reference.jpg)   | 로고 정식 레퍼런스 이미지                                                         |
| [assets/joo-i-reference.jpg](./assets/joo-i-reference.jpg) | 주이 6종 표정 정식 레퍼런스 이미지 (콘텐츠 생성 시 이 시트째 첨부)                |

> **모든 후속 콘텐츠**(인스타 카드뉴스, 릴스, OG, UI 일러스트)는 위 두 레퍼런스 이미지를 우선 참고한다. 레퍼런스에 없는 컷은 `character.md`의 사용 규칙, 생성 프롬프트 안에서만 신규 생성한다.

## 브랜드 코어 (작성 기준)

| 항목      | 내용                                                                              |
| --------- | --------------------------------------------------------------------------------- |
| 이름      | **위클리스쿨** / `@weekly-school`                                                 |
| 한 줄     | "매주 주일, 이거 하나 열면 됩니다"                                                |
| 타겟      | 가톨릭 교리교사 (중고등부 우선) + 모임 운영자                                     |
| 핵심 가치 | 주간 허브 — 출석, 축일, 멤버를 한곳에서                                           |
| 톤        | 따뜻함, 진정성, **친근한 귀여움**, 교리교사 관점 (광고 어조 X)                    |
| 정체성    | 가톨릭 친화 (직접적 종교 심볼은 회피, 미묘한 암시), 디지털 도구, 매주 반복 사이클 |

## 디자인 키워드

- **주간 사이클**: 7일 그리드, 주일 강조, 반복되는 만남
- **출석 마크**: `◎` `○` `△` (도메인 상수 `PRESENT_MARKS`와 시각적으로 연결)
- **빛/주일**: Sun-day의 빛, 미사 후의 따뜻함
- **모임**: 둘러앉음, 함께 채워가는 칸

## 디자인 회피

- 십자가/성배/성서 등 직접적 가톨릭 심볼 (포지셔닝상 "범용 모임 운영 도구"로 확장 가능해야 함)
- 책가방, 연필 등 일반 학교(공교육) 클리셰 (위클리스쿨은 학원이 아님)
- 화려한 그라데이션, 3D 광택 (2030세대 톤과 어긋남)
- **과한 키치는 회피하되 친근한 귀여움(둥근 형태, 큰 눈, 볼터치)은 OK** — 교리교사가 "데려가서 쓰고 싶은" 정도까지 허용. 단, 유아용 캐릭터 같은 과장된 표정, 말풍선 욕설/유행어는 금지

## 컬러 시스템 (확정 — 레퍼런스 추출)

| 역할           | 컬러                      | 용도                                          |
| -------------- | ------------------------- | --------------------------------------------- |
| Primary        | **Indigo `#4F46E5`**      | 로고 워드마크, 라인, 캐릭터 라인, ◎ 라인      |
| Accent Warm    | **Amber `#FBBF24`**       | 별, halo, sparkle, 메모 스티커, 캐릭터 포인트 |
| Background     | **Cream `#FAFAF9`**       | 모든 일러스트 기본 배경, 캐릭터 본체          |
| Container Warm | **베이지 `#F5EFE0`**      | 로고 컨테이너 외곽, 카드뉴스 보조 영역        |
| Blush          | **Pink `#FBCFE8`**        | 캐릭터 볼터치 (캐릭터 한정)                   |
| Neutral Text   | **Stone 800 `#292524`**   | 본문 텍스트                                   |
| Sacred Hint    | **Wine `#9F1239`** (선택) | 축일/전례력 한정 (보조 캐릭터 축일이)         |

> 별/halo/sparkle의 amber와 캐릭터 메모 스티커의 amber는 동일 색. 인디고는 라인 컬러로 통일해 로고-캐릭터 시각 연결을 강화.
