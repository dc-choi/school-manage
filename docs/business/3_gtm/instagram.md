### 사용자 인터뷰 콘텐츠

실제 사용하는 교리교사 인터뷰. 카드뉴스(10장) 형식.

#### 목적
- 제3자 증언을 통한 신뢰 확보 (개발자가 아닌 실사용자)
- 신규 사용자의 의심 해소: 전환 필요성, 진입 장벽, 팀 도입, 적합성, 서비스 지속성

#### 인터뷰 질문 (확정, 8개)
| # | 질문 | 의도 |
|---|------|------|
| 1 | "출석부 앱 쓰기 전에는 어떻게 관리하셨어요?" | 이전 고통 — 엑셀/수기 불편함 |
| 2 | "처음 써봤을 때 어땠어요?" | 진입 장벽 낮음 증명 |
| 3 | "다른 선생님들 반응은 어땠어요?" | 팀 도입 가능성 증명 |
| 4 | "지금 가장 자주 쓰는 기능이 뭐예요?" | 핵심 사용 패턴 |
| 5 | "써보면서 예상 못 했던 장점이 있었어요?" | 숨은 가치 (축일 검색 등) |
| 6 | "불편하거나 아쉬운 점은요?" | 진정성 확보 |
| 7 | "4년 넘게 쓰고 계신데, 다시 예전 방식으로 돌아갈 수 있으세요?" | 가장 강한 증언 |
| 8 | "다른 본당 교리교사한테 추천한다면 뭐라고 하실 거예요?" | 사용자 추천 카피 |

#### 편집 원칙
- 답변은 구어체 그대로 유지 (다듬으면 광고가 된다)
- 답변이 길면 핵심 한 문장만 카드에, 나머지는 캡션에

## CTA (Call to Action)

### 문의 유도
- 프로필 링크 클릭 유도
- "DM 주시면 안내해드려요"
- "프로필 링크에서 바로 시작하세요"

### 신청 프로세스
```
1. DM 문의
2. 간단한 안내 (사용 방법, 가입 링크)
3. 계정 생성 지원
4. 온보딩 체크리스트 공유
```

## 운영 원칙

### Do
- 실제 사용 화면 중심 (진정성)

### Don't

- 개인정보 노출 (학생 이름 등)
- 타 서비스 비방
