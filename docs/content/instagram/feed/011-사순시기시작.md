# 공감 콘텐츠 - "사순 시기 시작인데... 우리 반 몇 명이더라?"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

## 캡션
```
오늘 재의 수요일, 사순 시기가 시작됐어요.
부활절까지 40일, 출석 관리도 새 마음으로 시작해볼까요?
멤버 현황부터 축일까지, 한곳에서 확인하세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: (교리교사, 성당에서 이마에 재를 받으며) "사순 시기... 올해는 더 열심히 해야지."

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 성당 안에서 이마에 재의 십자가를 받고 있는 장면. 경건하고 다부진 표정. 말풍선에 "사순 시기... 올해는 더 열심히 해야지." 배경에 제대와 촛불이 희미하게 보인다. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: (집에 와서 메모장 뒤적이며) "그런데 지난주에 누가 왔더라...?"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 집 책상 앞에 앉아 종이 수첩과 카카오톡 메시지를 번갈아 보는 장면. 수첩에는 흐릿한 출석 표시가 보이고, 스마트폰에도 메시지가 여러 개 쌓여 있다. 말풍선에 "그런데 지난주에 누가 왔더라...?". 머리 위에 물음표 여러 개. 표정은 약간 난감한 모습. 4컷 만화 중 2컷 스타일, 정보가 여기저기 흩어진 혼란을 표현.
```

### 3컷 (전)
- 대사: "어? 여기 다 있네. 출석도, 축일도."

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰 화면을 보며 눈이 반짝이는 장면. 스마트폰 화면에 달력 형태의 출석부가 보이고, 달력 위에 "재의 수요일"이라는 축일 표시가 함께 보인다. 말풍선에 "어? 여기 다 있네. 출석도, 축일도." 캐릭터 머리 위에 반짝이는 별 이펙트. 밝은 표정. 4컷 만화 중 3컷 스타일, 발견의 기쁨을 표현.
```

### 4컷 (결)
- 대사: "사순 시기, 새 마음으로 시작!"
- 하단: "출석 + 축일 + 멤버, 한곳에서" + "프로필 링크에서 시작하세요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰을 한 손에 들고 밝게 미소 짓는 장면. 다른 손은 화이팅 주먹 포즈. 말풍선에 "사순 시기, 새 마음으로 시작!". 캐릭터 주변에 보라색 사순 시기 느낌의 은은한 별/십자가 이펙트. 하단에 텍스트 박스: "출석 + 축일 + 멤버, 한곳에서" 그리고 작은 글씨로 "프로필 링크에서 시작하세요". 4컷 만화 중 4컷(마무리) 스타일, 밝고 희망적인 톤. 배경에 연보라색 포인트.
```
