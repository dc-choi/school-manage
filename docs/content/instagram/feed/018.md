# 공감 콘텐츠 - "교리교사 한번 해보실래요?"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션
```
"금방 적응하실 거예요~" 라는 말에 시작한 교리교사 생활.

입문 계기가 비슷한 분, 댓글로 알려주세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: 수녀님 "OO 자매님~ 혹시 교리교사 한번 해보실 생각 없으세요?" / 주인공 "네? 저요?"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 미사 끝난 뒤 성당 마당 장면. 수녀 캐릭터(베일 쓴 온화한 표정)가 주인공 캐릭터(20대 여성, 아직 십자가 머리핀 없음)에게 다가와 부드럽게 말을 거는 장면. 수녀 말풍선 "OO 자매님~ 혹시 교리교사 한번 해보실 생각 없으세요?". 주인공 말풍선 "네? 저요?". 주인공은 당황한 표정으로 자기 자신을 가리키고 있다. 배경에 성당 건물이 보인다. 4컷 만화 중 1컷 스타일, 풀 컬러 파스텔톤 채색.
```

### 2컷 (승)
- 대사: 수녀님 "별거 아니에요~ 주일에 한 시간만! 금방 적응하실 거예요~"

```
같은 캐릭터 스타일 유지. 수녀 캐릭터가 양손을 모으고 환한 미소로 설득하는 장면. 수녀 주변에 반짝이와 꽃 이펙트가 떠 있다. 말풍선에 "별거 아니에요~ 주일에 한 시간만! 금방 적응하실 거예요~". 주인공 캐릭터가 수녀의 후광 같은 미소에 홀린 듯 고개를 살짝 끄덕이려는 순간. 주인공 머리 위에 작은 천사와 악마가 나타나 있다. 천사 "좋은 일이지!", 악마 "일요일 늦잠은...". 4컷 만화 중 2컷 스타일, 달콤한 권유에 흔들리는 순간.
```

### 3컷 (전)
- 대사: (6개월 뒤, 교리 자료 양손 가득) "수업 준비, 출석 체크, 행사 기획, 학부모 연락, 회의..."

```
같은 캐릭터 스타일 유지. 6개월 뒤 장면. 주인공 캐릭터(이제 십자가 머리핀 착용)가 양손 가득 교리 자료, 출석부, 행사 계획서를 들고 있는 장면. 어깨에 가방, 한쪽 손에 스마트폰(카톡 알림 폭주). 말풍선에 "수업 준비, 출석 체크, 행사 기획, 학부모 연락, 회의...". 머리 위에 할 일 목록이 떠 있고 끝이 안 보이게 아래로 길게 이어진다. 눈 밑에 다크서클. 배경 상단에 "6개월 뒤" 텍스트. 4컷 만화 중 3컷 스타일, 현실과 기대의 갭이 터지는 순간.
```

### 4컷 (결)
- 대사: (후배 교리교사에게 똑같이) "금방 적응하실 거예요~"

```
같은 캐릭터 스타일 유지. 주인공 교리교사 캐릭터(십자가 머리핀)가 새로 들어온 후배 캐릭터에게 수녀님과 똑같은 포즈로 양손을 모으고 환한 미소를 지으며 말하는 장면. 말풍선에 "금방 적응하실 거예요~". 주인공 주변에 2컷의 수녀님처럼 반짝이와 꽃 이펙트가 떠 있다. 후배 캐릭터가 "네? 저요?" 하며 1컷의 주인공과 똑같은 표정과 포즈. 1컷과 대칭되는 구도. 4컷 만화 중 4컷(마무리) 스타일, 교리교사 권유의 무한 루프가 반복되는 코믹한 여운.
```
