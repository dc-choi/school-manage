# 공감 콘텐츠 - "자 출석 부를게~" (쉿, 이미 없다)

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
미사 끝나고 고개 돌리면 이미 텅 빈 교실.
출석부 꺼내는 속도보다 사라지는 속도가 빠른 우리 반 애들.
여러분 반도 그렇죠...?
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "미사 끝! 자~ 출석 부를게!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 미사가 끝난 교실 장면. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 출석부를 들고 활짝 웃으며 학생들을 바라보는 장면. 자신감 넘치는 밝은 표정. 말풍선에 "미사 끝! 자~ 출석 부를게!". 교실 책상에 학생 캐릭터 5~6명이 앉아 있다. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: (교리교사가 출석부를 펼치며 고개를 숙이는 짧은 순간)

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 출석부를 펼치며 잠깐 고개를 숙여 이름을 확인하는 장면. 눈이 출석부를 향해 있다. 말풍선 없이, 머리 위에 작은 음표 하나가 떠 있다(콧노래 분위기). 배경에 학생들의 자리가 보이지만 학생들은 슬금슬금 자리에서 일어나는 중(뒷모습, 살금살금 빠져나가는 동작). 4컷 만화 중 2컷 스타일, 교리교사가 눈치 못 채는 코믹한 순간.
```

### 3컷 (전)
- 대사: "김민준~"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 고개를 들며 밝은 표정으로 첫 번째 이름을 부르는 장면. 말풍선에 "김민준~". 하지만 교실 책상들이 모두 텅 비어 있다. 책상 위에 학생들이 두고 간 성경책 한 권과 연필 하나만 덩그러니 놓여 있다. 교리교사의 표정이 밝은 미소에서 얼어붙기 시작하는 과도기 표정. 눈이 커지고 입꼬리가 멈춘 상태. 4컷 만화 중 3컷 스타일, 텅 빈 교실과 밝은 표정의 대비로 웃음을 유발.
```

### 4컷 (결)
- 대사: "......나만 남았네"

```
같은 캐릭터 스타일 유지. 완전히 텅 빈 교실에 교리교사 캐릭터 혼자 서 있는 장면. 출석부를 양손으로 꼭 쥔 채 허탈한 표정. 영혼이 반쯤 빠져나간 듯한 이펙트(머리 위에 작은 유령이 살짝 올라오는 모습). 말풍선에 "......나만 남았네". 교실 문이 열려 있고, 문 밖 복도로 뛰어가는 학생들의 작은 뒷모습 실루엣이 먼지 구름과 함께 보인다. 4컷 만화 중 4컷(마무리) 스타일, 허탈한 코믹 엔딩.
```
