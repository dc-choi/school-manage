# /push

현재 브랜치의 커밋을 원격 저장소에 푸시합니다.

## 실행 단계

1. `git status`로 현재 상태 확인 (커밋되지 않은 변경사항 체크)
2. `git log origin/<branch>..HEAD`로 푸시할 커밋 확인
3. 사용자에게 푸시할 내용 요약 표시
4. 사용자 확인 후 `git push` 실행

## 출력 정보

- 현재 브랜치명
- 푸시할 커밋 개수 및 목록
- 원격 저장소 URL

## 주의사항

- 커밋되지 않은 변경사항이 있으면 경고
- `--force`는 사용자가 명시적으로 요청할 때만 사용
- main/master 브랜치에 force push 시 추가 경고
