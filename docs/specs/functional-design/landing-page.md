# 기능 설계: 랜딩 페이지

> 인스타그램 유입 사용자가 서비스를 이해하고 가입을 결심하도록 설득하는 `/landing` 퍼블릭 페이지.

## 연결 문서

- PRD: `docs/specs/prd/school-attendance.md`, `docs/specs/prd/landing-faq.md`
- 전환 퍼널 분석: `docs/business/STATUS.md`
- 포지셔닝/가치 제안: `docs/business/3_gtm/gtm.md`, `docs/business/2_bm/bm.md`

## 기능 범위

| 기능 | 설명 | 상태 |
|------|------|------|
| 기본 랜딩 페이지 | 5단 풀스크린 스크롤 + 인터랙티브 데모 | 구현 완료 |
| 포지셔닝 개선 2차 (1단계) | "주간 허브" 리프레이밍 — 콘텐츠/메시지 교체 | 구현 완료 |
| FAQ 섹션 (2단계) | CTA 아래 아코디언 6개 항목 | 구현 완료 |
| 메타 태그 + 브랜딩 (2단계) | OG 태그, favicon, title, footer 연도 | 구현 완료 |

---

## 페이지 구조

6단 풀스크린 스크롤. FadeIn 애니메이션 (IntersectionObserver).

| 섹션 | 내용 |
|------|------|
| ① Hero | "매주 일요일, 이거 하나면 됩니다" + 서브카피 + 스크롤 가이드 |
| ② Pain Points | 공감 유발 3건 (출석/명단 관리/축일 조회) |
| ③ Features | 핵심 기능 4개 (아이콘 + 설명, "학생·축일 관리" 포함) |
| ④ Interactive Demo | 달력 + 출석 체크 체험 (로컬 데이터) |
| ⑤ Social Proof + CTA | "{N}개 단체" + CTA + 로그인 |
| ⑥ FAQ | "자주 묻는 질문" 아코디언 6개 (단일 열기) |

## Hero 콘텐츠 (포지셔닝 개선 2차)

| 요소 | 값 |
|------|------|
| h1 | "매주 일요일, 이거 하나면 됩니다" |
| 서브카피 | "**출석, 축일, 학생 현황**까지. / 주일학교 교리교사를 위한 주간 도구." |
| AuthLayout 동기화 | 데스크톱/모바일 h1·서브 = 랜딩 Hero와 동일 |

## FAQ 콘텐츠

| # | 질문 | 답변 요약 |
|---|------|----------|
| 1 | 정말 무료인가요? | 모든 기능 무료, 핵심 기능은 계속 무료 |
| 2 | 개인정보는 안전한가요? | 최소 수집, 제3자 미제공 |
| 3 | 선생님 여러 명이면? | 모임당 계정 하나, 공유 사용 |
| 4 | 스마트폰에서도? | 웹 기반, 앱 설치 불필요 |
| 5 | 가톨릭 주일학교만? | 최적화되어 있지만 출석 관리 필요한 모임 어디든 |
| 6 | 기존 데이터 이전? | 직접 입력, 대량 등록 준비 중 |

- UI: shadcn/ui Accordion (단일 열기, FadeInSection, 풀스크린 미적용)

## 메타 태그 + 브랜딩 정비

| 항목 | 값 |
|------|------|
| `<title>` | 주일학교 출석부 \| 매주 일요일, 이거 하나면 됩니다 |
| `<meta description>` | 출석, 축일, 학생 현황까지. 주일학교 교리교사를 위한 주간 도구. 무료로 시작하세요. |
| OG 태그 | og:title, og:description, og:image(/og-image.png 1200×630), og:url, og:type, og:locale(ko_KR) |
| favicon | `favicon.png` (PNG 32×32) + `apple-touch-icon.png` (180×180) |
| theme-color | 앱 primary 색상 |
| footer | "© 2022–{현재연도} 주일학교 출석부" (동적) |

## API

신규 API 없음. `account.count` (기존) 재사용 → 사회적 증거 표시.

## 예외/엣지 케이스

| 상황 | 처리 |
|------|------|
| account.count 실패 / 0 | 사회적 증거 영역 숨김 |
| 인증된 사용자 접근 | `/` 리다이렉트 |
| OG 이미지 로드 실패 | SNS 자체 폴백 |

## 측정

| 이벤트 | 설명 |
|--------|------|
| `landing_view` | 랜딩 페이지 진입 |
| `landing_section_view` | 섹션 뷰포트 진입 (section 파라미터) |
| `landing_cta_click` | CTA 버튼 클릭 |
| `landing_faq_click` | FAQ 항목 열기 |
