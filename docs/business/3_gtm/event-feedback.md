# 피드백 이벤트

## 메타
- 상태: 기획 완료
- 기획일: 2026-02-20

## 개요

| 항목 | 내용 |
|------|------|
| 목적 | 사용자 피드백 수집 + 인스타 확산 |
| 대상 | 누구나 (기존 사용자 + 잠재 사용자) |
| 채널 | 인스타그램 → 구글폼 |
| 경품 | 스타벅스 기프티콘 (5,000원) |
| 기간 | ~2026-02-28 |
| 당첨자 발표 | 3월 초 (DM 개별 연락) |
| 당첨 | 추첨 2~3명 |
| 예산 | 10,000~15,000원 |

## 전환 경로

```
인스타 피드 게시물 (이벤트 공지)
    ↓
댓글에 피드백 폼 링크 공유
    ↓
피드백 폼 응답
    ↓
추첨 → 당첨자 DM 발송
    ↓
결과 공유 (스토리 + 빌드 인 퍼블릭 콘텐츠)
```

## 구글폼 질문 설계

### 기본 정보
1. 본당/모임 이름 (선택, 미입력 가능)
2. 역할 (교리교사 / 모임 운영자 / 기타)

### 현행 관리 방식
3. 현재 출석/모임 관리를 어떻게 하고 계세요? (복수 선택)
   - [ ] 종이 출석부
   - [ ] 엑셀/구글 시트
   - [ ] 다른 앱
   - [ ] 따로 관리 안 함
   - [ ] 기타

### 서비스 사용 여부 분기
4. weekly-school을 사용해 보셨나요?
   - 사용 중 → 5a~7a
   - 가입만 하고 아직 → 5b
   - 아직 안 써봤어요 → 5c

### 사용 중인 분 (5a~7a)
5a. 사용하면서 가장 불편했던 점은? (자유 서술)
6a. 가장 유용했던 기능은? (자유 서술)
7a. "이런 기능이 있으면 좋겠다" 하는 것은? (자유 서술)

### 가입만 한 분 (5b)
5b. 가입 후 사용하지 않은 이유는? (복수 선택)
   - [ ] 시간이 없어서
   - [ ] 어떻게 쓰는지 모르겠어서
   - [ ] 필요한 기능이 없어서
   - [ ] 화면이 불편해서
   - [ ] 기타

### 미사용자 (5c)
5c. 아직 사용하지 않은 이유는? (복수 선택)
   - [ ] 몰라서
   - [ ] 필요 없어서
   - [ ] 현재 방식이 충분해서
   - [ ] 기타

### 공통 마무리
8. 추천 의향: 주변에 추천하고 싶으신가요? (1~5점)
9. 하고 싶은 말 (자유 서술, 선택)
10. 추첨 참여를 위한 연락처 (인스타 아이디 또는 연락처)
