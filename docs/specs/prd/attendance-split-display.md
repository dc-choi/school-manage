# PRD: 출석 인원 분리 표시

> 상태: Draft | 작성일: 2026-03-20

## 배경/문제 요약

- 참고: `docs/business/6_roadmap/roadmap.md` (2단계 — 가톨릭 특화)
- **문제**: 달력 뷰에서 출석 현황이 `출석/전체` 단일 수치로만 표시되어, 미사와 교리 참석 인원을 구분할 수 없음
- **현재 상태**: ◎(미사+교리), ○(미사만), △(교리만) 마크로 출석을 기록하지만, 달력 셀에서는 마크 종류와 무관하게 "출석한 학생 수 / 전체 학생 수"만 노출
- **목표 상태**: 달력 셀에서 미사 참석 인원과 교리 참석 인원을 분리 표시하여, 한눈에 미사/교리 참석 현황 파악 가능

## 목표/성공 기준

- **목표**: 교리교사가 달력을 보고 미사/교리 각각의 참석률을 즉시 파악할 수 있게 함
- **성공 지표**: 달력 페이지에서 날짜별 미사/교리 인원이 분리 표시됨
- **측정 기간**: 배포 후 1주 내 검증

## 사용자/대상

- **주요 사용자**: 교리교사, 관리자
- **사용 맥락**: 매주 출석 체크 후 달력 뷰에서 출석 현황 확인 시. 미사만 참석한 학생과 교리만 참석한 학생 비율을 파악하여 운영 판단에 활용

## 범위

### 포함

- 달력 셀 출석 표시 변경 (단일 수치 → 미사/교리 분리)
- Backend 달력 API 응답에 미사/교리 인원 추가
- 공유 타입 변경 (`CalendarDayAttendance`)

### 제외

- 출석 입력 모달 변경 (기존 미사/교리 체크 방식 유지)
- 테이블 뷰 출석 현황 변경
- 통계 페이지 변경

## 사용자 시나리오

1. **달력 확인**: 교리교사가 달력 페이지에서 이번 달 출석 현황을 본다. 각 날짜 셀에 미사 참석 인원과 교리 참석 인원이 분리되어 보인다.
2. **운영 판단**: 미사 참석은 많으나 교리 참석이 적은 날짜를 발견하고, 교리 참석 독려 방안을 고민한다.

## 요구사항

### 필수 (Must)

- [ ] 달력 셀에 미사 참석 인원과 교리 참석 인원을 분리 표시
- [ ] Backend API가 날짜별 미사/교리 인원을 각각 반환
- [ ] 출석 마크 기준: 미사 = ◎+○, 교리 = ◎+△

### 선택 (Should)

- [ ] 모바일(360px)에서 가독성 확보

### 제외 (Out)

- 테이블 뷰 출석 표시 변경
- 통계 페이지 미사/교리 분리 통계
- 출석 입력 방식 변경

## 제약/가정/리스크/의존성

- **제약**: 달력 셀 공간이 제한적 (h-24 sm:h-28). 미사/교리 두 수치를 넣되 가독성 유지 필요
- **가정**: 기존 출석 마크 체계 (◎/○/△) 유지
- **리스크**: 모바일 셀에서 두 줄 표시가 비좁을 수 있음
- **내부 의존성**: `@school/shared` CalendarDayAttendance 타입 변경, `get-calendar.usecase.ts` 수정
- **외부 의존성**: 없음

## 롤아웃/검증

- **출시 단계**: 일반 배포 (기존 기능 개선, 플래그 불필요)
- **검증**: 달력 페이지에서 미사/교리 인원 분리 표시 확인, 기존 출석 입력/삭제 정상 동작
