# PRD: 학생 엑셀 Import

## 배경/문제 요약

- **참고**: `docs/business/STATUS.md` (GA4 행동 분석, 파일럿 현황)
- **문제**:
  - 학생 등록이 1명씩만 가능하여 대량 등록 시 과도한 시간 소요
  - GA4 데이터: `/students/new` 페이지 체류 시간 3분 10초 (전체 최장) — 학생 입력 부담 시그널
  - **직접 피드백 (길음동)**: 90명+ 등록이 필요한데 1명씩 등록이 너무 오래 걸린다는 불만. 길음동은 이탈 확정 상태(14일 미활동)로, 등록 부담이 이탈 원인 가능성
  - 기존에 엑셀로 학생 명단을 관리하던 교사가 데이터를 옮기기 어려움
  - 참고: 파워 유저(구룡성당 111명, 가재울 122명)는 1명씩 등록을 완료했으나 불만 없음 — 대량 등록 부담의 임계점은 개인차 존재
- **현재 상태**: 학생 1명씩 개별 등록 (이름, 세례명, 성별, 나이 등 매번 수동 입력)
- **목표 상태**: 엑셀 파일 업로드로 학생 일괄 등록

## 목표/성공 기준

- **목표**:
  - 100명+ 학생을 한 번에 등록할 수 있는 경로 제공
  - 신규 도입 단체의 온보딩 마찰 해소
  - 기존 엑셀 데이터를 손쉽게 마이그레이션
- **성공 지표**:
  - `/students/new` 인당 평균 조회 수 감소 (현재 4.9회 → 2회 이하)
  - 학생 등록 완료까지 소요 시간 단축
  - 학생 등록 후 출석 기록까지 전환율 개선 (현재 퍼널: 학생 19 → 출석 12, 63%)
- **측정 기간**: 배포 후 4주

## 사용자/대상

- **주요 사용자**:
  - 신규 도입 교리교사: 기존 엑셀 명단을 시스템으로 마이그레이션
  - 대규모 주일학교 교리교사: 30명+ 학생 일괄 등록
- **사용 맥락**:
  - 서비스 최초 도입 시 기존 학생 데이터 이관
  - 학기 초 신규 학생 대량 등록

## 범위

### 포함

- 엑셀 파일(.xlsx) 업로드
- 업로드 데이터 미리보기 (검증 결과 포함)
- 검증 통과 데이터 일괄 등록
- 에러 행 표시 및 피드백

### 제외

- CSV 파일 지원 (추후 검토)
- 엑셀 Export (다운로드)
- 학생 정보 일괄 수정 (Import는 신규 등록만)
- 다른 시스템에서의 자동 동기화

## 사용자 시나리오

1. **엑셀 마이그레이션**: 교사가 기존 엑셀 명단 파일을 업로드 → 미리보기에서 매핑 확인 → 일괄 등록
2. **대량 신규 등록**: 교사가 양식에 맞게 엑셀 작성 → 업로드 → 미리보기 확인 → 일괄 등록
3. **부분 오류 처리**: 업로드한 파일에 일부 오류 행 존재 → 오류 행 확인 → 정상 행만 등록 또는 수정 후 재업로드

## 요구사항

### 필수 (Must)

- [ ] 엑셀 파일(.xlsx) 업로드 및 파싱
- [ ] 기본 컬럼 6개 순서 고정: 학년(groupId) → 이름(societyName) → 세례명(catholicName) → 성별(gender) → 전화번호(contact) → 축일(baptizedAt)
- [ ] 추가 컬럼 지원: 나이(age), 비고(description) — 기본 6개 뒤에 선택적 배치
- [ ] 업로드 데이터 미리보기 테이블
- [ ] 행별 검증 결과 표시 (성공/실패/경고)
- [ ] 검증 통과 행 일괄 등록
- [ ] 성별 입력 형식 다중 지원: 남/여, M/F → 내부적으로 M/F로 정규화
- [ ] 필수값(학년, 이름) 누락 검증
- [ ] 등록 결과 요약 (성공 N건, 실패 M건)
- [ ] 엑셀 템플릿 다운로드 (양식 제공, 업로드 전 필수 안내)
- [ ] 기존 개별 등록 UX 유지 (Import는 별도 진입점)

### 선택 (Should)

- [ ] 중복 학생 감지 (동일 이름+그룹)

### 제외 (Out)

- CSV/Google Sheets 지원
- 기존 학생 정보 업데이트 (upsert)
- 여러 그룹에 동시 배정
- 출석 데이터 Import

## 제약/가정/리스크

- **제약**:
  - 현재 학생은 하나의 그룹(group_id)에만 소속 가능 (1:N)
  - Import 시 대상 그룹을 미리 선택해야 함
- **가정**:
  - 교사들의 기존 엑셀 명단에는 최소 이름 컬럼이 존재함
  - 한 번에 업로드하는 학생 수는 최대 500명 이내
- **리스크**:
  - 다양한 엑셀 형식에 대한 파싱 호환성 (인코딩, 셀 병합 등)
  - 컬럼명이 표준화되지 않아 자동 매핑 실패 가능성

## 의존성

- **내부**:
