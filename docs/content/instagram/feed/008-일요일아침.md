# 공감 콘텐츠 - "알람 5개 다 씹고 일어남"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
일요일 아침의 나: 알람 5개 맞춰놨는데 전부 무시하는 능력자.
눈 떠서 시계 보는 순간이 진짜 공포영화...
나만 이러는 거 아니죠? 아니라고 해주세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: (전날 밤, 자신만만한 표정) "내일은 절대 안 늦는다. 알람 5개 완벽."

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 밤에 이불 속에서 스마트폰을 들고 알람을 설정하는 장면. 화면에 알람 목록이 주르륵 보인다(7:00, 7:10, 7:20, 7:30, 7:40). 자신감 넘치는 미소. 말풍선에 "내일은 절대 안 늦는다. 알람 5개 완벽.". 방 안은 어둡고 스마트폰 불빛만 얼굴을 비춤. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: (알람이 울리지만 이불 속에서) "...5분만..."

```
같은 캐릭터 스타일 유지. 아침, 스마트폰에서 알람이 요란하게 울리고 있다(음표, 진동 이펙트). 교리교사 캐릭터는 이불을 머리까지 뒤집어쓰고 손만 내밀어 스마트폰을 더듬는 장면. 말풍선에 "...5분만...". 눈은 완전히 감긴 상태, 평화로운 표정. 스마트폰 화면에 "7:00 AM" 표시. 4컷 만화 중 2컷 스타일, 누구나 공감하는 이불 밖 세상은 위험해 느낌.
```

### 3컷 (전)
- 대사: (벌떡 일어나며) "...8시 40분?!?!"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 벌떡 일어나 스마트폰을 두 손으로 움켜잡고 경악하는 장면. 눈이 하얗게 질린 표정, 입이 크게 벌어짐, 머리카락이 폭발한 듯 사방으로 흩어짐. 말풍선에 "...8시 40분?!?!". 스마트폰 화면에 "8:40 AM" 크게 표시. 머리 위에 벼락 이펙트와 해골 마크. 배경에 충격 집중선. 4컷 만화 중 3컷 스타일, 충격의 순간을 극대화.
```

### 4컷 (결)
- 대사: (세수도 못 하고 뛰어가며) "하느님 저 가고 있어요...!!!"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 머리는 부스스, 십자가 머리핀은 비뚤어진 채로 성당을 향해 전력 질주하는 장면. 한 손에 가방, 한 손에 성경책을 들고 있다. 말풍선에 "하느님 저 가고 있어요...!!!". 뒤에 먼지 구름과 속도감 표현 이펙트. 멀리 성당 건물이 보인다. 이마에 땀방울 여러 개. 4컷 만화 중 4컷(마무리) 스타일, 코믹한 질주와 가벼운 여운.
```
