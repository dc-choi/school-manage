# 리스크

## 리스크 목록

### 리스크 1: 보수적 문화로 인한 도입 저항

- 가설: 가톨릭 조직 문화 특성상 새로운 디지털 도구 도입이 느리거나 거부될 수 있다
- 근거: "가톨릭은 보수적"이라는 인식, 기존 방식(수기/엑셀) 유지 경향. 봉헌금 페이 서비스 상용화 실패 사례 (2026-01-25 피드백)
- 검증: 도입 제안→승인 전환율, 보류/거절 기록
- 대응: 소규모 파일럿과 성공사례 공유로 신뢰 기반 추천 유도

### 리스크 2: 사제 동의 지연

- 가설: 최종 의사결정자인 사제의 동의가 지연되면 파일럿 이후 전환이 늦어진다
- 근거: 도입과 결제의 최종 승인이 사제에게 있음
- 검증: 파일럿 종료 후 동의까지 리드타임, 보류/거절 코드
- 대응: 사제 보고용 요약(효과/통계) 템플릿 제공으로 동의 리드타임 단축

### 리스크 3: 첫인상 실패로 인한 영구 이탈

- 가설: 2030세대는 첫 경험에서 가치를 못 느끼면 재시도하지 않고 영구 이탈한다
- 근거: 고덕동 — 그룹 1개 생성 후 미사용 지속. ~~신당동/금호동 — 계정 모델 혼란~~ (계정 모델 전환으로 해소). 가재울/별내 등 S유형은 첫 경험 성공 시 바로 활성 전환
- 검증: 첫 세션 이탈률, 가입 후 첫 출석 기록까지 도달률 — 모임→첫 출석 누적 전환율 **03-16 44% (N=66) → 04-24 56% (N=93)** 개선 추이. 실시간 수치: `STATUS.md` 전환율 분석 참조
- 대응: 온보딩 각 단계에서 즉각적 결과 제공 (행동-보상 원칙, 상세: `docs/business/1_market/market.md`)

### 리스크 5: B2B2C 구조로 인한 최종 사용자 정착 실패

- 검증: 도입 경로별 4주 리텐션 및 활성도 비교
- 대응: 최종 사용자의 즉시 효용(출석/학생 관리) 강조

- 가설: 본당/모임 단위 구독이 1차 모델이나, 관리자 대표 결제 구조가 형성되지 않으면 유료 전환이 지연된다
- 근거: 본당 예산 결제는 승인/결재 절차로 리드타임이 길어질 수 있으며, 개인 프리미엄은 ARPU 불확실
- 검증: 본당 vs 개인 결제 분포, 승인 리드타임, 전환율/해지율
- 대응: 본당 구독 1차 + 개인 프리미엄 보조 (Slack/Notion 패턴)

### 리스크 9: 광고로 인한 사용자 이탈

- 가설: 광고가 UX를 해쳐 이탈률이 증가할 수 있다. 2030세대는 직관성에 민감하여 광고 이탈 리스크가 더 높음
- 검증: 광고 전후 주간/월간 이탈률, 위치별 불만 신고 건수
- 대응: 핵심 기능(출석 체크) 흐름 외부에 광고 배치

### 리스크 10: 부적절한 광고 노출 (광고 도입 보류로 잠정 비활성)

- 가설: 종교/교육 환경에 맞지 않는 광고 노출 시 신뢰도 하락
- 근거: AdSense 일반 카테고리는 도박/주류/타종교 광고 노출 가능. 가톨릭/주일학교 맥락에서 노출 시 사제·학부모의 신뢰도 손실 직접 발생
- 검증: 필터링 적용 후 부적절 광고 신고 건수 — **광고 도입 결정 시 측정 가능 (현재 보류)**
- 대응: 광고 네트워크 카테고리 필터링 + 사용자 신고 기능

### 리스크 11: 광고 수익 불확실성 (광고 도입 보류로 잠정 비활성)

- 가설: 틈새 시장의 낮은 트래픽(주 1회 사용)으로 광고 수익이 운영 비용을 충당하지 못할 수 있다
- 근거: 주간 운영 도구 특성상 평균 세션 빈도가 주 1~2회 수준 → AdSense CPM(국내 평균 ₩300~1,500) 기준 MAO 50곳으로는 월 1~3만원 수준 추정. 운영비 40만원 충당 불가. **광고 수익 vs 신뢰 손실 ROI 불확실로 광고 도입 자체 보류 (`2_bm/bm.md` 광고 섹션)**
- 검증: 월간 광고 수익/운영 비용 비율, CTR, CPM — **광고 도입 결정 시 측정 가능 (현재 보류)**
- 대응: 광고는 보조 수익원, 핵심은 유료 구독. 단기 자금은 개인 부담 + 도네이션으로 충당

### 리스크 12: 가치 체감 부족으로 인한 빠른 이탈
