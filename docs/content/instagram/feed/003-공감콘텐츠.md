# 공감 콘텐츠 - "전임 선생님 연락이 안 돼요"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: DM
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
교리교사 바뀌면 출석부도 같이 사라지죠.

같은 경험 있으시면 DM 주세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "저 이번에 새로 맡게 된 교리교사입니다!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 새로 부임한 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 밝은 표정으로 인사하는 장면. 한 손을 들어 반갑게 인사. 배경에 주일학교 교실. 말풍선에 "저 이번에 중등부 새로 맡게 된 교리교사입니다!". 의욕 넘치는 표정. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "출석부는... 전임 선생님이 가지고 계신데..." / "연락이 안 돼요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 다른 교리교사에게 출석부를 물어보는 장면. 상대 캐릭터가 난감한 표정으로 고개를 긁적이며 대답. 말풍선 2개: "출석부는... 전임 선생님이 가지고 계신데...", "연락이 안 돼요". 주인공 캐릭터의 밝았던 표정이 점점 굳어가는 모습. 4컷 만화 중 2컷 스타일, 현실의 벽을 느끼는 순간.
```

### 3컷 (전)
- 대사: "학생이 몇 명인지도 모르겠고..." / (빈 노트를 바라보며 한숨)

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 빈 노트 앞에 앉아 한숨을 쉬는 장면. 머리 위에 물음표 3개: "학생 몇 명?", "이름이?", "지난 출석은?". 빈 노트에 제목만 "출석부"라고 적혀있다. 탈진한 표정. 말풍선에 "학생이 몇 명인지도 모르겠고...". 4컷 만화 중 3컷 스타일, 막막한 상황을 코믹하게 표현.
```

### 4컷 (결)
- 대사: "처음부터 여기에 기록했으면 이런 일 없었을 텐데..."
- 하단: "교리교사가 바뀌어도 기록은 그대로" + "같은 경험 있으시면 DM 주세요 @weekly-school"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰 화면을 바라보며 아쉬운 듯 미소 짓는 장면. 스마트폰 화면에 깔끔하게 정리된 학생 명단과 출석 기록이 보인다. 말풍선에 "처음부터 여기에 기록했으면 이런 일 없었을 텐데...". 하단에 텍스트 박스: "교리교사가 바뀌어도 기록은 그대로" 그리고 작은 글씨로 "같은 경험 있으시면 DM 주세요 @weekly-school". 4컷 만화 중 4컷(마무리) 스타일, 약간의 아쉬움과 희망이 공존하는 톤.
```
