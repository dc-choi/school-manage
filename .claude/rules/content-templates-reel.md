# Content Templates — 릴스 (단독)

릴스 전용 (영상 제작 중심) 템플릿입니다.

> 역할/톤/규칙은 `rules/content.md` 참조
> 피드: `rules/content-templates-feed.md` / 통합: `rules/content-templates-combined.md`

> **핵심 원칙**: 모든 복붙 대상(캡션, 해시태그, 프롬프트)은 코드블록(```)으로 감싸서 바로 복사할 수 있게 작성한다.

## 사용 시나리오

릴스 단독 형식은 **영상 제작 중심**(POV/타임랩스/연출)이라 카드뉴스로 변환이 어색한 경우에만 사용한다.
대부분의 신기능 소개/피드백 반영형은 **통합 형식**(`content-templates-combined.md`)이 더 효율적
(한 자료로 피드 + 릴스 양쪽 게시).

## 제작 워크플로우

```
1. 콘셉트/스크립트 작성 (docs/content/instagram/reel/)
    ↓
2. 장면별 Veo 영상 프롬프트 작성
    ↓
3. Artlist AI Toolkit (Veo)에서 장면별 영상 생성
   - 텍스트 → 영상 (장면별 5~10초 클립)
    ↓
4. 영상 편집 (CapCut 등)
   - 장면 연결, 자막/BGM 추가, 9:16 세로 비율
    ↓
5. 인스타그램 릴스 업로드 (캡션 + 해시태그 복붙)
```

## 릴스 규격

- **비율**: 9:16 (세로, 1080×1920px)
- **길이**: 15~30초 권장 (최대 90초). 짧을수록 알고리즘 유리
- **자막**: 필수 (음소거 시청 대비)
- **음성 (선택)**: CapCut Adam TTS (한국 릴스 표준) 또는 클로바 더빙

> **주이 등장**: 릴스에 주이를 넣을 땐 Veo 영상 안에서 생성하지 말고, 나노바나나로 만든 정지 컷(레퍼런스 첨부 방식)을 인트로/아웃트로에 합성한다. 영상 생성형은 캐릭터 일관성을 못 지킨다. 컷 규칙은 `rules/content.md` "레퍼런스 이미지 첨부" 참조.

## 음성 사용 가이드 (선택)

| 항목   | 권장값                                                      |
| ------ | ----------------------------------------------------------- |
| 도구   | **CapCut 내장 TTS** (영상 편집 + TTS 한 도구에서 처리)      |
| 보이스 | **Adam** (한국어, 차분 톤. 한국 릴스에서 자주 쓰임)         |
| 속도   | 1.0배 또는 0.95배 (살짝 느리게 = 메시지 무게)               |
| 대안   | 어색하면 클로바 더빙 (clovadubbing.naver.com) — 보이스 풍부 |

> 음성 미적용 시 **BGM + 자막**만으로 충분. 짧은 슬라이드형은 음성 없는 게 표준 (장현도 피드백 2026-05-08).
