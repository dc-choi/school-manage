# 브레인스토밍 결과 보고서

## 주제

현재 GTM 전략이 맞는 방식인지 구조적 재검토. (운영자 본질 고민: "GTM이 맞는 방식인지")

## 토론 요약

### Round 1

biz-planner가 3개 아이디어 제안: (1) 사제 추천 키트, (2) 이탈 직전 리텐션 루틴, (3) 인터뷰 등장자 세일즈 전환. biz-critic은 4가지 비판 + 메타 비판 2건 제기:

- M1: **온보딩 전환율 70% 병목 미해결** (D유형 수색·성남동MH·보라동 전환만 해도 MAO +3)
- M2: **콘텐츠 유입(SEO) 미구현이 진짜 구조적 공백** (인스타 -78%는 채널 수용량 한계)

### Round 2

biz-planner가 비판 수용 후 5개 아이디어로 재구성: (1) 5/6 미팅 어젠다 킷, (2) 컨텍스트 배너 복귀 트리거, (3) 인터뷰 공유 앵커, (A) D유형 전환 가속, (B) SEO 키워드 조사. biz-critic은 트리거 설계 오류("학생 0" → "학생 있음+출석 0") 발견 + 우선순위 명확화 + 5/6 미팅 "지명 요청" 문화적 위험 지적.

### Round 3

biz-planner가 5개를 3개 묶음으로 압축. biz-critic이 메타 비판 2건 추가:

- **메타 1**: "GTM 재검토 ≠ GTM 점검". 사제 네트워크 + 인스타 모두 정체 채널인데 동일 채널 최적화는 답이 아님
- **메타 2**: 미시도 채널(교구 청소년국·교리교사 양성 과정·CPBC) 3라운드 누락 = 관성적 회피

biz-planner가 메타 비판 수용. SEO 보류, 교구 청소년국 컨택으로 대체. 5/6 미팅 성공 기준 명시 ("추가 추천 가능한 사제/본당 1곳 이상 구두 확인").

## 아이디어 목록 (최종 4개)

### 묶음 A: D유형 맞춤 메시지 — 수색 집중 돌파

- **한 줄 요약**: 학생 등록 후 출석 시작 안 한 D유형(수색 34/성남동MH 13/보라동 11/풍납동 D-4)에 케이스별 메시지 발송
- **사용자 가치**: "어떻게 시작하나" 모름 가설 검증 + 그룹→학생 70% 전환율 병목 직접 공략
- **실현 방법**: 본당별 현황 기반 "출석 기록 3단계 구체적 행동" 메시지 30분 작성·발송
- **차별점**: DB 기반 맥락("34학생 등록·출석 0") 개입 — 밴드/엑셀 불가능

### 묶음 B: 5/6 미팅 — 사제 네트워크 시스템화

- **한 줄 요약**: "지명 요청" 폐기 → "보고+감사+백업 채널 탐색" 프레임 + 명시적 성공 기준
- **사용자 가치**: 협력 신부 1인 의존 구조 해소 (45일 단절 재발 방지)
- **실현 방법**: 03-14 후 성과 1페이지 (계정 263, MAO 43, 본당 72, 학생 2,684, 외부링크 318) + 백업 어젠다 ("지구 내 다른 사제 추천 의향")
- **차별점**: 사제 네트워크는 가톨릭 전용 채널, 03-14 +57 계정 실증된 최고 레버리지

### 묶음 C: G유형 배너 — 이벤트 트리거형 재활성화

- **한 줄 요약**: 흑석동 등 이탈 위험 7곳에 "마지막 출석일 + 14~30일 미활동" 트리거 배너 (조건부 착수)
- **사용자 가치**: R18 시험기간 가설 검증 후 자연 복귀 마찰 제거
- **실현 방법**: 05-05 흑석동 탈락 관찰 후 착수 결정. 착수 시 1~2h 개발
- **차별점**: 가톨릭 주일학교 운영 맥락 활용 (시험기간·전례력 인지)

### 묶음 D: 교구 청소년국 컨택 — 미시도 채널 첫 탐색

- **한 줄 요약**: 서울대교구/수원교구 청소년국 공식 이메일 1통 발송 (자료실 등록 또는 교육 협력 의향)
- **사용자 가치**: 사제 네트워크와 독립된 교구 단위 채널 가능성 탐색
- **실현 방법**: 이메일 1통 30분 (SEO 배정 시간 전용)
- **차별점**: gtm.md에 "접점 존재" 명시되어 있으나 3라운드 동안 미활용 — 관성적 회피 해소

## 누락된 관점 (후속 브레인스토밍 주제)

- **CPBC/가톨릭신문 미디어 채널 탐색**: 1인 5h/주 제약 + 5월 데드라인으로 보류. 6월 이후 콘텐츠 유입 전략 재검토 시 포함.
- **교리교사 양성 과정 직접 접근**: 묶음 D 교구 청소년국 컨택 결과에 따라 후속 액션 도출. 별도 브레인스토밍 불필요.
- **온보딩 학생 등록 UX 개선**: 그룹→학생 70% 전환율 병목의 시스템적 해결안. 이번 라운드는 "1회 직접 개입" 단발성 대응이고, 시스템 개선은 별도 SDD 필요.
- **가톨릭 신자 네트워크 확장 (개신교 ChMS와 차별화된 신규 시장)**: 메타 비판 1의 "구조적 재검토" 의도에 가까운 깊은 피벗 가능성. 향후 별도 브레인스토밍 주제로 검토.
