# 사용자 컨텍스트

이 프로젝트의 1인 개발자(소유자) 본인에 대한 영구 컨텍스트. Conductor 워크스페이스 글로벌 메모리는 휘발성이라 영속 정보는 본 파일에 기록한다.

## 역할

- **개발자 + 교리교사**: 출석부 프로덕트의 1인 개발자이자, 본인이 가톨릭 주일학교 교리교사로 활동 중. 사용자 본인이 1차 페르소나(타겟)에 속함
- **서울대교구 청소년국 활동 겸직**: 교구 청소년국 동반 모임에 정기 참여. 교구 직원(간사) 및 사제 라인에 자연스러운 접근 채널 보유

## 활용

- 페르소나 인터뷰 시 사용자 본인이 직접 수집 가능한 시그널이 다양함 — 외부 인터뷰 섭외 부담이 낮음
- 사업 문서/피드백 entry 작성 시 "사용자가 직접 들었다"는 형태의 1차 데이터로 자연스럽게 인용 가능
- 5/6 협력 신부 미팅, 청소년국 동반 모임 등 자연 대면 자리가 피드백 수집 채널로 작동 중 (예: 2026-05-02 청소년국 간사 보안 피드백 → R25 신설)
