# 기능 소개 + 업데이트 - "부서 관리가 생겼어요"

## 메타

| 항목 | 내용 |
|------|------|
| 유형 | 피드 게시물 (카드뉴스, 캐러셀) |
| 타겟 | 교리교사 |
| CTA | 프로필 링크 |
| 카드 수 | 5장 |
| 사이즈 | 1080×1080px (1:1 정사각형) |

### 공통 스타일 프롬프트
```
카드뉴스 스타일. 따뜻한 베이지/크림 계열 배경. 포인트 컬러는 전례력 색상 연동 — 현재 사순시기이므로 깊은 보라색 사용 (연중시기=초록, 부활/성탄=흰색+골드, 성령강림=빨강 등 시기에 따라 변경). 제목은 세리프 느낌의 품위 있는 서체, 본문은 깔끔한 산세리프 서체. 성당 게시판의 현대판 — 신뢰감 있고 친근하면서 깔끔한 느낌. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻한 톤의 부드럽고 품위 있는 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

```

학생은 학년 1개 + 부서 여러 개에 소속되고

```

### Card 3 — 디자인 개선

**헤더**:


**본문**:
```
차가운 회색 대신 따뜻한 색감으로 바뀌고
포인트 색상이 전례력에 따라 바뀌어요

모바일 화면도 새로 정리하고
에러 메시지도 한국어로 바꿨어요
```

```
카드뉴스 기능 소개 슬라이드. 따뜻한 크림색 배경. 상단에 세리프 느낌의 크고 품위 있는 한국어 텍스트 "화면도 따뜻하게 바뀌었어요". 중앙에 Before/After로 나뉜 구성. 왼쪽 Before: 작고 흐린 스마트폰 화면에 차가운 회색 톤의 UI. 오른쪽 After: 밝고 큰 스마트폰 화면에 따뜻한 베이지/크림 배경, 전례력 색상(현재 사순=보라색) 포인트의 깔끔한 UI — 대시보드 카드가 정돈되어 있고, 한국어 텍스트가 선명하다. 화면 한쪽에 전례 색상이 시기별로 바뀌는 것을 보여주는 작은 색상 팔레트(보라-초록-흰색-빨강)가 표시된다. After 쪽에 작은 체크마크와 반짝이. 하단에 산세리프 본문 텍스트. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 따뜻한 톤의 부드럽고 품위 있는 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.

```

## 캡션

```
부서 관리 기능이 새로 생겼어요.

학년(중1, 중2…) 외에 부서(전례부, 봉사부…)를 따로 만들 수 있어요.
학생은 학년 1개 + 부서 여러 개에 소속할 수 있고, 부서별로 학생을 한꺼번에 추가/제거할 수 있어요.

디자인도 바뀌었어요.
차가운 회색 대신 따뜻한 색감으로 새로 입혔어요. 포인트 색상은 전례력에 따라 바뀌어요 — 사순시기엔 보라, 연중시기엔 초록. 모바일 화면 정리, 에러 메시지 한국어화까지.

그 외 패치:
- 통계 시간대 보정, 졸업생 자동 필터링, 출석률 계산 정확도 개선
- 학생 정보 수정 한 번에 저장
- 전화번호 어떤 형식이든 입력 가능
- 로그인 유지 시간 연장

프로필 링크에서 바로 시작할 수 있습니다.
```
