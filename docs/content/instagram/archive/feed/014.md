# 업데이트 공지 - "선생님, 그거 업데이트됐어요"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사 / 모임 운영자
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 보조 캐릭터로 스마트폰 화면 속 앱 캐릭터(작은 체크마크 모양 마스코트)가 등장한다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "비밀번호가 뭐였더라... 못 들어가겠다"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 일요일 아침 스마트폰을 들고 로그인 화면 앞에서 난감한 표정을 짓고 있는 장면. 스마트폰 화면에 로그인 창과 비밀번호 입력칸이 보이고 빨간 에러 메시지가 표시되어 있다. 말풍선에 "비밀번호가 뭐였더라... 못 들어가겠다". 머리 위에 물음표 3개가 떠 있다. 배경에 창문 너머로 일요일 아침 햇살이 들어온다. 4컷 만화 중 1컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

### 2컷 (승)
- 대사: "어? 비밀번호 찾기가 생겼네!"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰 화면을 보며 눈이 반짝이는 장면. 스마트폰 화면에 로그인 페이지가 보이고, 하단에 "비밀번호 찾기" 버튼이 강조되어 보인다. 말풍선에 "어? 비밀번호 찾기가 생겼네!". 표정이 어두운 데서 밝아지는 전환. 캐릭터 옆에 작은 반짝임 이펙트. 4컷 만화 중 2컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

### 3컷 (전)
- 대사: "서비스 소개 페이지도 있고, 로그인도 깔끔해졌다"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰을 양손으로 들고 흐뭇하게 화면을 살펴보는 장면. 스마트폰 화면이 2~3개로 분할되어 표시된다: 하나는 랜딩 페이지(서비스 소개), 하나는 깔끔해진 로그인 화면, 하나는 개인정보 동의 체크 화면. 각 화면 옆에 작은 체크마크 아이콘이 달려 있다. 말풍선에 "서비스 소개 페이지도 있고, 로그인도 깔끔해졌다". 만족스러운 미소를 짓고 있다. 4컷 만화 중 3컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

### 4컷 (결)
- 대사: "계속 나아지고 있구나, 이 출석부"
- 하단: "조용히, 꾸준히 업데이트 중" + "프로필 링크에서 확인하세요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 교실 책상에 앉아 스마트폰으로 출석부를 사용하며 편안하게 미소 짓는 장면. 스마트폰 화면에 출석 달력이 보인다. 말풍선에 "계속 나아지고 있구나, 이 출석부". 옆에 커피잔이 놓여 있고, 창밖으로 따뜻한 햇살이 들어온다. 주변에 작은 하트와 별 이펙트가 떠 있어 만족감을 표현. 하단에 텍스트 박스: "조용히, 꾸준히 업데이트 중" 그리고 작은 글씨로 "프로필 링크에서 확인하세요". 4컷 만화 중 4컷(마무리) 스타일, 따뜻하고 안정감 있는 분위기. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```
