# 공감 콘텐츠 - "읽음 30 답장 0"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
공지 올리고 조용한 단톡방 보면 마음이 웅장해지죠.
읽음 30... 답장 0... 그리고 다음 주 "몰랐어요" 릴레이.
나만 그런 거 아니죠?
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "학부모님들~ 다음 주 야외학습 준비물 안내드립니다!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 밝은 표정으로 스마트폰을 들고 메시지를 보내는 장면. 자신감 넘치는 미소. 화면에 초록색 메신저 채팅방이 보이고, 말풍선에 "학부모님들~ 다음 주 야외학습 준비물 안내드립니다!". 타이핑하는 손가락에 반짝이 이펙트. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: (스마트폰 화면 클로즈업) "읽음 30"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰 화면을 바라보고 있는 장면. 스마트폰 화면이 크게 보이고, 채팅방에 긴 공지 메시지 아래 "읽음 30"이 표시되어 있다. 답장은 하나도 없이 텅 빈 채팅창. 교리교사 표정이 미묘하게 굳어지기 시작한 모습. 눈이 살짝 작아지고 입꼬리가 내려간다. 4컷 만화 중 2컷 스타일, 정적이 감도는 화면.
```

### 3컷 (전)
- 대사: 학부모1 "선생님 그런 공지가 있었어요?" / 학부모2 "저 못 봤는데..." / 학부모3 "언제 보내셨어요?"

```
같은 캐릭터 스타일 유지. 일주일 뒤 장면. 교리교사 캐릭터 주변에 학부모 캐릭터 3명이 둘러싸고 동시에 말을 거는 장면. 학부모1 말풍선 "선생님 그런 공지가 있었어요?", 학부모2 말풍선 "저 못 봤는데...", 학부모3 말풍선 "언제 보내셨어요?". 교리교사가 눈이 휘둥그레지며 충격받은 표정. 머리 위에 물음표 여러 개. 배경에 "일주일 뒤"라는 작은 텍스트. 4컷 만화 중 3컷 스타일, 몰랐어요 폭탄이 터지는 혼란의 순간.
```

### 4컷 (결)
- 대사: (교리교사 멍한 표정) "...분명히 읽음 30이었는데"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰을 두 손으로 들고 허공을 바라보며 멍한 표정을 짓는 장면. 눈에 하이라이트가 사라진 무표정. 말풍선에 "...분명히 읽음 30이었는데". 머리 위에 유령처럼 혼이 빠져나가는 이펙트. 배경에 희미하게 "읽음 30" 숫자가 유령처럼 떠다닌다. 4컷 만화 중 4컷(마무리) 스타일, 코믹한 허무함과 가벼운 여운.
```
