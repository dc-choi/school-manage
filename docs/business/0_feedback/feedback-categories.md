- 출석부 자동 저장 (체크/페이지 전환 즉시 저장)
- 정렬 기준 (가나다/우수 출석)
- 의무축일/행사 출석 구분
- 출석부 UI 전면 개편 요구 (다수) — 모바일 폰트 과대/디자인 톤 요청 (성북동 미카엘라, 04-19)
- 출석 페이지에서 전체 그룹에 속한 학생 확인 (2026-02-28 피드백 이벤트) — 2단계 독립 항목. 현재 1:N 구조에서도 구현 가능

> 출석 관련 피드백의 대부분은 장위동 초등부 파일럿에서 수집됨
> (초등부 전체 의견 일반화 아님, 초기 가치 체감이 이탈을 좌우한다는 레퍼런스)
