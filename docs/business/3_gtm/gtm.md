### 메시지 점검 3원칙

모든 포지셔닝/콘텐츠 메시지는 게시 전 아래 3가지로 점검한다. 적용 사례: `docs/content/instagram/feed/029.md` 보안 카드뉴스 — 마지막 카드 메인 카피를 운영자 "무거운 책임"(철학)에서 "여기까지가 우리가 지키는 방식"(니즈 확답)으로 전환.

1. **3초 후킹**: 첫 화면(표지/썸네일/헤드라인)이 타겟이 스크롤을 멈출 이유인가. 기능명 나열이 아니라 고객이 멈출 장면으로.
2. **30초 니즈 설득**: 판매자 언어("리드타임 단축", "자동화")가 아니라 고객이 원하는 결과로 말하는가.
3. **고객은 가격, 후기부터 본다**: 후기/실사용 근거를 표면에 두는가. 가격이 미정이면 숨기지 말고 현황을 명시.

### 확산 채널 상세 (2026-02-26 확정, 03-16 갱신)

- **사제 협력 채널 (주교 경로) — 2026-05-11 재정의**: 사제에게 **직접 추천 부탁은 어려움** (봉사직 + 사적 만남 중심 + 직업 아님). 03-14 +57은 자연 발생 입소문 1회 사례로 인식. 5/6 사적 미팅도 어젠다 4건 모두 미진입(분위기상 부적절), 응원 메시지만 수령 → **사제 1:1 직접 영업 라인은 닫힘**. 대신 사제 라포를 **주교(교구 단위) 협력**으로 향하는 협력 채널로 재포지셔닝 — 사제 추천(미시) 대신 주교/교구 차원 도입(거시) 추진. 단계: 사제 라포 유지 → 주교 의향 청취 가능 시점 포착 → 교구 단위 도입 협의
- 세일즈 모션: **bottom-up(교리교사 직접 영업/교리교사→교리교사 추천) 중심**. 사제 라인은 거시 협력(주교 경로)으로 장기 운용, top-down(사제→교리교사 직접 추천)은 자연 발생만 인정

```
인스타그램 게시물/하이라이트 → 프로필 링크 → /landing → /signup 또는 /login → / (대시보드)
```

- **재개 승인 시 계획 경로**: 프로필 링크는 UTM이 붙은 `/landing`을 단일 진입점으로 사용한다. 현재 인스타 프로필의 실제 외부 URL은 저장소에서 확인할 수 없으므로 재개 직전에 앱 설정을 검증한다.
- 랜딩 페이지 `/landing` 구현 완료 (2026-02-12). 로그인 페이지의 서비스 소개는 보조 진입점으로 유지한다.
- **이벤트 단위 주의**: `sign_up`은 계정 가입 이벤트, `first_group_created`는 조직의 첫 그룹 이벤트다. 주간 총 이벤트 수를 단순 나눗셈하지 않고, 계정 전환은 동일 사용자 코호트의 가입 후 7일 경로로 측정한다. 조직 전환은 별도 조직 코호트로 측정한다.

## 세일즈 모션

### 자발적 운영 지원 경로

```text
최신 활성 개인과 실제 연락 가능 인원 재산출 → 30,000원 단일 요청
    → 실제 입금, 문의와 관리시간, 요청 전후 활동 변화를 기록
```

- 누적 계정 수나 단체 단위 MAO를 개인 운영 지원 전환율의 분모로 사용하지 않는다.
- 처음 요청한 20명 중 실제 입금이 5명 미만이면 확대 요청을 중단한다.
- 후원자 기능, 배지, 서비스 기간을 제공하지 않는다.

### 무료 제품 도입 경로

```
모임 운영자 직접 접촉 → 개인 가입/모임 생성 → 같은 본당 교리교사 초대
    → 2-4주 파일럿 → 책임자 공유/동의
```

무료 제품 도입을 곧바로 Basic/Pro 구독 전환으로 연결하지 않는다.

### 90일 본당 패키지 판매 경로, 2026-09-01부터 2026-11-29까지

```text
실제 예산 결정자 선정 → 10만원 유료 운영 진단 또는 범위가 명확한 30만원 패키지 제안
    → 전액 선입금 → 고정 범위 납품과 전체 시간 기록
    → 30만원 패키지를 4시간 이내 납품하고 해당 건 양수 공헌이익 확인
    → 다른 구매자에게 60만원 연간 지원 제안
```

- 실제 예산 결정자 5명에게 순차 제안하고, 최소 2명은 기존 개인 관계 밖에서 선정한다.
- 진단과 패키지를 합해 실제 결제 2건 이상, 30만원 패키지 1건 이상, 개인 관계 밖 결제 1건 이상을 초기 신호로 본다.
- 60만원 제안은 30만원 전액 선입금 1건, 4시간 이내 납품, 해당 건 노동 조정 공헌이익 양수를 모두 확인한 경우에만 별도 구매자에게 진행한다.
- 5명 모두 유료 진단과 30만원 패키지 전액 선입금을 거부하면 본당 패키지 가설을 중단한다.
- 같은 구매자에게 여러 가격을 동시에 보여주거나, 입금 전에 기능을 개발하거나, 유료 진단을 무료 납품으로 바꾸지 않는다.
- 진단 단계에서는 학생 개인정보 원본을 받지 않는다. 명단 원본을 받기 전에는 본당의 처리 권한, 범위, 안전한 전달, 보관과 삭제 절차를 문서화한다.
- 성공 계약뿐 아니라 실패한 상담, 견적, 후속 연락, 납품, 재작업 시간을 모두 기록한다.
- 상세 상품 범위와 날짜별 중단 기준은 `docs/brainstorm/2026-07-15/revenue-viability.md`를 따른다.
