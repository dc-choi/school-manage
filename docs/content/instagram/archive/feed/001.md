# 서비스 소개 - "출석부 어디 갔지?"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 캡션
```
출석부 파일 찾느라 미사 시작 5분 전까지 허둥대본 적 있으시죠?
이제 폰 하나면 됩니다.
프로필 링크에서 바로 시작하세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "오늘도 화이팅~!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 20대 여성 교리교사 캐릭터가 밝은 표정으로 교실 문을 열고 들어서는 장면. 말풍선에 "오늘도 화이팅~!" 텍스트. 배경은 심플한 교실. 캐릭터는 머리에 십자가 머리핀을 달고 있다. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "출석부... 어디 갔지?" / "김 선생님한테 있지 않아요?" / "김 선생님 오늘 안 오시는데..."

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 식은땀을 흘리며 가방을 뒤지는 장면. 옆에 또 다른 교리교사 캐릭터가 걱정스러운 표정으로 서 있다. 배경에 벽시계가 9:55를 가리키고 있다. 말풍선 3개: "출석부... 어디 갔지?", "김 선생님한테 있지 않아요?", "김 선생님 오늘 안 오시는데...". 4컷 만화 중 2컷 스타일.
```

### 3컷 (전)
- 대사: "잠깐, 이거 있잖아"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 자신감 넘치는 표정으로 스마트폰을 들어 보이는 장면. 스마트폰 화면에 간단한 출석부 UI가 보인다. 캐릭터 머리 위에 반짝이는 전구 아이콘. 배경에 빛줄기 효과. 말풍선에 "잠깐, 이거 있잖아". 4컷 만화 중 3컷 스타일, 반전 느낌을 살린다.
```

### 4컷 (결)
- 대사: "이제 누가 빠져도 출석은 문제없지~"
- 하단: "학생 명단 / 출석 체크 / 통계까지 한곳에서" + "프로필 링크에서 시작하세요"

```
같은 캐릭터 스타일 유지. 두 교리교사 캐릭터가 나란히 앉아 각자 스마트폰을 보며 웃는 장면. 편안하고 만족스러운 표정. 말풍선에 "이제 누가 빠져도 출석은 문제없지~". 하단에 텍스트 박스: "학생 명단 / 출석 체크 / 통계까지 한곳에서" 그리고 작은 글씨로 "프로필 링크에서 시작하세요". 4컷 만화 중 4컷(마무리) 스타일, 밝고 따뜻한 톤.
```
