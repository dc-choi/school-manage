# 기능 소개 - "이번 주 축일이 뭐였지?"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사 / 모임 운영자
- CTA: 프로필 링크
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션
```
"이번 주 축일이 뭐였지?" 매주 검색하셨죠?
출석부 열면 축일도 바로 보입니다.
출석 확인하면서 축일도 한눈에 확인하세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: (일요일 아침, 교리 수업 직전) "오늘 축일이 뭐였지...? 학생들한테 말해줘야 하는데."

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 교실에서 수업 시작 직전 스마트폰을 들고 급하게 검색하는 장면. 표정은 약간 당황한 모습. 말풍선에 "오늘 축일이 뭐였지...? 학생들한테 말해줘야 하는데." 뒤에 칠판이 보이고, 시계가 수업 시작 5분 전을 가리킨다. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: (검색 중) "가톨릭 축일 검색... 이번 주 축일... 의무 축일..."

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰으로 검색하는 장면. 스마트폰 화면에 검색창이 보이고 "가톨릭 축일 2월" 등의 검색어가 입력되어 있다. 여러 검색 결과가 나열되어 있지만 정보가 제각각. 말풍선에 "가톨릭 축일 검색... 이번 주 축일... 의무 축일...". 눈이 빙빙 도는 표정. 머리 위에 작은 소용돌이. 4컷 만화 중 2컷 스타일, 정보를 찾느라 분주한 느낌.
```

### 3컷 (전)
- 대사: "어? 출석부에 축일이 같이 나오네?"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 스마트폰 화면을 보며 놀라는 장면. 스마트폰 화면에 달력 형태의 출석부가 보이고, 달력 날짜 옆에 축일 이름이 함께 표시되어 있다 (예: "성 베드로 사도좌 축일" 등). 말풍선에 "어? 출석부에 축일이 같이 나오네?". 눈이 반짝이며 감탄하는 표정. 캐릭터 머리 위에 반짝이는 별 이펙트. 4컷 만화 중 3컷 스타일, 발견의 기쁨.
```

### 4컷 (결)
- 대사: "이제 따로 검색 안 해도 되겠다!"
- 하단: "출석부 + 축일, 한 화면에" + "프로필 링크에서 시작하세요"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 학생들 앞에서 자신감 있게 축일을 설명하는 장면. 한 손에 스마트폰을 들고, 다른 손으로 설명하는 제스처. 말풍선에 "이제 따로 검색 안 해도 되겠다!". 학생 캐릭터 2~3명이 앞에서 감탄하는 표정. 밝고 자신감 넘치는 분위기. 하단에 텍스트 박스: "출석부 + 축일, 한 화면에" 그리고 작은 글씨로 "프로필 링크에서 시작하세요". 4컷 만화 중 4컷(마무리) 스타일, 밝고 시원한 톤.
```
