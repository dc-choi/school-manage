# 출석 체크 3초 챌린지

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 프로필 링크
- 스타일: 앱 화면 녹화 / 챌린지

## 콘셉트
실제 앱 화면 녹화. 달력 탭 → 이름 체크 → 자동 저장. 타이머 오버레이로 "진짜 3초면 끝남" 증명.

## 구성

### 장면 1 (0~2초)
- 자막: "출석 체크, 진짜 3초면 끝남"
- 타이머 시작

### 장면 2 (2~5초)
- 앱 화면: 달력에서 날짜 탭
- 앱 화면: 학생 이름 옆 체크 탭탭탭
- 자동 저장 표시
- 타이머 멈춤

### 엔딩 (5~8초)
- 자막: "끝. 자동 저장됨."
- 자막: "엑셀 꺼내지 마세요"

## 캡션
```
출석 체크, 탭 한 번이면 끝.
매주 일요일에 이거 하나 열면 됩니다.
```
