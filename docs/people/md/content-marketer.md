# 콘텐츠 마케터 온보딩 플랜

대표의 콘텐츠 역할을 인수받아 Instagram 운영·콘텐츠 제작·DM 응대를 단독 수행하는 것이 목표다.
4주 후 콘텐츠 기획→제작→게시→지표 추적까지 대표 리뷰 없이 단독 수행 가능해야 한다.

> **AI가 콘텐츠 제작 파트너다.** `/biz content` 커맨드로 AI가 톤앤매너·템플릿·포지셔닝을
> 자동으로 참고해서 콘텐츠 초안을 만들어준다. 마케팅 경험이 부족해도 규칙 파일이 가드레일 역할.
> AI가 처음이면: `ai-setup-guide.md` (설치) → `ai-basics-guide.md` (기초) → `ai-usage-guide.md` (실전) 순서.

## 핵심 포지셔닝 이해

### 왜 "출석 앱"이 아닌가

- 사용자가 원하는 것: **"일요일에 이거 하나 열면 다 되는 것"**
- 출석 기록보다 **빠른 조회 + 축일 확인 + 멤버 현황**이 핵심 가치
- 엑셀/카카오톡/수기로 분산된 정보를 한 곳에 모은 것이 차별점

### 유통 채널 우선순위

1. **사제 네트워크** (최고 레버리지, 검증됨: 1명 추천으로 2일 만에 +57명)
2. **교리교사 네트워크** (수평 추천, 느리지만 꾸준)
3. **Instagram** (랜딩 페이지 유입 경로, 바이럴 채널 아님)

### 경쟁 환경

- 가톨릭 전용 교회 운영 도구 = **한국 시장에 없음** (블루오션)
- 개신교 ChMS (오지교육, MannaERP) = 용어/구조 불일치로 진입 장벽
- 범용 출석 앱 (Attendable, 왔소) = 가톨릭 특화 없음
- 현재 대안: 엑셀 + 카카오톡 + 수기 (분산, 유실 위험)

### 톤앤보이스 핵심

- 공감적, 비판적이지 않은 톤
- "엑셀 그만 쓰세요" (X) → "더 편한 방법이 있어요" (O)
- 기능이 아닌 결과를 보여줌: "출석 기록" → "누가 왔는지 한눈에"
- 사용자 언어 사용: 세례명, 의무축일, 교리교사, 본당
