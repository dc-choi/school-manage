# 기능 설계: 반응형 디자인 수정

> 비기능적 요구사항 — 모바일/태블릿 디바이스에서 깨지는 레이아웃 수정

## 연결 문서

- 디자인 규칙: `.claude/rules/design.md`
- 웹 앱 규칙: `.claude/rules/web.md`

## 배경

모바일(< 640px) 및 태블릿(640px~1024px) 환경에서 다수 페이지의 레이아웃이 깨지는 이슈 발견. 사이드바 항상 노출, 고정 패딩, 테이블 가로 스크롤 미처리 등이 주요 원인.

## 이슈 목록

### 1. 사이드바 레이아웃 (P0)

**현상**: 모바일에서도 사이드바가 80px 고정 너비로 항상 표시. 콘텐츠 영역이 극도로 좁아짐.

**수정 방향**:
- 모바일(< 768px): 사이드바 숨김 + 햄버거 메뉴로 전환 (Sheet 오버레이)
- 태블릿 이상(≥ 768px): 기존 사이드바 유지 (아이콘만 → sm에서 전체 표시)

**상태 전이**:
```
[모바일] 사이드바 숨김 → 햄버거 클릭 → Sheet 오버레이 표시 → 메뉴 선택 → Sheet 닫힘
[데스크톱] 사이드바 상시 표시
```

### 2. 헤더 패딩 과다 (P0)

**현상**: MainLayout 헤더에 px-12(48px), mr-12 고정 적용. 모바일 320px 기기에서 사용자 정보 영역이 잘림.

**수정 방향**:
- 반응형 패딩: 모바일 px-4 → md에서 px-8 → lg에서 px-12
- 우측 마진도 동일하게 반응형 적용

### 3. 테이블 가로 스크롤 (P0)

**현상**: 공통 Table 컴포넌트에 가로 스크롤 없음. 다중 컬럼 테이블이 화면 밖으로 넘침.

**영향 범위**:
- 학생 목록 (8컬럼): 체크, 이름, 세례명, 학년, 성별, 나이, 연락처, 축일
- 학년별 통계 (8컬럼): 학년명, 인원, 주간/월간/연간 출석률, 주간/월간/연간 평균
- 학년 상세 > 학생 테이블 (4컬럼)
- 삭제된 학생 / 졸업생 모달 내 테이블 (5컬럼)

**수정 방향**:
- 공통 Table 컴포넌트에 가로 스크롤(overflow-x-auto) 래퍼 추가
- GroupStatisticsTable에도 동일하게 적용

### 4. EditableField 고정 너비 (P1)

**현상**: 학생 상세 페이지의 EditableField에서 레이블이 128px 고정 너비. 모바일에서 값 표시 영역이 좁고, 편집 시 저장/취소 버튼이 화면 밖으로 넘칠 수 있음.

**수정 방향**:
- 모바일: 레이블+값 세로 배치 (스택)
- 태블릿 이상: 기존 가로 배치 유지
- 편집 모드 버튼도 반응형 대응

### 5. 미사용 Header 컴포넌트 정리 (P2)

**현상**: `Header.tsx`에 모바일 햄버거 메뉴(Sheet 기반)가 완전히 구현되어 있지만, 어떤 레이아웃에서도 사용되지 않는 데드 코드 상태.

**수정 방향**:
- 이슈 1번(사이드바 레이아웃)의 모바일 메뉴 구현 시 Header.tsx의 로직 재활용
- 구현 완료 후 Header.tsx 삭제

## 예외/엣지 케이스

| 상황 | 처리 방법 |
|------|----------|
| 320px 초소형 기기 | 최소 너비 보장, 가로 스크롤 허용 |
| 가로 모드 모바일 | 데스크톱과 동일하게 사이드바 표시 (md 이상) |
| Sheet 오버레이 열린 상태에서 화면 회전 | md 이상이면 Sheet 자동 닫힘 |
| 출석부 테이블 (sticky 이름 컬럼) | 가로 스크롤 시에도 이름 컬럼 고정 유지 (현재 동작 유지) |

## 테스트 시나리오

### 정상 케이스

1. **TC-1**: 모바일(375px)에서 사이드바 숨김 확인 + 햄버거 메뉴로 네비게이션 가능
2. **TC-2**: 모바일에서 학생 목록 테이블 가로 스크롤 가능, 콘텐츠 잘리지 않음
3. **TC-3**: 모바일에서 대시보드 학년별 통계 테이블 가로 스크롤 가능
4. **TC-4**: 모바일에서 학생 상세 EditableField 레이블+값 세로 배치
5. **TC-5**: 태블릿(768px) 이상에서 기존 사이드바 레이아웃 유지

### 예외 케이스

1. **TC-E1**: 모바일에서 Sheet 열린 상태에서 뒤로가기 → Sheet 닫힘
2. **TC-E2**: 모바일에서 모달 내 테이블 가로 스크롤 가능 (삭제/졸업생 관리)
