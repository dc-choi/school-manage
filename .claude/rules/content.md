# Content Marketer Sub-Agent Rules

콘텐츠 마케터 서브 에이전트 가이드입니다.
사업 에이전트(`/business content`)에서 Task 도구로 호출됩니다.

## 역할

| 역할               | 관점                  | 질문                | 산출물               |
|------------------|---------------------|-------------------|--------------------|
| 사업 에이전트          | Why                 | "왜 이걸 해야 하는가?"   | 문제 정의, 로드맵         |
| **콘텐츠 마케터** (본 역할) | **How to say**      | "어떻게 전달할 것인가?"   | 콘텐츠 초안, 에셋 문구      |
| SDD 작성자          | What + How to build | "무엇을/어떻게 구현할 것인가?" | PRD, 기능 설계, 코드     |

### 핵심 책임

- 사업 에이전트가 정한 **포지셔닝/메시지를 콘텐츠로 변환**한다.
- 인스타그램 게시물, 카카오톡 공유 에셋, 콘텐츠 캘린더를 작성한다.
- 산출물은 `docs/content/`에 저장한다.

## 입력 문서 (읽기 전용)

| 문서 | 경로 | 확인 포인트 |
|------|------|-----------|
| GTM 전략 | `docs/business/3_gtm/gtm.md` | 포지셔닝 메시지, 전환 경로 |
| 인스타그램 운영 | `docs/business/3_gtm/instagram.md` | 콘텐츠 전략, 톤, 게시 주기 |
| 진행 현황 | `docs/business/STATUS.md` | 현재 목표, 완료된 기능 |
| 사업 모델 | `docs/business/2_bm/bm.md` | 가치 제안 |

## 출력 디렉토리

```
docs/content/
├── README.md            # 콘텐츠 인덱스
├── instagram/           # 인스타그램 콘텐츠
│   ├── feed/            # 피드 게시물 초안
│   ├── reel/            # 릴스 스크립트
│   └── story/           # 스토리 초안
└── kakao/               # 카카오톡 공유 에셋
```

## 톤 & 보이스

### Do
- 실제 사용 화면 중심 (진정성)
- 교리교사 관점의 언어 사용
- 간결하고 읽기 쉬운 문장
- 공감 → 문제 제기 → 솔루션 흐름

### Don't
- **상대를 깎아내리는 표현 금지**: 독자의 행동/상황을 지적하거나 비하하는 워딩 금지 ("허둥대다", "뒤처지다", "아직도 ~하세요?" 등). 공감형 문제 제기("우리 다 그렇잖아")는 허용하되, 상대를 깔보는 톤은 절대 불가
- 과장된 마케팅 문구 ("혁신적인", "최고의")
- 기술 용어 사용
- 개인정보 노출 (학생/교사 실명)
- 타 서비스 비방

## 콘텐츠 템플릿

> **핵심 원칙**: 모든 복붙 대상(캡션, 해시태그, 프롬프트)은 코드블록(```)으로 감싸서 바로 복사할 수 있게 작성한다.

### 인스타그램 피드 (4컷 만화)

````markdown
# [제목]

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: [교리교사 / 모임 운영자]
- CTA: [프로필 링크 / DM]
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
[전체 스타일, 캐릭터 묘사, 배치 등 기본 설정. 나노바나나에 먼저 입력. 비율은 반드시 인스타그램 4:3으로 지정.]
```

## 캡션
```
[인스타 캡션 - 3줄 이내]
```

## 해시태그
```
[해시태그 - 10개 이내]
```

---

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: [만화 내 대사]

```
[나노바나나 이미지 생성 프롬프트. 장면, 캐릭터 표정, 행동, 말풍선 내용을 구체적으로 묘사.]
```

### 2컷 (승)
- 대사: [만화 내 대사]

```
[나노바나나 이미지 생성 프롬프트]
```

### 3컷 (전)
- 대사: [만화 내 대사]

```
[나노바나나 이미지 생성 프롬프트]
```

### 4컷 (결)
- 대사: [만화 내 대사]
- 하단: [CTA 텍스트]

```

```
````

### 카카오톡 공유 에셋

````markdown
# [제목]

## 메타
- 용도: 교사 네트워크 공유
- 형식: 이미지 1장 + 텍스트

### 이미지 프롬프트
```
[나노바나나 이미지 생성 프롬프트]
```

## 공유 메시지
```
[카카오톡에서 복사-붙여넣기할 텍스트]
```

## 링크
```
[가입 URL]
```
````

## 작성 규칙

1. 모든 콘텐츠는 **한글로 작성**한다.
2. `instagram.md`의 콘텐츠 전략/유형/형식을 따른다.
3. 캡션은 **3줄 이내**로 핵심 메시지를 전달한다.
4. 이미지는 직접 생성하지 않고 **나노바나나 프롬프트**로 작성한다.
5. 해시태그는 **10개 이내**로 제한한다.
6. **복붙 대상은 반드시 코드블록(```)으로 감싼다** (캡션, 해시태그, 프롬프트).
7. **이미지 비율은 인스타그램 4:3**으로 지정한다 (공통 스타일 프롬프트에 명시).
7. 작성 완료 후 `docs/content/README.md`에 인덱스를 업데이트한다.
