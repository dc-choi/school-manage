- `sign_up` 사용자와 단체 생성 이후의 `first_*`를 같은 코호트로 연결하는 계측이 완료될 때까지 활성화 전환율은 `미측정`이다.
- 유입 소스에 `consent`, `reconsent`, `signup`이 섞여 있고 소스별 세션 합계도 전체 세션과 일치하지 않는다. 개인정보 동의 이벤트의 `source` 파라미터와 값이 일치하므로 획득 채널 비중은 계측 정비 전까지 사용하지 않는다.
- `organization_name`, `account_name` 맞춤 차원은 실제 단체와 계정 값을 제공하지 않는다. 단체별 WAU와 WVO는 산출하지 않는다.

## 인터뷰와 의사결정 구조

- 본당 단위: 교리교사가 제안하고 사제가 최종 승인한다.
- 교구 시스템과 예산: 주교 또는 교구 권한자의 승인이 필요하다.
- 피드백 항목 수와 인터뷰 참여자 수는 같은 지표가 아니다. 누적 인터뷰는 정의를 고정해 다시 집계하고, 추가 PG 의향 인터뷰의 실제 진행률은 외부에서 확인한다.
- 유료 검증은 타깃, 플랜 시그니처, 결제 주체를 먼저 고정한 뒤 진행한다.
