# 공감 콘텐츠 - "캠프 조편성, 퍼즐보다 어렵다"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "자, 캠프 조편성 하겠습니다~ 참가자 명단 뽑아왔어요!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 회의실 테이블 앞에서 밝은 표정으로 학생 이름이 적힌 종이를 펼치는 장면. 옆에 다른 교리교사 캐릭터 2명이 앉아 있다. 말풍선에 "자, 캠프 조편성 하겠습니다~ 참가자 명단 뽑아왔어요!". 테이블 위에 펜과 포스트잇이 놓여 있다. 의욕 넘치는 분위기. 4컷 만화 중 1컷 스타일, 풀 컬러 파스텔톤 채색.
```

### 2컷 (승)
- 대사: 교리교사A "걔네 남매니까 다른 조로 해야 해요" / 교리교사B "걔랑 같은 조 하면 안 온대요"

```
같은 캐릭터 스타일 유지. 회의 장면 계속. 교리교사 캐릭터 2명이 동시에 조건을 말하는 장면. 교리교사A 말풍선 "걔네 남매니까 다른 조로 해야 해요", 교리교사B 말풍선 "걔랑 같은 조 하면 안 온대요". 주인공 교리교사가 명단 위에서 이름을 옮기다 멈추며 당황하는 표정. 명단 위에 화살표와 X표시가 복잡하게 그려져 있다. 4컷 만화 중 2컷 스타일, 조건이 쏟아지기 시작하는 순간.
```

### 3컷 (전)
- 대사: (머리를 감싸쥐며) "...이러면 조가 30개가 돼야 하는데요?"

```
같은 캐릭터 스타일 유지. 주인공 교리교사 캐릭터가 양손으로 머리를 감싸쥐고 명단을 내려다보는 장면. 명단 종이 위에 화살표, X표시, 물음표, 방 번호가 뒤엉켜 거미줄처럼 복잡한 도식이 그려져 있다. 말풍선에 "...이러면 조가 30개가 돼야 하는데요?". 머리 위에 빙글빙글 소용돌이 이펙트. 옆에 있던 교리교사들도 고개를 숙이고 한숨 쉬는 모습. 4컷 만화 중 3컷 스타일, 퍼즐이 안 풀리는 절망의 순간.
```

### 4컷 (결)
- 대사: (탈진한 채로) "...끝났다" / 시계: 3시간 경과

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터 3명이 회의 테이블에 탈진한 채 기대어 있는 장면. 명단 종이가 수정 흔적으로 가득하고 포스트잇이 여기저기 흩어져 있다. 주인공 교리교사가 겨우 완성된 조편성표를 들고 지친 미소를 지으며 말풍선에 "...끝났다". 시계가 보이는데 3시간이 지나 있다. 테이블 위에 빈 커피컵과 과자 봉지가 놓여 있다. 피곤하지만 해냈다는 안도감이 묻어나는 표정. 4컷 만화 중 4컷(마무리) 스타일, 3시간 회의 끝에 겨우 끝낸 교리교사의 현실.
```
