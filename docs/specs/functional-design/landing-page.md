# 기능 설계: 랜딩 페이지

> 인스타그램 유입 사용자가 서비스를 이해하고 가입을 결심하도록 설득하는 `/landing` 퍼블릭 페이지.

## 연결 문서

- PRD: `docs/specs/prd/school-attendance.md`, `docs/specs/prd/landing-faq.md`
- 전환 퍼널 분석: `docs/business/STATUS.md`
- 포지셔닝/가치 제안: `docs/business/3_gtm/gtm.md`, `docs/business/2_bm/bm.md`

## 기능 범위

| 기능                                     | 설명                                                                                                                                  | 상태      |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| 기본 랜딩 페이지                         | 5단 풀스크린 스크롤 + 인터랙티브 데모                                                                                                 | 구현 완료 |
| 포지셔닝 개선 2차 (1단계)                | "주간 허브" 리프레이밍 — 콘텐츠/메시지 교체                                                                                           | 구현 완료 |
| FAQ 섹션 (2단계)                         | CTA 아래 아코디언 항목                                                                                                                | 구현 완료 |
| 메타 태그 + 브랜딩 (2단계)               | OG 태그, favicon, title, footer 연도                                                                                                  | 구현 완료 |
| **가톨릭 모임 정체성 확장 (2026-05-09)** | Hero 서브헤딩 "가톨릭 모임을 위한 주간 도구" + "이런 가톨릭 모임에서 사용 중" 섹션(주일학교·청년·레지오·군종·성가대 5카드) + FAQ 정합 | 구현 완료 |

## 페이지 구조

7단 풀스크린 스크롤. FadeIn 애니메이션 (IntersectionObserver).

| 섹션                 | 내용                                                                    |
| -------------------- | ----------------------------------------------------------------------- |
| ① Hero               | "매주 주일, 이거 하나면 됩니다" + 서브카피 + 스크롤 가이드              |
| ② Pain Points        | 공감 유발 3건 (출석/명단 관리/축일 조회)                                |
| ③ Features           | 핵심 기능 4개 (아이콘 + 설명, "구성원과 축일 관리" 포함)                |
| ③.5 Target Groups    | "이런 가톨릭 모임에서 사용 중" 5카드 (주일학교·청년·레지오·군종·성가대) |
| ④ Interactive Demo   | 달력 + 출석 체크 체험 (로컬 데이터). 헤더 "구성원 정보"                 |
| ⑤ Social Proof + CTA | "{N}개 본당에서 {M}명의 선생님들이 ...명과 함께" + CTA + 로그인         |
| ⑥ FAQ                | "자주 묻는 질문" 아코디언 (단일 열기)                                   |

| AuthLayout 동기화 | 데스크톱/모바일 h1·서브 = 랜딩 Hero와 동일. 모바일 h1에 `text-balance` 적용 |

## Target Groups 섹션 (2026-05-09 신규)

비로그인 청년/레지오/군종/성가대 운영자에게 도구 적합성을 즉시 인지시키는 5카드 후킹 섹션. Features ↔ Demo 사이 위치.

| 카드     | 아이콘 (lucide-react) | 1줄 카피                        |
| -------- | --------------------- | ------------------------------- |
| 주일학교 | BookOpen              | 매주 학생 출석을 한 곳에서      |
| 청년     | Sparkles              | 청년 모임 참석 기록을 간편하게  |
| 레지오   | Heart                 | 주회 출석과 활동을 빠르게       |
| 군종     | Shield                | 장병들의 미사 참여를 한눈에     |
| 성가대   | Music                 | 연습 출석을 카톡 대신 한 곳에서 |

- 반응형 레이아웃: 모바일 2열, 데스크톱 5열
- analytics 이벤트: `landing_section_view` (`target_groups`)

## FAQ 콘텐츠

| #   | 질문                   | 답변 요약                                                                      |
| --- | ---------------------- | ------------------------------------------------------------------------------ |
| 1   | 정말 무료인가요?       | 모든 기능 무료, 핵심 기능은 계속 무료                                          |
| 2   | 개인정보는 안전한가요? | 최소 수집, 제3자 미제공                                                        |
| 3   | 선생님 여러 명이면?    | 모임당 계정 하나, 공유 사용                                                    |
| 4   | 가입하면 뭘 먼저?      | "학년(그룹)을 만들고 학생(멤버)을 등록 → 출석 체크" 라벨 병기 (라벨 분기 안내) |
| 5   | 스마트폰에서도?        | 웹 기반, 앱 설치 불필요                                                        |
| 6   | 가톨릭 주일학교만?     | "주일학교, 청년, 레지오, 군종, 성가대 등 가톨릭 모임 운영에 사용"              |
| 7   | 기존 데이터 이전?      | 직접 입력, 대량 등록 준비 중                                                   |

- UI: shadcn/ui Accordion (단일 열기, FadeInSection, 풀스크린 미적용)

## 메타 태그 + 브랜딩 정비

| 항목                 | 값                                                                                                                 |
| -------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `<title>`            | 주일학교 출석부 \| 매주 주일, 이거 하나면 됩니다                                                                   |
| `<meta description>` | 출석, 축일, 구성원 현황까지. 가톨릭 모임을 위한 출석 관리 도구. 주일학교, 청년, 레지오, 군종, 성가대까지 한곳에서. |
| OG 태그              | og:title, og:description, og:image(/og-image.png 1200×630), og:url, og:type, og:locale(ko_KR)                      |
| favicon              | `favicon.png` (PNG 32×32) + `apple-touch-icon.png` (180×180)                                                       |
| theme-color          | 앱 primary 색상                                                                                                    |
| footer               | "© 2022–{현재연도} 주일학교 출석부" (동적)                                                                         |
