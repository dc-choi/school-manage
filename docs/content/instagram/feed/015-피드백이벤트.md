## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "혹시... 쓰면서 불편한 거 없으셨어요?"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 개발자 캐릭터(20대 남성, 안경, 후드, 십자가 목걸이)가 노트북 앞에 앉아 조심스럽게 스마트폰을 들고 있는 장면. 스마트폰 화면에 사용자 후기 알림이 보이지만 내용이 비어 있다. 말풍선에 "혹시... 쓰면서 불편한 거 없으셨어요?". 약간 걱정스러우면서도 궁금한 표정. 배경에 모니터 화면에 출석부 앱 대시보드가 보인다. 4컷 만화 중 1컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

### 2컷 (승)
- 대사: "솔직한 의견 들려주시면 진짜 도움이 됩니다"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 두 손을 모으고 간절한 표정으로 정면을 바라보는 장면. 캐릭터 주변에 말풍선 모양의 아이콘들이 떠 있다: 느낌표, 하트, 전구 모양 등 피드백을 상징하는 아이콘. 말풍선에 "솔직한 의견 들려주시면 진짜 도움이 됩니다". 진심 어린 눈빛. 배경에 구글폼 화면이 크게 보이고, 화살표가 구글폼을 가리키고 있다. 4컷 만화 중 2컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

### 3컷 (전)
- 대사: "참여해 주신 분 중 추첨으로 스타벅스 기프티콘!"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 한 손에 스타벅스 커피 컵을, 다른 손에 스마트폰을 들고 환하게 웃는 장면. 스마트폰 화면에 구글폼 제출 완료 화면이 보인다. 캐릭터 주변에 스타벅스 로고가 그려진 기프티콘 카드가 반짝이며 떠 있다. 말풍선에 "참여해 주신 분 중 추첨으로 스타벅스 기프티콘!". 배경에 커피잔 아이콘과 별 이펙트가 흩어져 있다. 밝고 경쾌한 분위기. 4컷 만화 중 3컷 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 양팔을 벌리고 환영하는 포즈를 취하며 활짝 웃는 장면. 캐릭터 양쪽에 다양한 사람들(교리교사, 학생, 일반인 등)의 작은 실루엣이 구글폼을 향해 걸어가는 모습이 보인다. 말풍선에 "사용자든 아니든 누구나 참여 가능!". 하단에 텍스트 박스: "참여 링크는 댓글에서 확인하세요" 그리고 작은 글씨로 "1분이면 끝!". 배경에 하트와 커피잔 아이콘이 떠 있고, 따뜻하고 환한 분위기. 4컷 만화 중 4컷(마무리) 스타일. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다.
```
