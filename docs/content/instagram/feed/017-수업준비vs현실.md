# 공감 콘텐츠 - "교리 수업 준비 3시간, 실제 수업 3분 컷"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션
```
토요일 밤, 교리 수업 PPT 만드느라 새벽 2시.
일요일 수업, "선생님 오늘 뭐 해요?" 3분 만에 끝남.
이 허무함을 아는 사람만 좋아요 눌러주세요.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: (토요일 밤, 노트북 앞에서) "이번 주 교리는 성체성사! 영상도 넣고, 활동지도 만들고..."

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 토요일 밤 책상 앞에서 노트북을 펼치고 열정적으로 수업 자료를 만드는 장면. 노트북 화면에 PPT가 보이고, 옆에 성경책과 교리서가 펼쳐져 있다. 말풍선에 "이번 주 교리는 성체성사! 영상도 넣고, 활동지도 만들고...". 눈이 반짝이는 의욕 넘치는 표정. 시계가 밤 11시를 가리킨다. 4컷 만화 중 1컷 스타일, 풀 컬러 파스텔톤 채색.
```

### 2컷 (승)
- 대사: (새벽 2시, 눈 충혈) "완벽해... 이번엔 애들이 집중할 수밖에 없어"

```
같은 캐릭터 스타일 유지. 같은 책상, 시계가 새벽 2시를 가리킨다. 교리교사 캐릭터 눈이 충혈되어 있고 머리가 살짝 헝클어져 있지만, 만족스러운 미소를 짓고 있다. 노트북 화면에 완성된 PPT가 보이고, 옆에 인쇄된 활동지가 가지런히 놓여 있다. 말풍선에 "완벽해... 이번엔 애들이 집중할 수밖에 없어". 노트북 화면에서 빛이 나는 이펙트. 책상 위에 빈 커피컵 2개. 4컷 만화 중 2컷 스타일, 밤샘 끝에 완성한 자부심.
```

### 3컷 (전)
- 대사: 학생1 "선생님 이거 다 읽어요? 길어요" / 학생2 (엎드려 자는 중) "zzz"

```
같은 캐릭터 스타일 유지. 일요일 교리 수업 교실 장면. 교리교사 캐릭터가 앞에서 PPT를 보여주고 있는데, 학생 캐릭터들의 반응이 처참한 장면. 학생1은 활동지를 들고 말풍선 "선생님 이거 다 읽어요? 길어요", 학생2는 책상에 엎드려 "zzz" 자고 있고, 학생3은 창밖을 바라보고 있다. 교리교사의 표정이 얼어붙는 모습. PPT 화면에 정성들인 자료가 보이지만 아무도 안 보고 있다. 4컷 만화 중 3컷 스타일, 현실의 벽을 마주한 충격의 순간.
```

### 4컷 (결)
- 대사: (멍한 표정으로 빈 교실에서) "...내 토요일 밤 돌려줘"

```
같은 캐릭터 스타일 유지. 수업이 끝난 뒤 텅 빈 교실에서 교리교사 캐릭터가 홀로 서 있는 장면. 책상 위에 활동지가 그대로 남아 있고, 한 장도 안 쓴 활동지가 보인다. 교리교사가 활동지를 한 장 집어 들며 멍한 표정. 말풍선에 "...내 토요일 밤 돌려줘". 머리 위에 혼이 빠져나가는 이펙트. 하지만 입꼬리가 살짝 올라간 씁쓸한 미소. 4컷 만화 중 4컷(마무리) 스타일, 허무하지만 다음 주에 또 만들 자신을 아는 코믹한 여운.
```
