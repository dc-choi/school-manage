# 경쟁사 분석

> 조사 목적: "다른 출석부 앱이나 웹에 비해 교리교사/모임 운영자가 이 서비스를 써야 하는 이유"를 파악하기 위한 경쟁사 분석
>
> 조사일: 2026-02-13

## 1. 카테고리별 요약

### 1-1. 개신교 ChMS (7개)

교회 행정 전체(교적/재정/심방)를 타겟으로 하며, 출석은 종합 패키지의 일부 기능이다. **가톨릭 지원 없음**.

> **⚠️ 직접 경쟁자 발견 (2026-03-16)**: **브랜치스(branches.team)** — 개신교 청년부·청소년부 모임 운영 특화. 출석/멤버/그룹 + 일정/주보/설문/재정 통합. **95개 교회, 220명 이용. 사업자등록 완료(리드랩)**. 문제 인식과 기능 범위가 우리와 가장 유사.

### 1-2. 가톨릭 관련 (2개)

- **가톨릭 하상**: 신자 개인 신앙생활 앱. 주일학교 학생 출석/명단/통계 관리 기능 **없음**.
- **본당양업시스템**: 본당 사무실 행정 도구. 교리교사가 직접 접근하여 출석을 관리하는 용도가 **아님**.

> **핵심 발견: 가톨릭 교리교사가 사용할 수 있는 출석/멤버 관리 전용 도구가 시장에 0개**

### 1-3. 범용 출석/도구 (5개)

교회 도메인 특화 기능(의무축일, 학년 전환 등) 없이 범용 출석 체크 또는 수작업 도구를 사용하는 현행 대체재.

### 1-4. 해외 ChMS (3개, 참고)

영어권 교회 대상. 한국어 미지원, 한국 교회 구조와 불일치. 가격/비즈니스 모델만 참고.

## 3. 차별점 (가설)

### 우리가 가진 것

1. **가톨릭 주일학교 전용**: 시장에 가톨릭 교리교사/모임 운영자용 출석 관리 도구가 **없음**
2. **교리교사 워크플로우 UX**: 의무축일 표시, 학년 자동 전환, 달력 기반 출석 UI
3. **단순함**: 출석/멤버/통계 핵심만 제공 (교적/재정/헌금 없음)
4. **무료 핵심 기능**: 진입 장벽 최소화
5. **웹 기반 접근성**: 앱 설치 없이 브라우저에서 사용, 모바일 반응형
6. **가톨릭 도메인 장벽**: 교구→본당→모임 계층, 전례력, 세례명/축일. 브랜치스의 교차 진입 비용 높음

### 우리에게 없는 것

1. ~~엑셀 Import~~/Export (Import는 학생 일괄 등록으로 구현 완료)
2. QR/NFC 출석 (3단계 로드맵)
3. 보호자 알림 (카카오톡/SMS)
4. 교구/교단 연동
5. 공지/커뮤니케이션, 행사/일정 관리
6. 오프라인 지원, 네이티브 앱

### 교리교사가 우리를 써야 하는 이유 (가설)

1. **"가톨릭 하상은 신자 앱이지, 교리교사 도구가 아니다"** — 주일학교 명단/출석/통계 관리 기능 없음
2. **"개신교 ChMS를 쓸 이유가 없다"** — 용어, 구조, 기능이 가톨릭과 다름
3. **"엑셀보다 빠르고, 카카오톡보다 안전하다"** — 출석 → 통계/보고 자동화, 데이터 유실 방지
4. **"학년 전환 때 다시 만들 필요 없다"** — 자동 학년 전환
5. **"누가 빠지든 기록 가능"** — 개인 계정으로 접근, 데이터 공유

## 4. 시장 빈틈 (기회)

1. **가톨릭 교리교사용 도구의 완전한 부재** — 하상(신자 앱), 양업(사무실), 개신교 ChMS(구조 불일치). 전용 도구 0개
2. **개신교 ChMS의 과잉 기능** — "출석만 찍고 통계만 보고 싶은" 사용자를 위한 경량 도구 부재
3. **수기→SaaS 전환 장벽** — 사제 설득이 아닌 "교리교사가 혼자 바로 쓸 수 있는" 셀프 온보딩 + 무료 진입이 핵심
4. **가톨릭 모임 전체 확장** — 레지오, 성가대, 청년회 등 본당당 5~10개 모임도 동일 문제
5. **교구 보고서 연동** — 출석 데이터 → 교구 보고 양식 자동 생성으로 사제 설득력 증가
