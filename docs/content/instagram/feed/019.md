# 사용자 인터뷰 - "4년째 쓰고 있는 교리교사에게 물었습니다"

## 메타
- 유형: 피드 (카드뉴스 10장)
- 타겟: 교리교사
- CTA: "무료로 시작하기 → 프로필 링크"
- 스타일: 인터뷰 Q&A 카드뉴스
- 인터뷰이: 장위동 교감 이한별 소피아

### 공통 스타일 프롬프트
```
인터뷰 카드뉴스 스타일. 깔끔한 흰색 배경에 부드러운 파스텔톤 포인트 컬러. 둥글둥글한 귀여운 일러스트 아이콘 포함. 인스타그램 1:1 정사각형 비율. 질문은 작고 가벼운 서체, 답변은 크고 굵은 서체로 강조. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션
```
4년째 출석부 앱을 쓰고 계신 장위동 교감 이한별 소피아 선생님께 물었습니다.

수기 출석부에서 앱으로 바꾸면 어떤 게 달라지는지,
다른 선생님들 반응은 어땠는지,
불편한 점은 없었는지,
솔직하게 답해주셨어요.

"한 번 쓰면 어떻게 뽑아먹을지 자기전에 생각 많이 날거야"
이 한마디가 모든 걸 말해주는 것 같습니다.

프로필 링크에서 무료로 시작할 수 있어요.
```
