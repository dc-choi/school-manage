**서브 카피**:

```
이제 내 계정으로 접속할 수 있어요

```

### Card 2 — 개인 계정 소개

**헤더**:
```
내 아이디로 바로 접속
```

**본문**:
```
아이디와 비밀번호로 가입하면
내 계정이 생겨요

본당이나 모임을 찾아서 합류하면
바로 출석부를 쓸 수 있어요
```

```
카드뉴스 기능 소개 슬라이드. 깔끔한 흰색 배경에 연한 하늘색 포인트 컬러. 상단에 크고 굵은 한국어 텍스트 "내 아이디로 바로 접속". 중앙에 스마트폰 목업이 있고, 화면 안에 회원가입 폼이 보인다 — 아이디, 이름, 비밀번호 입력란이 깔끔하게 배치되어 있다. 스마트폰 옆에 화살표가 흘러가며 "가입 → 모임 합류 → 출석부 사용"의 3단계 흐름이 작은 아이콘으로 표현된다. 하단에 본문 텍스트 "아이디와 비밀번호로 가입하면 내 계정이 생겨요. 본당이나 모임을 찾아서 합류하면 바로 출석부를 쓸 수 있어요". 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

```

```
```
카드뉴스 기능 소개 슬라이드. 깔끔한 흰색 배경에 연한 살구색 포인트 컬러. 상단에 크고 굵은 한국어 텍스트 "개인 계정이면 이런 게 달라져요". 중앙에 네 가지 장점이 아이콘과 함께 세로로 나열되어 있다: (1) 여러 사람 아이콘 + "여러 선생님이 동시에 접속", (2) 시계 아이콘 + "로그인 시간이 더 길어져요", (3) 악수 아이콘 + "인수인계도 간편하게", (4) 반짝이는 별 아이콘 + "개인화 기능 더 추가 예정". 각 항목 사이에 얇은 구분선이 있고, 아이콘은 귀엽고 둥글둥근 스타일. 인스타그램 1:1 정사각형 비율. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 모든 텍스트는 반드시 한국어로 렌더링할 것.
```
