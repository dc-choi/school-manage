# 지표

## North Star

### 공통 정의
- `월간 활성 주일학교`: 계정 기준 주간 로그인 + 출석 기록을 4주 이상 지속 등록한 주일학교

### NSM 1 (무료 핵심)
- 정의: 월간 활성 주일학교 중, 주간 출석 기록 1회 이상을 지속 수행한 비율
- 핵심 가치: 주간 출석 기록의 지속성과 운영 리듬 유지

### NSM 2 (유료 확장)
- 정의: 월간 활성 주일학교 중, 유료 기능 1종 이상을 사용한 비율
- 핵심 가치: 운영 확장 기능이 실제 업무에 포함되는지 여부

### 현재 운영 목표
- **WAU (주간 활성 사용자)**: 7일 내 로그인 고유 사용자 수
- 기준선: 7명 (2026-02-18, 최근 7일)
- 목표: **12명** (5월, 02-26 GTM 브레인스토밍에서 현실화. 기존 10명에서 상향)
- 달성 경로: 사용 중 3곳(창동/길음/가재울) 안착 + 추가 2곳 활성화
- 측정: GA4 로그인 이벤트 기준

### 안착 기준 (02-26 확정)
| 단계 | 기준 | 현재 해당 |
|------|------|----------|
| 1단계 안착 | 3주 연속 출석 입력 | 길음동, 창동, 가재울 |
| 완전 안착 | 4주+ 지속 + 개발자 개입 없는 자가 운영 | 장위동 중고등부, 정릉4동 |

## 측정 체계

### 데이터 소스
| 분류          | 소스              | 보조           | 비고          |
|-------------|-----------------|--------------|-------------|
| NSM/활성화/리텐션 | 제품 DB + 서버 로그   | GA4, 수기      | 서버 로그 기준 집계 |
| 채널/획득       | GA4 (UTM 포함)    | 협력 채널, 랜딩 문의 | 오프라인은 수기    |
| 수익/전환       | 결제/정산 시스템       | 회계 기록        | 환불/취소 반영    |
| 운영/신뢰       | 지원 티켓, 감사/백업 로그 | 장애 공지        | 월간 점검       |
| 도입/승인       | 도입 요청/승인 기록     | 결제 기록        | 리드타임 집계     |

### GA4 역할
- 획득/전환: UTM, 공지 링크 클릭, 체험 신청
- 내부 퍼널: 가입 → 학생 등록 → 첫 출석
- **주 피드백 채널 (2026-03-01)**: 2030세대 Silent Churn 특성상 피드백 수집(구글폼 등)보다 GA4 행동 데이터가 이탈 원인 파악에 더 정확. 26개 이벤트로 어디서 멈추는지 추적
- 핵심 운영 지표는 제품 로그 기준

### 리포트 주기
- 주간/월간 고정 주기 확인

## 활성화

| 지표           | 정의                    | 목표 |
|--------------|-----------------------|----|
| 첫 세션 성공률 | 첫 방문에서 그룹 생성까지 도달률 | - |
| 7일 내 학생/반 등록 | 가입 후 7일 내 등록 완료율      | -  |
| 7일 내 첫 출석    | 가입 후 7일 내 첫 출석 기록 완료율 | -  |
| 14일 내 행사 생성  | 가입 후 14일 내 행사 1건 생성율  | -  |

> **2030세대 참고 (2026-03-01)**: "첫 세션 성공률"이 가장 중요한 선행 지표. 2030세대는 첫인상에서 결판나므로 첫 세션에서 그룹 생성까지 도달하지 못하면 영구 이탈 가능 (상세: `docs/business/1_market/market.md` 사용자 특성)
