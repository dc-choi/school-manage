- 유형: 피드 (4컷 만화)
- 타겟: 교리교사 / 모임 운영자
- CTA: DM
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 남성 개발자, 안경을 쓰고 후드티를 입고 있으며 가슴에 작은 십자가 뱃지를 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치.
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "출석부 서비스 만들었다!!"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 20대 남성 개발자 캐릭터(안경, 후드티, 가슴에 작은 십자가 뱃지)가 노트북 앞에서 두 팔을 번쩍 들고 환호하는 장면. 노트북 화면에 출석부 UI가 보인다. 말풍선에 "출석부 서비스 만들었다!!". 눈이 반짝반짝, 의욕 가득한 표정. 주변에 별/반짝이 이펙트. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "이탈율... 82.6%...?" / (GA4 대시보드를 멍하니 바라보는 장면)

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 노트북 화면을 멍하니 바라보는 장면. 노트북 화면에 큰 글씨로 "이탈율 82.6%" 숫자가 빨간색으로 표시되어 있다. 캐릭터 표정은 하얗게 질린 얼굴, 영혼이 빠져나가는 듯한 모습. 머리 위에 금이 간 하트 이펙트. 말풍선에 "이탈율... 82.6%...?". 배경이 어둡게 변하며 먹구름 효과. 4컷 만화 중 2컷 스타일, 현실 직면의 충격을 코믹하게 표현.
```

### 3컷 (전)
- 대사: "그래도... 쓰고 있는 곳이 있으니까" / (노트북을 다시 펴며 결의에 찬 표정)

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 입술을 꽉 깨물고 다시 노트북 앞에 앉는 장면. 한 손은 주먹을 불끈 쥐고, 다른 손은 키보드 위에 올려놓고 있다. 캐릭터 뒤로 작은 말풍선들이 떠다닌다: "랜딩 페이지 만들자", "온보딩 개선하자", "피드백 받자". 메인 말풍선에 "그래도... 쓰고 있는 곳이 있으니까". 눈에 작은 불꽃 이펙트. 배경이 다시 밝아지며 살짝 노을빛 톤. 4컷 만화 중 3컷 스타일, 좌절 후 다시 일어서는 결의를 표현.
```

### 4컷 (결)
- 대사: "그래서 여러분의 솔직한 의견이 필요합니다"
- 하단: "불편한 점, 원하는 기능, 뭐든 좋습니다" + "DM으로 알려주세요 @weekly-school"

```
같은 캐릭터 스타일 유지. 개발자 캐릭터가 정면을 바라보며 양손을 모아 간절하게 부탁하는 장면. 진심 어린 표정, 살짝 웃으며 눈이 촉촉한 모습. 말풍선에 "그래서 여러분의 솔직한 의견이 필요합니다". 캐릭터 옆에 작은 하트와 말풍선 아이콘이 떠다닌다. 하단에 텍스트 박스: "불편한 점, 원하는 기능, 뭐든 좋습니다" 그리고 작은 글씨로 "DM으로 알려주세요 @weekly-school". 4컷 만화 중 4컷(마무리) 스타일, 따뜻하고 진정성 있는 톤.
```
