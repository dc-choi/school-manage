## 전환 퍼널 분석 (2026-02-12)

> 이탈율 81.8% 원인 분석. 유저가 가입까지 도달하는 심리 흐름이 끊겨 있다.

### 만들어야 하는 유저 여정

```
인스타 → /landing
    ① "이게 뭔데?"      → 후킹 헤드라인 + 핵심 가치
    ② "무슨 기능이 있는데?" → 주요 기능 시각적 소개
    ③ "한번 써볼까?"     → CTA (무료 시작하기) + 신뢰 요소
        ↓
    /signup → /login → / (대시보드)
    ④ "오~~~ 괜찮은데"   → 셀프 온보딩 (사람 없이 첫 가치 체감)
```

### 단계별 설득 요소

| 단계        | 유저 심리     | 필요한 것                        | 담당     |
|-----------|-----------|------------------------------|--------|
| ① 이게 뭔데?  | 3초 안에 판단  | 후킹 헤드라인 + 한 줄 설명             | 랜딩 페이지 |
| ② 기능이 뭔데? | 구체적 가치 확인 | 기능 3~4개 시각적 소개 (스크린샷/아이콘)    | 랜딩 페이지 |
| ③ 써볼까?    | 리스크 판단    | "무료", "가입 30초", 사용 중인 곳 수    | 랜딩 페이지 |
| ④ 괜찮은데    | 첫 가치 체감   | 그룹 생성 → 멤버 등록 → 첫 출석 (사람 없이) | 셀프 온보딩 |

### 현재 문제

- **①②③ 부재**: 로그인 페이지가 설득 없이 바로 행동(가입/로그인)을 요구
- **④ 부재**: 기존 파일럿 성공은 옆에서 직접 가르쳐서 가능했음. 제품만으로 첫 가치 체감 불가
- ⚠️ ④가 해결되지 않으면 랜딩에서 전환되어도 이탈함 — 두 문제 모두 해결 필요

### 해결 방향

| 문제           | 해결책            | 라우팅              |
|--------------|----------------|------------------|
| ①②③ 설득 흐름 없음 | 랜딩 페이지 신규 구현   | `/landing` (퍼블릭) |
| ④ 셀프 온보딩 없음  | 가입 후 가이드 흐름 구현 | `/` 대시보드 내 온보딩   |
