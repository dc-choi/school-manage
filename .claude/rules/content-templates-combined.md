# Content Templates — 통합 (피드 + 릴스)

피드 카드뉴스 + 릴스 슬라이드 영상 양쪽 게시용 통합 형식입니다.

> 역할/톤/규칙은 `rules/content.md` 참조
> 피드 단독: `rules/content-templates-feed.md` / 릴스 단독: `rules/content-templates-reel.md`

> **핵심 원칙**: 한 자료로 양쪽 게시. 카피=자막(중복 X), 음성 대본은 옵션.

## 사용 시나리오

> **통합의 본질**: 피드 + 릴스 양쪽 게시 + **음성 + 자막 + BGM 풀 영상**까지 갖춘 자료.
> 음성이 없으면 통합으로 분류할 가치 없음 (그냥 카드뉴스 슬라이드쇼). 그 경우 `feed` 단독 또는 `reel` 단독으로 분류한다.

### 적합 (사용 권장)

- **신기능 소개 / 업데이트**: 음성으로 친근하게 전달하면 메시지 강도 ↑ (4~6장)
- **사용자 피드백 반영형**: "들었어요 → 반영했어요" — 음성 진정성이 핵심
- **Before/After**: 음성으로 변화 포인트 강조 가능

### 부적합 (단독 형식 권장)

- **4컷 만화** — 가시성 저조로 중단 (`feed`)
- **사용자 인터뷰 8~10장** — 30초 영상에 못 담음 (`feed`)
- **단순 텍스트 안내** — 영상화 가치 없음 (`feed`)
- **POV/타임랩스/연출 중심** — 사진으로 변환 어색 (`reel`)
- **음성 추가 가치가 없는 콘텐츠** — BGM + 자막만으로 충분하면 `reel` 단독으로

## 표준 포맷 (장현도 피드백 2026-05-08 반영)

> "카드뉴스 슬라이드쇼 느낌으로 BGM 하나 넣고 사진만 빠르게 넘김. 30초도 길어. 일시정지하고 보는 시청자에게 정독 맡김."

| 항목      | 표준값                                           | 비고                                                |
| --------- | ------------------------------------------------ | --------------------------------------------------- |
| 길이      | **15~20초**                                      | 30초는 상한, 짧을수록 조회수 유리                   |
| 카드/장면 | **4~6장**                                        | 너무 많으면 빠른 슬라이드가 어색해짐                |
| 장면당    | **2~4초**                                        | 한 화면 가독 최저선                                 |
| BGM       | **필수**                                         | 잔잔 acoustic / piano / 트렌딩 오디오               |
| 자막      | **필수** (음소거 대비)                           | 카피와 동일 — 별도 자막 섹션 X                      |
| 음성      | **표준** (없으면 통합 의미 X)                    | CapCut Adam TTS. 음성 불필요면 `reel` 단독으로 분류 |
| 비율      | 1:1 카드 5장 → 9:16 캔버스 배치 + 상하 여백 채움 | 또는 두 비율로 각각 생성                            |

## 음성 가이드 (표준)

> 통합 콘텐츠는 음성을 표준으로 포함한다. 음성이 자연스럽지 않으면 `reel` 단독으로 분류.

| 항목   | 권장값                                                            |
| ------ | ----------------------------------------------------------------- |
| 도구   | **CapCut 내장 TTS** (영상 + 자막 + 음성 + BGM 한 도구에서 처리)   |
| 보이스 | **Adam** (한국어 차분 톤, 한국 릴스에서 자주 쓰임)                |
| 속도   | 1.0배 또는 0.95배                                                 |
| 적용   | 자막 클립 선택 → 우측 패널 "텍스트를 음성으로" → Adam 선택 → 적용 |
| 대안   | 어색하면 클로바 더빙(clovadubbing.naver.com) — 보이스 풍부        |

## 카드 구성 원칙

각 카드/장면마다 다음 4개 블록:

1. **카피** (피드 노출 + 릴스 자막 공용. 짧게, 18자/줄 이내)
2. **🎙 음성 대본** (필수, 자연스러운 구어체 — 카피보다 풍부해도 OK)
3. **나노바나나 프롬프트** (이미지 생성용)
4. (필요 시 헤더/서브)

## 게시별 추가 메모 — 필수 섹션

문서 끝에 다음 섹션 포함:

- **피드 게시** (1:1 카드뉴스 캐러셀): 5장 그대로 업로드, 캡션/해시태그 복붙
- **릴스 게시** (9:16 슬라이드 영상화):
    - CapCut에서 1:1 카드를 9:16 캔버스 배치
    - 장면당 2~4초
    - BGM 추가 (음성 시 자동 ducking -10dB)
    - 자막은 카피 그대로
    - 음성 사용 시 Adam TTS
