## 게시별 추가 메모

### 피드 게시 (1:1 카드뉴스) — ✅ 게시 완료

- 5장 그대로 캐러셀 업로드 완료 (2026-05-10 일 새벽)
- 캡션 + 해시태그 그대로 사용

### 릴스 게시 (9:16 슬라이드 영상화) — ✅ 게시 완료

> 통합 표준 — BGM + 자막 + 음성 풀 영상 (~18초). 짧을수록 알고리즘 유리.

- **CapCut**에서 5장 이미지를 9:16 캔버스에 배치 (각 **3~4초씩 = 총 ~18초**)
- **자막**: 카드 본문 카피 그대로 (음소거 대비)
- **음성**: CapCut Adam TTS — 위 "🎙 음성 대본" 5건 입력 → 각 슬라이드 매칭. BGM 자동 ducking (-10~-12dB)
- **BGM**: 잔잔한 acoustic/piano 또는 트렌딩 오디오 — 분위기만 잡기
- **전환**: 부드러운 페이드/슬라이드 (0.3~0.5초)
- 같은 캡션 + 해시태그 사용
