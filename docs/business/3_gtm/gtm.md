# GTM

## 포지셔닝

### 범용 메시지
"가톨릭 교회 모임 운영자가 출석·멤버·행사를 한곳에서 일관되게 관리하는 도구"

### 교리교사 메시지 (1차 검증)
"주일학교 교리교사가 출석·학생·행사를 한곳에서 일관되게 관리하는 도구"

### 사제/책임자 메시지
"통계/보고 신뢰도 향상과 의사결정 리드타임 단축"

## 채널

| 채널          | 설명                     | 현황                              |
|-------------|------------------------|---------------------------------|
| 인스타그램       | 1차 공지/리마인드 → 홈페이지 연결   | **계정 개설 완료** (`@weekly-school`) |
| 모임 운영자 네트워크 | 본당/지구 내 추천             | 온라인 접근 경로 필요                    |
| 교구 청소년국     | 홍보/교육 협력               | 접점 존재 (자료실/공지)                  |
| 콘텐츠 유입      | 실무용 자료 (출석부 템플릿/체크리스트) | 미구현                             |

## 전환 경로

### 인스타그램 → 가입
```
인스타그램 게시물/하이라이트 (인지 + 설명)
    ↓
프로필 링크 클릭
    ↓
로그인 페이지 (서비스 소개 + 스크린샷 포함)
    ↓
회원가입
```

- **A. 인스타그램 자체를 랜딩으로 활용**: 프로필 소개 + 하이라이트(기능소개/FAQ/사용후기)로 서비스 설명 → 추가 개발 없이 즉시 가능
- **B. 로그인 페이지에 서비스 소개 추가**: 한 줄 소개 + 스크린샷 2~3개로 가입 직전 이탈 방지 (SDD 핸드오프 필요)
- A + B 병행 → 인스타에서 충분히 이해한 뒤 로그인 페이지에서 한 번 더 확인
- 별도 랜딩페이지는 2단계에서 검토

### 교사 네트워크 → 가입
```
기존 사용자 (장위동 레퍼런스)
    ↓ 카카오톡 공유 (에셋: 이미지 + 가입 링크)
관심 있는 교리교사
    ↓ 인스타그램 확인 or 직접 가입 링크
회원가입
```

- 카카오톡 공유용 에셋 준비 필요: 서비스 스크린샷 1장 + 가입 링크
- "이거 써보는데 괜찮아" 한마디에 붙일 수 있는 링크

### 계정 모델 안내
- **주일학교(모임) 하나당 계정 하나** — 개인 계정이 아님을 가입 시 명확히 안내
- 사용자가 개인당 하나의 계정으로 오해하는 경향이 있으므로 가입 화면에서 명시 필요

## 세일즈 모션

```
모임 운영자 직접 접촉 (1차: 교리교사)
    ↓
2~4주 파일럿 (운영자 주도)
    ↓
책임자 공유/동의 (필요 시)
    ↓
운영자 개인 결제 중심 유료 전환
```

## 온보딩

- 목표: 첫 주 안에 "멤버 등록 + 1회 출석 기록" 완료
- 핵심: 주간 반복 업무 루프 진입, 초기 가치 체감
