## 게시별 추가 메모

### 피드 게시 (1:1 카드뉴스) — ✅ 게시 완료

- 6장 그대로 캐러셀 업로드 완료 (2026-05-09 토 밤)
- 카드 3, 4 (브라우저 단계)는 실제 모바일 스크린샷 대체 권장사항이었음

### 릴스 게시 (9:16 슬라이드 영상화) — ✅ 게시 완료

> 통합 표준 — BGM + 자막 + 음성 풀 영상 (~18초). PWA 출시 동기화.

- **CapCut**에서 6장 이미지를 9:16 캔버스에 배치 (각 3초씩 = 총 ~18초)
- **자막**: 카드 본문 카피 그대로 (음소거 대비)
- **음성**: CapCut Adam TTS — 위 "🎙 음성 대본" 6건 입력 → 각 슬라이드 매칭
- **BGM**: 잔잔한 acoustic/lo-fi 또는 트렌딩 오디오 — 친근한 팁 톤. 음성 들어가면 자동 ducking (-10~-12dB)
- **전환**: 부드러운 페이드 또는 슬라이드 (0.3초)
- 같은 캡션 + 해시태그 사용
