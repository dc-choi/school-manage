### F1. 4월 콘텐츠 도달 -78%

- 2-3월 TOP5 평균 조회수: **771**

### F2. 인스타 프로필 링크 = 주요 유입 채널 가설

- 누적 외부 링크 클릭 **318건**은 프로필 밖으로 이동한 트래픽이며 가입 완료 건수가 아니다
- 단기 28건/7일 = 일평균 **4건**. 4월 신규 도입 시기와 겹치지만 동일 사용자 귀속이 없어 인과로 확정하지 않는다
- 가입 전환은 UTM이 붙은 랜딩 방문과 동일 사용자 `sign_up` 퍼널로 별도 검증한다

### F3. 알고리즘 신선도 하락

- 비팔로워 도달 비율 54.9% → **43.9%** (-11%p)
- 게시 빈도 부족 → 알고리즘이 신규 사용자에게 덜 추천 추정
- 단기 게시 비중: 스토리 91.6% / 게시물 8.3% — **게시물 발행 부족**

### F4. 4컷 만화 데이터 모순 (의사결정 재고 필요)

- `instagram.md` "가시성 저조 피드백으로 중단 (02-26)"

### F5. 그림+글 결합 콘텐츠 우위

- 1위(4컷 만화)/2위(인터뷰 카드뉴스)/이후 기능 소개 카드뉴스 모두 **그림+글 결합 형식**
- 단발 텍스트 위주 콘텐츠(4/17 행사 안내 47조회)는 도달 약함
- 인지 효율: 그림+글 결합이 정보 전달과 시각적 매력 모두 충족

### F6. CTR 하락 (3.7% → 2.5%)

- 콘텐츠 매력도/CTA 약화 또는 비핵심 사용자 도달 비율 증가
