# 교리교사의 일요일 타임랩스

## 메타
- 유형: 릴스
- 타겟: 교리교사
- CTA: 없음 (순수 공감)
- 스타일: 타임랩스/스피드업

## 콘셉트
교리교사의 일요일을 빠르게 보여주는 타임랩스. 알람 끄기 → 허겁지겁 준비 → 미사 → 교리 → 집 도착하면 오후 2시. 마지막에 "...그리고 다음 주에 또 옵니다"

## 구성

### 장면 1 (0~3초)
- 알람 5개 울리는 화면 → 손으로 끄기
- 자막: "일요일 아침 7시"

### 장면 2 (3~6초)
- 허겁지겁 준비하는 모습 (스피드업)
- 자막: "지각이다 지각"

### 장면 3 (6~9초)
- 미사 참례 모습
- 자막: "미사"

### 장면 4 (9~12초)
- 교리 수업 / 출석 부르는 모습
- 자막: "교리"

### 장면 5 (12~15초)
- 집 도착, 시계 보면 오후 2시
- 자막: "...벌써 오후 2시"

### 엔딩 (15~18초)
- 검은 화면
- 자막: "그리고 다음 주에 또 옵니다"

## 캡션
```
일요일은 짧고 교리는 길다.
...그래도 다음 주에 또 갑니다.
```
