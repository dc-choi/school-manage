# 브레인스토밍 결과 보고서

## 주제

리브랜딩 선행 후 확장 전략 — 안준성(마케터) 재능기부 리브랜딩 제안과 이한별 미니프로그램(사다리타기/룰렛/메모/회의록+멤버십) 확장 제안을 어떤 순서/범위로 진행할지.

전제: 이번 건은 "이름(위클리스쿨) 유지 + 비주얼 아이덴티티 정비"다. 브랜드명 변경(루멘/피앗)은 bm-funding.md 기준 법인 전환 시 검토로 별개 레이어.

## 토론 요약

### Round 1

- 기획자가 시퀀스 A(리브랜딩 선행), B(회의록 병렬 + 사다리타기 보류), C(MAO 먼저) 3개를 제안하고 A+C 혼합을 권고.
- 비판자 핵심 지적: (1) "색조 불일치 → 전환율 저하" 인과 근거 없음 — 랜딩→가입 전환율 약 68%로 낮지 않음, (2) 색상 토큰 교체 1일 공수는 과소산정(전례력 4색과의 WCAG AA contrast 재검증 필요), (3) 법인 전환 시 브랜드명 변경과의 충돌을 안준성에게 고지할지 딜레마, (4) 이한별 1명 발언으로 회의록 수요 일반화 불가(인터뷰 모수 청년 편중은 STATUS 오픈 이슈), (5) 시퀀스 C의 차별점은 "리브랜딩 안 함" 하나뿐.
- 기획자 수용: 리브랜딩 근거를 "전환율 개선"에서 "비주얼 일관성 정비 + 9월 광고 소재 준비"로 교체. 공수 2~4일로 수정. 시퀀스 C를 A+C 혼합에 흡수. 브랜드명 변경 가능성은 지금 꺼내지 않되 저작권 귀속은 미팅에서 합의.

### Round 2

- 비판자: 시안 데드라인 부재(무기한 대기 구조), 상업적 사용 허락 누락, PG 지연 플랜 B 부재, 7~8월 SDD 3개(결제/알림/메모) 병렬은 1인 개발 과적, 메모의 플랜 배치(베이직/프로) 미결, "광고 집행 결정 후 색상 교체" 구조의 의존성 역전.
- 기획자: 시안 데드라인 8월 15일("9월 재시작 전 적용" 프레이밍). 저작권 = 안준성 소유 + 위클리랩(최동철) 사업 목적 무제한 사용 허락, 카톡 요약 기록. SDD 우선순위를 자금 확보 최우선 원칙으로 정렬(결제 1순위 → D유형 알림 2순위 → 메모 3순위). 색상 토큰 교체를 광고 집행에서 분리해 "시안 수령 즉시 적용"으로 변경 — 의존성 역전과 안준성 결과물 방치 리스크 동시 해소.

### Round 3

- 비판자: 미팅 일정 자체가 미확정(안준성 마지막 대화 6/4 이후 일주일 공백), PG 완료 여부는 오늘 확인할 일, GA4 온보딩 진단은 비용 0이므로 6월 중 완료, Q1 응답 모수 기준 필요(이한별 1명 과의존 반복), 온보딩 트리거 수치 기준 미정(9월 자기 기만 위험), 이한별 무응답은 Silent Churn성 관계 리스크. 10개 항목 승인/조건부 매트릭스 제시.
- 기획자: 전 조건 수용. 온보딩 트리거 수치 기준 "sign_up → first_group 전환율 4주 연속 33% 이상"(현재 17%의 2배) 확정. "지금 당장 4액션" 도출.

### 핵심 결론

"리브랜딩 선행"의 실질 의미는 개발자 리소스의 선행 투입이 아니다. 안준성이 시안을 만드는 동안 개발자는 PG 확인, 결제 플로우 SDD, 인터뷰를 진행하고, 색상 토큰을 실제로 건드리는 시점은 시안 수령 직후(8월 중, 2~4일)다. 리브랜딩, 이한별 기능 제안, MAO 방어는 순차가 아닌 병렬 트랙이다.

## 아이디어 목록

### 비주얼 아이덴티티 리브랜딩 (안준성 협업 트랙)

- 한 줄 요약: 이름 유지, 로고/키컬러/톤앤매너/인스타 템플릿을 안준성 재능기부로 정비하고 시안 수령 즉시 색상 토큰 교체.
- 사용자 가치: 전환율 개선 근거는 없음(Round 1에서 폐기). 가치는 인스타-랜딩-앱 비주얼 일관성과 9월 광고 소재 사전 준비.
- 실현 방법: 6월 미팅 아웃풋 3개 — (a) 앵커 컬러 = 인디고/보라 계열, 전례력 4색 = 시즌 콘텐츠 컬러로 2레이어 분리 합의, (b) 시안 범위 = 로고 + 색상 토큰 + 인스타 3분할 템플릿, 8/15 데드라인, (c) 저작권 = 안준성 소유 + 위클리랩 사업 목적 무제한 사용 허락. 적용 시 WCAG AA contrast 재검증 포함 2~4일.
- 차별점: 가톨릭 전례력 색을 브랜드 시스템에 녹인 경쟁자 부재(브랜치스는 일반 교회 청년부 톤).

### 메모/회의록 기능 (이한별 제안 + bm.md 프로 플랜 해자 연결)

- 한 줄 요약: 이한별의 "저장은 멤버십" 제안을 Q1(단순 메모 저장 WTP)과 Q2(AI 분류 + PDF 회의록 WTP) 두 가설로 분리 검증 후 착수.
- 사용자 가치: 이한별(성북)과 장현도 발언 2건 — 모수 부족, 7~8월 인터뷰(STATUS 우선 접근 순서 ①~⑤)로 검증 필요.
- 실현 방법: 7~8월 인터뷰 2부에 Q1/Q2 포함 → 8월 플랜 배치 결정(3명 이상 중 Q1 "예" 2명 이상이면 베이직 결제 기능 배치 + SDD) → 회의록 AI는 Q2 3명 이상 + 구글 캘린더 연동 사양 결정의 이중 조건 충족 시 10~11월 착수.
- 차별점: 회의록 자동 분류 + PDF + 전례력 캘린더 연동은 밴드/카카오워크에 없음.

### 사다리타기/룰렛/제비뽑기 미니프로그램

- 한 줄 요약: 출석부 안에 가벼운 수업용 뽑기 도구를 넣자는 제안 — 현재 형태로는 보류.
- 보류 근거: 카카오 주사위, 웹 사다리타기 등 기존 범용 도구로 이미 해결된 니즈(YAGNI). pentecost-gift-draw.md 기각 사유 인용은 맥락 불일치로 Round 1에서 교체됨.
- 재논의 조건: "가톨릭 특화 뽑기(성경 구절/축일 기반 이름 뽑기 등)" 기획이 성립하는 시점에 별도 브레인스토밍.

### MAO 우선 독립 시퀀스 (시퀀스 C)

- 한 줄 요약: 리브랜딩을 미루고 MAO 38→50에 집중 — 독립 시퀀스로는 기각.
- 기각 사유: 차별 작업(D유형 자동 알림 등)이 어느 시퀀스에서든 필수 작업이라 독립 경로의 실체가 없음. 강점(자금 확보 정합성)은 A+C 혼합에 흡수.

## 실행 계획

**지금 당장 (오늘~이번 주, 합산 1~2시간)**

1. PG 본계약 완료 여부 직접 확인. 완료 시 결제 플로우 SDD 0단계 즉시 착수.
2. GA4에서 sign_up 이후 이벤트 시퀀스 진단: `/groups` 진입 비율 vs 즉시 이탈 비율. 50% 기준 분기 — 경로 미진입이면 비로그인 모달 SDD 격상, 즉시 이탈이면 온보딩 트리거 조건 재정의.
3. 이한별에게 카톡: Q1/Q2 직접 질문 + 사다리타기는 가톨릭 특화 버전으로 추후 재논의 공유.
4. 안준성에게 카톡: 미팅 일정 제안(이번 주/다음 주).

**6월**: 안준성 미팅(아웃풋 3개 확정), PG 완료 즉시 결제 SDD + 인터뷰 모집 개시(① 목포해군부터), GA4 진단 완료.
**7월**: 결제 플로우 개발(1순위), 인터뷰 1차 2~3명, GA4 분기 결과 이행, D유형 자동 알림 SDD(2순위, 병렬 가능 범위).
**8월**: 인터뷰 2차(② 자모/베테랑 + ③ 초등부, 방학 후반), 결제 MVP 배포 목표, 8/15 시안 수령 즉시 색상 토큰 교체, 메모 플랜 배치 결정.
**9월 4트리거 점검**: ② 온보딩 단절 해소는 "sign_up → first_group 전환율 4주 연속 33% 이상" 수치 기준으로 평가. 충족 시 인스타 재시작(비주얼은 이미 적용 상태), 미충족 시 광고만 보류.
**10~11월**: Q2 + 캘린더 사양 이중 조건 충족 시 회의록 AI 착수. 결제 배포 후 4주 전환율 0이면 즉시 인터뷰로 원인 진단(R8 대응).

## 인터뷰 항목

- Q1: "출석부 안에 텍스트 메모 저장(회의 내용, 공지 기록)만 되는 기능이면, 저장 기능에 월 X원을 낼 의향이 있는가?"
- Q2: "녹음 입력 → AI 자동 분류 → PDF 생성까지 되는 회의록 자동화라면 월 Y원을 낼 의향이 있는가?"
- 대상: 이한별(즉시, 카톡) + 7~8월 인터뷰 대상 전원(목포해군 → 자모/베테랑 → 초등부 P1, STATUS 우선 접근 순서 준수). 판정 기준: 3명 이상 중 Q1 "예" 2명 이상.
- 인터뷰 세션 구조: 1부 페르소나 수요 파악 → 2부 가격 anchor + Q1/Q2 (가격 질문의 역오염 차단).

## 누락된 관점

- 9월 첫 광고 캠페인 예산 가설 미설정(그 시점 자금 상황으로 결정하기로 — 과도한 사전 설계 방지).
- 인스타 콘텐츠 형식 정책(그림+글 결합) 재정의는 9월 재검토 별건 — 안준성 템플릿과의 정합은 미팅에서 일부 선반영 가능.
- 안준성 제안 중 "기존 콘텐츠 일부 재발행"의 구체 범위는 미팅에서 확정 필요.

## 사용자 결정 대기

1. **A+C 혼합 최종 타임라인 승인 여부** — 특히 "지금 당장 4액션"(PG 확인, GA4 진단, 이한별 카톡, 안준성 카톡)을 이번 주 안에 실행할지.
2. **앵커 컬러 방향** — 미팅 전에 본인 입장(현 Indigo #4F46E5 유지 vs 보라 계열 신규) 1차 확정 필요. 5/26 카톡에서는 "보라 계열"로 기울었음.
3. **온보딩 트리거 수치 기준** — "4주 연속 33% 이상" 채택 여부. 채택 시 STATUS 4트리거 정의에 반영 필요.
4. **메모/회의록 Q1/Q2 검증 설계 승인** — 이한별에게 보낼 카톡 문구 포함.
5. 승인 항목의 로드맵/TARGET 편입은 `/bs-to-target`으로 별도 진행(사용자 승인 없이 사업 문서 자동 수정 금지).
