# 브레인스토밍 결과 보고서 — 브랜드 전략 레이어

## 주제

위클리스쿨 브랜드 전략 레이어 재정립 — 포지셔닝, 타겟, 브랜드 약속/핵심 가치, 파운더 내러티브. 사용자 지시: "시각 말고 전략, 언어, 경험 다 고민 필요. 기존 bs 문서는 참고하되 구속 없이 재고민." 시각 레이어(인디고/로고 v2/주이/랜딩 고정, [2026-06-11/brand-direction.md](../2026-06-11/brand-direction.md))와 이름 유지는 불변 제약.

**최종 산출물(전략 문서 확정안)은 [brand-strategy-draft.md](./brand-strategy-draft.md) 참조.** 본 문서는 토론 경위와 판정 기록.

## 아이디어 목록 (Phase 1 후보)

### 후보 A — 가톨릭 교리교사 전용 도구 (채택, 수정 10건 반영)

- 한 줄: "가톨릭 교리교사가 매주 주일 여는 유일한 전용 도구". 가톨릭 전용 베팅, 교리교사 단독 우선, 허브 선언 보류.

### 후보 B — 주간 허브, 가톨릭부터 증명 (기각)

- 한 줄: "교리교사의 주일 너머 가톨릭 모임 전체의 허브". 기각 사유: 허브 근거가 이한별 1건(인터뷰 4명 청년 편중), 회의록/공지/행사 미구현 상태의 허브 약속은 R12(가치 체감 부족 이탈) 변형, 브랜치스(개신교 95교회 사업화)와 같은 카테고리로 스스로 진입. 단 "방향 나침반"으로 내부 보존 — 최종안의 내부용 포지셔닝에 흡수.

### 후보 C — 가톨릭 전용이되 이름은 중립 (기각)

- 한 줄: 범용 확장을 위해 "가톨릭" 표현 후퇴. 기각 사유: 가톨릭 전용이 유일한 방어막(브랜치스 진입 비용 = 전례력/교구 계층/세례명 구조). 가톨릭 네트워크(사제 추천, 청소년국)가 가톨릭 전용이라 작동하는 GTM 채널. 1인 개발 조건에서 해자 있는 작은 시장 > 해자 없는 큰 시장.

## 토론 요약 (수렴 경로)

### Round 1 — 후보 A의 약점 4개

- "반복의 신뢰" 가치는 제품이 만드는 게 아니라 교리교사 봉사 루틴 — 제품 근거 없음 → 폐기.
- 브랜드 약속 "기억하는 것"이 실사용 패턴(빠른 조회, bm.md)과 어긋남 → 교체.
- 도구 포지셔닝이 해자(회의록)를 설명 못 하는 모순 → "주일 운영" 컨테이너 언어 도입.
- 파운더 내러티브 "만든 사람의 권위" vs "같은 처지의 공감" 미선택 → 공감 선택(근거는 현장 경험 권위).
- 콘텐츠 소재를 도구 경험만으로 한정하면 9월 이후 고갈 → 레이어 2(파운더 시각: AI 회칙 해석 등 — 교안 아님, 교구 비경쟁 장르) 조건부 도입.

### Round 2 — 수정안의 새 허점

- "주일 운영 전부"는 회의록 미구현 상태에서 과약속 → 포지셔닝 내부용/외부용 분리.
- "2분이면 충분"은 미검증 수치(체류 197s = 3.3분, R24 학생 미등록 이탈 4곳 반례) → 수치 폐기, "미사 직전" 상황 표현으로.
- "낯설지 않은 도구"는 부정형이라 준성이 카피로 변환 불가 → 긍정형 전환.
- 레이어 2 "반응" 판단 기준 부재 → 저장 수 지표로 수치화.
- 보이스 기준 문장 1개로는 톤앤매너 가이드 입력 불충분 → 5개 스펙트럼으로 확장.

### Round 3 — 최종 판정 (기각 0)

- 12개 항목 승인 7 + 조건부 5. 조건 전부 기획자 수용: 외부용 동사 전환("여는" → "운영하는"), 내외부 물리 분리, 타겟 "서울대교구를 시작으로 전국" 정밀화, 약속의 "속도이지 기능 범위 아님" 인계 명시, 가치 2에 결과 문장 추가, 가치 순서 1-3-2, 스펙트럼 5(AI 개발자+신앙인)는 레이어 2 전용 제한, 레이어 2 기준에 절대치(저장 5건) 추가.

## 누락된 관점 (다음 세션)

- **언어 레이어**: 슬로건/보이스&톤 가이드(준성 톤앤매너 입력 패키지) — 이번 전략 문서가 입력.
- **경험 레이어**: 콘텐츠 캘린더(9월 재시작), 접점 설계, 확장 우선순위 — 전략 확정 후.
- 3단계(알림/공지) 출시 시점의 포지셔닝 중간 마일스톤 — 비판자가 식별, 3단계 기획 세션에 위임.

## 사용자 결정 대기

1. **전략 문서 확정안 승인 여부** — [brand-strategy-draft.md](./brand-strategy-draft.md) 전문 검토. 핵심 결정 3개: (a) 가톨릭 전용 베팅(범용 확장 언어 배제), (b) 외부 포지셔닝 "가톨릭 교리교사의 주일을 운영하는 도구", (c) 브랜드 약속 "주일 아침 미사 직전에도, 이거 하나면 됩니다".
2. 승인 시 brand README 반영 + 언어 레이어 세션(준성 패키지) 진행 여부.
3. 파운더 내러티브 1단락(본인 이야기) — 본인 언어로 다듬을 부분이 있는지.
