# GTM

## 포지셔닝

### 교리교사 메시지
"중고등부 주일학교 교리교사가 출석·학생·행사를 한곳에서 일관되게 관리하는 도구"

### 사제 메시지
"통계/보고 신뢰도 향상과 의사결정 리드타임 단축"

## 채널

| 채널        | 설명                     | 현황                    |
|-----------|------------------------|-----------------------|
| 교리교사 네트워크 | 본당/지구 내 추천             | 온라인 접근 경로 필요          |
| 교구 청소년국   | 홍보/교육 협력               | 접점 존재 (자료실/공지)        |
| 콘텐츠 유입    | 실무용 자료 (출석부 템플릿/체크리스트) | 미구현                   |
| 인스타그램     | 1차 공지/리마인드 → 홈페이지 연결   | 운영 계획: `instagram.md` |

## 세일즈 모션

```
교리교사 직접 접촉
    ↓
2~4주 파일럿 (교리교사 주도)
    ↓
사제 공유/동의 (필요 시)
    ↓
교리교사 개인 결제 중심 유료 전환
```

## 온보딩

- 목표: 첫 주 안에 "학생 등록 + 1회 출석 기록" 완료
- 핵심: 주간 반복 업무 루프 진입, 초기 가치 체감

### 진입점: 회원가입 (완료)

| 항목    | 이전 상태      | 현재 상태 (2026-01-25 완료) |
|-------|------------|-----------------------|
| 계정 생성 | 관리자 직접 생성  | 사용자 자가 가입             |
| 진입 장벽 | 높음 (접근 불가) | 낮음 (즉시 시작 가능)         |
| 전환 경로 | 없음         | 가입 → 그룹 생성 → 사용       |
