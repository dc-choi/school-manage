# 공감 콘텐츠 - "사순 시기 시작되면 교리교사 머릿속"

## 메타
- 유형: 피드 (4컷 만화)
- 타겟: 교리교사
- CTA: 없음 (순수 공감 콘텐츠)
- 스타일: 밈/짤, 4컷 만화

### 공통 스타일 프롬프트
```
4컷 만화, 밈/짤 느낌. 둥글둥글한 귀여운 캐릭터 스타일, 깔끔한 흰 배경에 심플한 선화. 주인공은 20대 여성 교리교사, 머리에 십자가 머리핀을 달고 있다. 인스타그램 4:3 비율, 4컷을 한 장에 2x2 배치. 선화만이 아닌 풀 컬러 채색으로 그려야 하며, 파스텔톤의 부드럽고 따뜻한 색감을 사용한다. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션
```
사순 시기 시작되면 교리교사 머릿속에 떠오르는 것들.
부활 행사, 판공성사, 십자가의 길...
준비할 게 한가득이지만, 다 해내는 우리 교리교사들 💪
```

## 나노바나나 프롬프트

### 1컷 (기)
- 대사: "재의 수요일이네... 올해 부활 행사 뭐 하지?"

```
귀여운 둥글둥글한 캐릭터 스타일, 밈/짤 느낌. 교리교사 캐릭터(20대 여성, 십자가 머리핀)가 달력을 보며 고민하는 장면. 달력에 '재의 수요일'이 동그라미 쳐져 있고, '부활 대축일'까지 화살표가 그려져 있다. 말풍선에 "재의 수요일이네... 올해 부활 행사 뭐 하지?". 턱을 괴고 생각하는 표정. 4컷 만화 중 1컷 스타일, 깔끔한 흰 배경에 심플한 선화.
```

### 2컷 (승)
- 대사: "판공성사 일정 잡아야지... 십자가의 길도 같이 해볼까?"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터 머리 위에 생각 풍선이 떠 있는 장면. 풍선 안에 '판공성사 일정', '십자가의 길?', '부활 달걀?', '연극?' 등이 적혀 있다. 말풍선에 "판공성사 일정 잡아야지... 십자가의 길도 같이 해볼까?". 턱에 손을 대고 고민하는 표정. 메모장과 펜을 들고 있다. 4컷 만화 중 2컷 스타일, 이것저것 고민이 떠오르는 느낌.
```

### 3컷 (전)
- 대사: "...출석 확인도 해야 하는데 시간이 없다"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 책상에 엎드려 있는 장면. 주변에 메모, 교리 자료, 출석부가 쌓여 있다. 말풍선에 "...출석 확인도 해야 하는데 시간이 없다". 머리 위에 소용돌이 이펙트. 피곤하지만 포기 못하는 표정. 4컷 만화 중 3컷 스타일, 할 일에 파묻힌 느낌.
```

### 4컷 (결)
- 대사: (학생들과 함께 웃으며) "자, 우리 사순 시기에 어떤 실천을 해볼까?"

```
같은 캐릭터 스타일 유지. 교리교사 캐릭터가 학생들 앞에서 환하게 웃으며 수업하는 장면. 학생 캐릭터 3~4명이 손을 들고 밝은 표정으로 참여하고 있다. 말풍선에 "자, 우리 사순 시기에 어떤 실천을 해볼까?". 교리교사 머리 위에 반짝이는 후광 이펙트. 따뜻한 분위기. 4컷 만화 중 4컷(마무리) 스타일, 모든 준비가 보람으로 돌아오는 순간. 전체적으로 밝고 따뜻한 톤.
```
