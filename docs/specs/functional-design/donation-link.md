# 기능 설계: 도네이션 링크 + 앱 내 후원 안내

> PRD를 기반으로 구체적인 기능 흐름과 인터페이스를 정의합니다.

## 연결 문서

- PRD: `docs/specs/prd/donation-link.md`
- Task: `docs/specs/target/functional/tasks/donation-link.md`
- Development: `docs/specs/target/functional/development/donation-link-frontend.md`

## 흐름/상태

### 사용자 플로우

1. 사용자가 사이드바 푸터(데스크톱) 또는 모바일 메뉴 하단에서 "후원하기" 링크를 발견
2. 클릭 시 설정 페이지 후원 섹션(`/settings#donation`)으로 이동
3. 카카오페이 버튼을 클릭
4. 새 탭에서 카카오페이 송금 페이지가 열림
5. GA4 이벤트 전송 (`donation_link_click`)

### 상태 전이

없음 — 외부 링크 연결만 수행하며, 앱 상태 변경 없음

## UI/UX

### 화면/컴포넌트

| 화면 | 설명 | 주요 요소 |
|------|------|----------|
| Sidebar 푸터 | 저작권 아래 후원 링크 | "후원하기" 텍스트 링크 (Heart 아이콘) |
| 모바일 메뉴 하단 | 네비게이션 아래 후원 링크 | "후원하기" 텍스트 링크 (Heart 아이콘) |
| SettingsPage 후원 섹션 | 카드 형태 후원 안내 | 메시지 + 카카오페이 버튼 |

### Sidebar 푸터 레이아웃

```
┌─────────────────────┐
│ © 2022–2026 주일학교  │
│       출석부          │
│                     │
│    ♥ 후원하기         │
└─────────────────────┘
```

- 저작권 텍스트 아래에 "후원하기" 링크 배치
- `Heart` 아이콘 + 텍스트, 클릭 시 `/settings#donation` 으로 이동
- 데스크톱(lg)에서만 표시 (기존 푸터와 동일)

### 모바일 메뉴 하단 레이아웃

- SheetContent 네비게이션 아래에 동일한 "후원하기" 링크 배치
- 클릭 시 메뉴 닫힘 + `/settings#donation` 이동

### SettingsPage 후원 섹션

```
┌─────────────────────────────────┐
│ ♥ 후원하기                       │

│ 이 서비스는 교리교사가 만든        │

│ 후원금은 서버 운영비로 사용됩니다.   │
```

- Card 컴포넌트 사용 (기존 설정 섹션 스타일과 통일)
- AccountDeleteSection 위, MembersSection 아래에 배치
