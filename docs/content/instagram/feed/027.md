# 사용자 인터뷰 - "처음 써본 성북동 교감 선생님께 물었습니다"

## 메타

- 유형: 피드 (카드뉴스 9장)
- 타겟: 교리교사
- CTA: "바로 시작하기 → 프로필 링크"
- 스타일: 인터뷰 Q&A 카드뉴스
- 인터뷰이: 성북동 교감 선생님 (익명)

### 공통 스타일 프롬프트

```
인터뷰 카드뉴스 스타일. 깔끔한 흰색 배경에 부드러운 파스텔톤 포인트 컬러(연한 민트색과 하늘색의 조합). 둥글둥글한 귀여운 일러스트 아이콘 포함. 인스타그램 1:1 정사각형 비율. 질문은 작고 가벼운 서체, 답변은 크고 굵은 서체로 강조. 풀 컬러 채색, 파스텔톤의 부드럽고 따뜻한 색감. 산뜻한 첫인상 분위기. 말풍선 및 모든 텍스트는 반드시 한국어로 렌더링할 것.
```

## 캡션

```
이번엔 출석부 앱을 처음 써보신 성북동 교감 선생님께 여쭤봤습니다.

써보기 전엔 어떻게 출석을 관리하셨는지,
처음 써봤을 때 느낌은 어땠는지,
다른 선생님들 반응은 어땠는지,
짧고 솔직하게 답해주셨어요.

"생각보다 이용 방법이 어렵지 않아서 좋았습니다."
"부서별 출석 확인이 가능하다는 점이 좋았습니다."

직접 계산해야 하는 부분은 아쉬웠다고 솔직히 말씀해주셨고,
그래도 추천한다면 이렇게 말씀하시겠다고 하셨어요.

"편리하게 출석 관리가 가능합니다!"

프로필 링크에서 바로 시작하실 수 있어요.
```
